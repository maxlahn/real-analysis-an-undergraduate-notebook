    Section 3.1 Introduction to Metric Geometry

    To begin, we introduce a generalization of our usual notion of distance which is called a metric. The properties of physical distance make it an extremely useful tool for formalizing the analytical notions mentioned above.
    🔗

    Subsection 3.1.1 Introduction to Metrics

    Not all properties of physical distance are generic enough to generalize to other settings in which we might want to measure distance-like quantities. This question of which properties of a specific example (usually Euclidean space) are salient enough to merit requirement in a generalization is ubiquitous throughout mathematical exposition, and will recur frequently throughout this text.
    🔗

    Definition 3.1.1. Metric space.

    A real-valued function \(\rho \colon X^{ 2 } \to \R \) on the Cartesian square \(X^{ 2 } \) of a set \(X \text{,}\) which is called the ground set and whose elements are called points, is a metric on \(X \) if it has the following properties:
    1. Non-degeneracy.
      For any points \(x_{ 1 } , x_{ 2 } \in X \text{,}\) \(\rho \of{ x_{ 1 } , x_{ 2 } } = 0 \) if and only if \(x_{ 1 } = x_{ 2 } \text{.}\)
      🔗
      🔗
    2. Symmetry.
      \(\rho \of{ x_{ 1 } , x_{ 2 } } = \rho \of{ x_{ 2 } , x_{ 1 } } \) for any points \(x_{ 1 } , x_{ 2 } \in X \text{.}\)
      🔗
      🔗
    3. Triangle inequality.
      \(\rho \of{ x_{ 1 } , x_{ 3 } } \leq \rho \of{ x_{ 1 } , x_{ 2 } } + \rho \of{ x_{ 2 } , x_{ 3 } } \) for any points \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{.}\)
      🔗
      🔗
    In this case, the ordered pair \(\of{ X , \rho } \) is called a metric space.
    🔗
    🔗

    Remark 3.1.2. Implied metrics.

    We will see that, in general, there are many metrics on a given set, no one of which may truly be considered canonical. In fact, even though some distinct metrics give identical answers to point-set topological concerns such as convergence and continuity, in most cases even a choice one of these classes of metrics is arbitrary in some respects.
    🔗
    However, sometimes, if a metric \(\rho \colon X^{ 2 } \to \R \) on a set \(X \) is clear from context, previously specified, or implicitly present, some mathematicians will refer to the metric space \(\of{ X , \rho } \) by only the underlying set \(X \text{.}\) We will try to avoid such ambiguities, and we will attempt to always explicitly specify the metric with which we will be working.
    🔗
    🔗

    Example 3.1.3. Metric spaces.

    The following are examples of metrics and metric spaces:
    1. Let \(n \in \N \) be a natural number. The Euclidean metric \(d \colon \of{ \R^{ n } }^{ 2 } \to \R \) on \(n \)-dimensional real coordinate space \(\R^{ n } \) is the function defined by the formula
      \begin{equation*} d \of{ \vect{ x } , \vect{ y } } = \sqrt{ \sum_{ k = 1 }^{ n }{ \of{ x_{ k } - y_{ k } }^{ 2 } } }\text{.} \end{equation*}
      Similarly, the Euclidean metric \(d \colon \of{ \C^{ n } }^{ 2 } \to \R \) on \(n \)-dimensional real coordinate space \(\C^{ n } \) is the function defined by the formula
      \begin{equation*} d \of{ \vect{ w } , \vect{ z } } = \sqrt{ \sum_{ k = 1 }^{ n }{ \abs{ w_{ k } - z_{ k } }^{ 2 } } }\text{.} \end{equation*}
      🔗
      As defined above, the Euclidean metrics are metrics on \(\R^{ n } \) and \(\C^{ n } \text{,}\) respectively. The metric spaces \(\of{ \R^{ n } , d } \) and \(\of{ \C^{ n } , d } \) are called \(n \)-dimensional real Euclidean space and \(n \)-dimensional complex Euclidean space, respectively. \(1 \)- and \(2 \)-dimensional real Euclidean space \(\of{ \R , d } \) and \(\of{ \R^{ 2 } , d } \) are called the real line and the Euclidean plane, respectively, and \(1 \)-dimensional complex Euclidean space \(\of{ \C , d } \) is called the complex plane.
      🔗
      🔗
    2. Consider the real-valued function \(\rho \colon \opint{ 0 }{ \infty }^{ 2 } \to \R \) defined by the formula
      \begin{equation*} \rho \of{ x , y } = \abs{ \ln{ \of{ \frac{ x_{ 1 } }{ x_{ 2 } } } } } \end{equation*}
      As defined above, \(\rho \) is a metric on the open interval \(\opint{ 0 }{ \infty } \text{,}\) and so \(\of{ \opint{ 0 }{ \infty } , \rho } \) is a metric space.
      🔗
      🔗
    3. The discrete metric \(\delta_{ X } \colon X^{ 2 } \to \set{ 0 , 1 } \) on a set \(X \) is the function defined by the formula
      \begin{equation*} \delta_{ X } \of{ x_{ 1 } , x_{ 2 } } = \begin{cases} 1 \amp x_{ 1 } = x_{ 2 } \\ 0 \amp x_{ 1 } \neq x_{ 2 } \text{.}\end{cases} \end{equation*}
      As defined above, the discrete metric \(\delta_{ X } \) is a metric on \(X \text{,}\) and the metric space \(\of{ X , \delta_{ X } } \) is called a discrete metric space.
      🔗
      🔗
    🔗
    🔗
    The definition of a metric is pretty abstract, and it may be difficult to see how or why requiring only nondegeneracy, symmetry, and the triangle inequality would give a useful generalization of distance, especially given the strange and outlandish properties of the discrete metric described above, which we will explore later in this section. However, requiring the three above properties actually forces a metric to have many of the properties of our usual notion of distance, and so we can often use our knowledge of distance in the physical world to inform our intuition about metric spaces.
    🔗
    As an example, consider for a moment the notion of negative distance. In the real world, it makes no sense to consider negative distances between physical objects, and we would be equally confused by negative distances between points in a metric space. However, the defining axioms of a metric do not a priori explicitly forbid this nonsensical outcome from occurring. At first glance, it might seem like a metric might be allowed to take negative values, even though the three examples of a metric given above are certainly nonnegative. In fact, this is never the case.
    🔗

    Lemma 3.1.4. Metrics are non-negative.

    A metric takes only non-negative values.
    🔗
    🔗

    Proof.
    Let \(\rho \) be a metric on a set \(X \text{,}\) and note that
    \begin{align*} \rho \of{ x_{ 1 } , x_{ 2 } } \amp = \frac{ 2 \rho \of{ x_{ 1 } , x_{ 2 } } }{ 2 } = \frac{ \rho \of{ x_{ 1 } , x_{ 2 } } + \rho \of{ x_{ 1 } , x_{ 2 } } }{ 2 }\\ \amp = \frac{ \rho \of{ x_{ 1 } , x_{ 2 } } + \rho \of{ x_{ 2 } , x_{ 1 } } }{ 2 } \geq \frac{ \rho \of{ x_{ 1 } , x_{ 1 } } }{ 2 } = \frac{ 0 }{ 2 } = 0 \end{align*}
    for all points \(x_{ 1 } , x_{ 2 } \in X \text{.}\)
    🔗
    🔗
    The above argument shows that, just as in the physical world, the distance between any two points in a metric space must be nonnegative. In fact, unlike many of the spaces which are studied by analysts (some of which we will encounter in the following chapters), metric spaces can be surprisingly accessible using only our intuitions about distance in the physical world. For example, the triangle inequality expresses that there are no “shortcuts” between two points in a metric space through a third point; in geometric terms, we require that no leg of a triangle may exceed in length the sum of the lengths of the other two legs. While the triangle inequality establishes an upper bound on the length of a leg of a triangle, the following property of a metric is called the reverse triangle inequality, and it establishes a lower bound on the length of a leg of a triangle.
    🔗

    Proposition 3.1.5. Reverse triangle inequality.

    Let \(\of{ X , \rho } \) be a metric space. Then
    \begin{equation*} \rho \of{ x_{ 1 } , x_{ 2 } } \geq \abs{ \rho \of{ x_{ 1 } , x_{ 3 } } - \rho \of{ x_{ 2 } , x_{ 3 } } } \end{equation*}
    for all points \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{.}\)
    🔗
    🔗

    Proof.
    If \(\rho \of{ x_{ 1 } , x_{ 3 } } \leq \rho \of{ x_{ 2 } , x_{ 3 } } \text{,}\) then \(\rho \of{ x_{ 1 } , x_{ 3 } } - \rho \of{ x_{ 2 } , x_{ 3 } } \leq 0 \text{,}\) and so
    \begin{align*} \abs{ \rho \of{ x_{ 1 } , x_{ 3 } } - \rho \of{ x_{ 2 } , x_{ 3 } } } \amp = \rho \of{ x_{ 2 } , x_{ 3 } } - \rho \of{ x_{ 1 } , x_{ 3 } }\\ \amp \leq \rho \of{ x_{ 2 } , x_{ 1 } } + \rho \of{ x_{ 1 } , x_{ 3 } } - \rho \of{ x_{ 1 } , x_{ 3 } } = \rho \of{ x_{ 1 } , x_{ 2 } }\text{.} \end{align*}
    On the other hand, if \(\rho \of{ x_{ 1 } , x_{ 3 } } \geq \rho \of{ x_{ 2 } , x_{ 3 } } \text{,}\) then \(\rho \of{ x_{ 1 } , x_{ 3 } } - \rho \of{ x_{ 2 } , x_{ 3 } } \geq 0 \text{,}\) and so
    \begin{align*} \abs{ \rho \of{ x_{ 1 } , x_{ 3 } } - \rho \of{ x_{ 2 } , x_{ 3 } } } \amp = \rho \of{ x_{ 1 } , x_{ 3 } } - \rho \of{ x_{ 2 } , x_{ 3 } }\\ \amp \leq \rho \of{ x_{ 1 } , x_{ 2 } } + \rho \of{ x_{ 2 } , x_{ 3 } } - \rho \of{ x_{ 2 } , x_{ 3 } } = \rho \of{ x_{ 1 } , x_{ 2 } }\text{.} \end{align*}
    🔗
    🔗
    We have just seen that metric spaces often agree with our geometric intuitions about distance. However, we must be careful not to assume too much about the geometry of metric spaces in general; even though many of the usual properties of distance in the physical world are implied by the defining axioms of a metric space, there is still lots of room for geometric peculiarities. In particular, some of the metrics defined above disagree with our geometric intuitions in confusing ways. This is because they satisfy a stronger form of the triangle inequality than is true in the physical world. We will explore these ultrametrics in the reading questions for this section.
    🔗
    🔗

    Subsection 3.1.2 Constructing New Metrics

    We will now see how to construct new metrics out of given metric spaces. Given a metric space, there is a natural metric on any subset of its ground set. This restriction of the metric structure to a subset is done in the obvious way; the distance between any points in the subset is just their distance when considered as points in the bigger space. In this way, a metric on a set is said to induce a metric on all of its subsets.
    🔗

    Definition 3.1.6. Induced metric.

    Let \(U \subseteq X \) be a subset of a metric space \(\of{ X , \rho } \text{.}\) The induced metric on \(U \) is the restriction \(\rest{ \rho }_{ U^{ 2 } } \colon U^{ 2 } \to \R \) of \(\rho \) to \(U^{ 2 } \text{.}\) The metric space \(\of{ U , \rest{ \rho }_{ U } } \) is called a metric subspace of the metric space \(\of{ X , \rho } \text{.}\)
    🔗
    In practice, we often write that \(\rho \) is a metric on \(U \text{,}\) even if the domain of \(\rho \) is strictly larger than \(U^{ 2 } \text{.}\) We will not distinguish between a metric and its restriction to a smaller space.
    🔗
    🔗

    Example 3.1.7. Discrete metric subspaces.

    Let \(U \subseteq X \) be a subset of a set \(X \text{.}\) The discrete metric \(\delta_{ U } \) on \(U \) is induced on \(U \) by the discrete metric \(\delta_{ X } \) on \(X \text{;}\) that is, \(\rest{ \delta_{ X } }_{ U^{ 2 } } = \delta_{ U } \text{.}\) Thus the discrete metric space \(\of{ U , \delta_{ U } } \) is a metric subspace of the discrete metric space \(\of{ X , \delta_{ X } } \text{.}\)
    🔗
    🔗
    Thus far, we have been very careful when working in a metric space to specify not only the ground set but also the metric. This is because we can in general find many metrics on a given underlying set. In fact, given one or more metrics on a set, there are several ways of constructing new metrics on that set. In particular, any finite positive pointwise linear combination of metrics is itself a metric.
    🔗

    Proposition 3.1.8. Linear combinations of metrics.

    Any positive real linear combination of metrics is a metric.
    🔗
    🔗

    Proof.
    Let \(X \) be a set, and consider positive real numbers \(\alpha_{ 1 } , \dotsc , \alpha_{ n } \in \opclint{ 0 }{ \infty } \) and metrics \(\rho_{ 1 } , \dotsc , \rho_{ n } \colon X^{ 2 } \to \R \) on \(X \text{.}\) Denote by \(\rho \colon X^{ 2 } \to \R \) the linear combination defined by the formula
    \begin{equation*} \rho \of{ x_{ 1 } , x_{ 2 } } = \sum_{ k = 1 }^{ n }{ \alpha_{ k } \rho_{ k } \of{ x_{ 1 } , x_{ 2 } } }\text{.} \end{equation*}
    First let \(x_{ 1 } , x_{ 2 } \in X \) be points, and suppose that \(\rho \of{ x_{ 1 } , x_{ 2 } } = 0 \text{.}\) Then
    \begin{equation*} \sum_{ k = 1 }^{ n }{ \alpha_{ k } \rho_{ k } \of{ x_{ 1 } , x_{ 2 } } } = \rho \of{ x_{ 1 } , x_{ 2 } } = 0\text{.} \end{equation*}
    Since \(\alpha_{ k } \gt 0 \) and \(\rho_{ k } \of{ x_{ 1 } , x_{ 2 } } \geq 0 \) for all indices \(1 \leq k \leq n \text{,}\) we conclude that \(\rho_{ k } \of{ x_{ 1 } , x_{ 2 } } = 0 \) for all indices \(1 \leq k \leq n \text{.}\) In particular, \(x_{ 1 } = x_{ 2 } \text{.}\)
    🔗
    Conversely, now suppose that \(x_{ 1 } = x_{ 2 } \text{.}\) Then
    \begin{equation*} \rho \of{ x_{ 1 } , x_{ 2 } } = \sum_{ k = 1 }^{ n }{ \alpha_{ k } \rho_{ k } \of{ x_{ 1 } , x_{ 2 } } } = \sum_{ k = 1 }^{ n }{ \alpha_{ k } \cdot 0 } = 0\text{.} \end{equation*}
    Thus \(\rho \) is non-degenerate.
    🔗
    Note that
    \begin{equation*} \rho \of{ x_{ 1 } , x_{ 2 } } = \sum_{ k = 1 }^{ n }{ \alpha_{ k } \rho_{ k } \of{ x_{ 1 } , x_{ 2 } } } = \sum_{ k = 1 }^{ n }{ \alpha_{ k } \rho_{ k } \of{ x_{ 2 } , x_{ 1 } } } = \rho \of{ x_{ 2 } , x_{ 1 } } \end{equation*}
    for all points \(x_{ 1 } , x_{ 2 } \in X \text{,}\) and so \(\rho \) is symmetric.
    🔗
    Finally, note that
    \begin{align*} \rho \of{ x_{ 1 } , x_{ 3 } } \amp = \sum_{ k = 1 }^{ n }{ \alpha_{ k } \rho_{ k } \of{ x_{ 1 } , x_{ 3 } } } \leq \sum_{ k = 1 }^{ n }{ \alpha_{ k } \of{ \rho_{ k } \of{ x_{ 1 } , x_{ 2 } } + \rho_{ k } \of{ x_{ 2 } , x_{ 3 } } } }\\ \amp = \sum_{ k = 1 }^{ n }{ \alpha_{ k } \rho_{ k } \of{ x_{ 1 } , x_{ 2 } } } + \sum_{ k = 1 }^{ n }{ \alpha_{ k } \rho_{ k } \of{ x_{ 2 } , x_{ 3 } } } = \rho \of{ x_{ 1 } , x_{ 2 } } + \rho \of{ x_{ 2 } , x_{ 3 } } \end{align*}
    for all points \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{,}\) and so \(\rho \) satisfies the triangle inequality. Since \(\rho \) is non-degenerate, symmetric, and satisfies the triangle inequality, it is a metric on \(X \text{.}\)
    🔗
    🔗
    In particular, the sum of two metrics is a metric, and any positive real multiple of a metric is a metric.
    🔗
    🔗

    Subsection 3.1.3 Normed Vector Spaces

    The ground sets of many of the metric spaces which we will investigate over the course of this chapter can also be considered as the ground sets of vector spaces. Moreover, in practically all of these cases, the linear-algebraic structure of the vector space is compatible with the analytical structure of the metric, in a way which we will make precise shortly. We now introduce a mathematical structure called a normed vector space, the study of which is called functional analysis and lies at the intersection of linear algebra and mathematical analysis. Aptly named, a normed vector space is a vector space equipped with the additional structure of a norm. Just as metrics generalize the notion of the distance between two points, norms on vector spaces generalize the related notion of the length of a vector.
    🔗

    Definition 3.1.9. Normed vector space.

    A real-valued function \(\norm{ \oparg } \colon V \to \R \) on a real or complex vector space \(V \) is a norm on \(V \) if it has the following properties:
    1. Non-degeneracy.
      For any vector \(\vect{ v } \in V \text{,}\) \(\norm{ \vect{ v } } = 0 \) if and only if \(\vect{ v } = \vect{ 0 } \text{.}\)
      🔗
      🔗
    2. Absolute homogeneity.
      \(\norm{ \alpha \vect{ v } } = \abs{ \alpha } \norm{ \vect{ v } } \) for any scalar \(\alpha \) and vector \(\vect{ v } \in V \text{.}\)
      🔗
      🔗
    3. Sub-additivity.
      \(\norm{ \vect{ v }_{ 1 } + \vect{ v }_{ 2 } } \leq \norm{ \vect{ v }_{ 1 } } + \norm{ \vect{ v }_{ 2 } } \) for all vectors \(\vect{ v }_{ 1 } , \vect{ v }_{ 2 } \in V \text{.}\)
      🔗
      🔗
    In this case, the ordered pair \(\of{ V , \norm{ \oparg } } \) is called a normed vector space.
    🔗
    🔗
    Normed vector spaces are ubiquitous throughout mathematics. In fact, many of the vector spaces which you will have encountered in any course in linear algebra have canonical norms associated to them; lots of the vector spaces which are familiar to you have been normed vector spaces all along.
    🔗

    Example 3.1.10. Normed vector spaces.

    The following are examples of normed vector spaces:
    1. Let \(n \in \N \) be a natural number, and fix a positive real number \(p \gt 0 \text{.}\) The \(p \)-norm \(\norm{ \oparg }_{ p } \colon \R^{ n } \to \R \) on \(n \)-dimensional real coordinate space \(\R^{ n } \) is the real-valued function defined by the formula
      \begin{equation*} \norm{ \vect{ x } }_{ p } = \sqrt{ \sum_{ k = 1 }^{ n }{ x_{ k }^{ 2 } } }\text{.} \end{equation*}
      Similarly, the \(p \)-norm \(\norm{ \oparg }_{ p } \colon \C^{ n } \to \R \) on \(n \)-dimensional complex coordinate space \(\C^{ n } \) is the real-valued function defined by the formula
      \begin{equation*} \norm{ \vect{ z } }_{ p } = \sqrt{ \sum_{ k = 1 }^{ n }{ \abs{ z_{ k } }^{ 2 } } }\text{.} \end{equation*}
      If \(p \geq 1 \text{,}\) then the \(p \)-norms are norms on \(\R^{ n } \) and \(\C^{ n } \text{,}\) and so \(\of{ \R^{ n } , \norm{ \oparg }_{ p } } \) and \(\of{ \C^{ n } , \norm{ \oparg }_{ p } } \) are normed vector spaces.
      🔗
      In particular, the \(2 \)-norms \(\norm{ \oparg }_{ 2 } \) on \(\R^{ n } \) and \(\C^{ n } \) are called the Euclidean norms on \(\R^{ n } \) and \(\C^{ n } \text{.}\) Because the Euclidean norms are so ubiquitous, we will often drop the subscripts in our notation, so that \(\norm{ \oparg } = \norm{ \oparg }_{ 2 } \text{.}\)
      🔗
      🔗
    2. Let \(n \in \N \) be a natural number. The families of \(p \)-norms on \(n \)-dimensional real and complex coordinate space \(\R^{ n } \) and \(\C^{ n } \) can be extended to the case \(p = \infty \text{.}\) The \(\infty \)-norm \(\norm{ \oparg }_{ \infty } \colon \R^{ n } \to \R \) on \(\R^{ n } \) is the real-valued function defined by the formula
      \begin{equation*} \norm{ \vect{ x } }_{ \infty } = \max_{ 1 \leq k \leq n }{ \abs{ x_{ k } } }\text{.} \end{equation*}
      Similarly, the \(\infty \)-norm \(\norm{ \oparg }_{ \infty } \colon \C^{ n } \to \R \) on \(\C^{ n } \) is the real-valued function defined by the formula
      \begin{equation*} \norm{ \vect{ z } }_{ \infty } = \max_{ 1 \leq k \leq n }{ \abs{ z_{ k } } }\text{.} \end{equation*}
      The \(\infty \)-norms are norms on \(\R^{ n } \) and \(\C^{ n } \text{,}\) and so \(\of{ \R^{ n } , \norm{ \oparg }_{ \infty } } \) and \(\of{ \C^{ n } , \norm{ \oparg }_{ \infty } } \) are normed vector spaces.
      🔗
      🔗
    3. Any Hermitian inner product \(\inner{ \oparg }{ \oparg } \) on a real or complex vector space \(V \) induces a norm \(\norm{ \oparg } \colon V^{ 2 } \to \R \) on \(V \) defined by the formula
      \begin{equation*} \norm{ \vect{ v } } = \sqrt{ \inner{ \vect{ v } }{ \vect{ v } } }\text{.} \end{equation*}
      In fact, for all natural numbers \(n \in \N \text{,}\) the norms produced in this manner by the dot product on \(n \)-dimensional real and complex coordinate space \(\R^{ n } \) and \(\C^{ n } \) are precisely the Euclidean norms. For this reason, the dot product is often called the Euclidean inner product on \(\R^{ n } \) and \(\C^{ n } \text{.}\)
      🔗
      🔗
    The above examples show that many of our familiar vector spaces have canonical norms readily available.
    🔗
    🔗

    Remark 3.1.11. \(p \)-norms on \(1 \)-dimensional vector spaces.

    Note that
    \begin{equation*} \norm{ x }_{ p } = \of{ \abs{ x }^{ p } }^{ \frac{ 1 }{ p } } = \abs{ x } = \max \of{ \set{ \abs{ x } } } = \norm{ x }_{ \infty } \end{equation*}
    for all real numbers \(x \in \R \text{,}\) and similarly
    \begin{equation*} \norm{ z }_{ p } = \of{ \abs{ z }^{ p } }^{ \frac{ 1 }{ p } } = \abs{ z } = \max \of{ \set{ \abs{ z } } } = \norm{ z }_{ \infty } \end{equation*}
    for all complex numbers \(z \in \C \text{;}\) that is, all of the \(p \) norms (including the case \(p = \infty \)) on the real numbers \(\R \) and the complex numbers \(\C \) agree. Such statements do not hold in higher dimensions, although we will see that the \(p \)-norms remain related for topological concerns.
    🔗
    🔗
    Just as in the case of metrics, we would prefer to consider only non-negative vector lengths. It makes no sense to consider negative lengths of physical vectors, and it is just as difficult to make sense of negative values of a norm in a normed vector space. Thankfully, the defining axioms of a norm prevent this from occurring.
    🔗

    Lemma 3.1.12. Norms are non-negative.

    A norm takes only non-negative values.
    🔗
    🔗

    Proof.
    Let \(\norm{ \oparg } \colon V \to \R \) be a norm on a real or complex vector space \(V \text{,}\) and note that
    \begin{align*} \norm{ \vect{ v } } \amp = \frac{ 2 \norm{ \vect{ v } } }{ 2 } = \frac{ \norm{ \vect{ v } } + \norm{ \vect{ v } } }{ 2 } = \frac{ \norm{ \vect{ v } } + \abs{ -1 } \norm{ \vect{ v } } }{ 2 }\\ \amp = \frac{ \norm{ \vect{ v } } + \norm{ -\vect{ v } } }{ 2 } \geq \frac{ \norm{ \vect{ v } - \vect{ v } } }{ 2 } = \frac{ \norm{ \vect{ 0 } } }{ 2 } = \frac{ 0 }{ 2 } = 0 \end{align*}
    for all vectors \(\vect{ v } \in V \text{.}\)
    🔗
    🔗

    Theorem 3.1.13. Normed vector spaces are metric spaces.

    A norm \(\norm{ \oparg } \colon V \to \R \) on a real or complex vector space \(V \) induces a metric \(\rho \colon V^{ 2 } \to \R \) defined by the formula
    \begin{equation*} \rho \of{ \vect{ v }_{ 1 } , \vect{ v }_{ 2 } } = \norm{ \vect{ v }_{ 1 } - \vect{ v }_{ 2 } }\text{.} \end{equation*}
    🔗
    🔗

    Proof.
    First, let \(\vect{ v }_{ 1 } , \vect{ v }_{ 2 } \in V \) be vectors, and suppose that \(\rho \of{ \vect{ v }_{ 1 } , \vect{ v }_{ 2 } } = 0 \text{.}\) Then
    \begin{equation*} \norm{ \vect{ v }_{ 1 } - \vect{ v }_{ 2 } } = \rho \of{ \vect{ v }_{ 1 } , \vect{ v }_{ 2 } } = 0\text{,} \end{equation*}
    and so \(\vect{ v }_{ 1 } - \vect{ v }_{ 2 } = \vect{ 0 } \text{;}\) that is, \(\vect{ v }_{ 1 } = \vect{ v }_{ 2 } \text{.}\) Conversely, if \(\vect{ v }_{ 1 } = \vect{ v }_{ 2 } \text{,}\) then
    \begin{equation*} \rho \of{ \vect{ v }_{ 1 } , \vect{ v }_{ 2 } } = \norm{ \vect{ v }_{ 1 } - \vect{ v }_{ 2 } } = \norm{ \vect{ 0 } } = 0\text{.} \end{equation*}
    Thus \(\rho \) is non-degenerate.
    🔗
    Since
    \begin{align*} \rho \of{ \vect{ v }_{ 1 } , \vect{ v }_{ 2 } } \amp = \norm{ \vect{ v }_{ 1 } - \vect{ v }_{ 2 } } = \abs{ -1 } \norm{ \vect{ v }_{ 1 } - \vect{ v }_{ 2 } }\\ \amp = \norm{ \vect{ v }_{ 2 } - \vect{ v }_{ 1 } } = \rho \of{ \vect{ v }_{ 2 } , \vect{ v }_{ 1 } } \end{align*}
    for all vectors \(\vect{ v }_{ 1 } , \vect{ v }_{ 2 } \in V \text{,}\) \(\rho \) is symmetric.
    🔗
    Finally, since
    \begin{align*} \rho \of{ \vect{ v }_{ 1 } , \vect{ v }_{ 3 } } \amp = \norm{ \vect{ v }_{ 1 } - \vect{ v }_{ 3 } } = \norm{ \of{ \vect{ v }_{ 1 } - \vect{ v }_{ 2 } } + \of{ \vect{ v }_{ 2 } - \vect{ v }_{ 3 } } }\\ \amp \leq \norm{ \vect{ v }_{ 1 } - \vect{ v }_{ 2 } } + \norm{ \vect{ v }_{ 2 } - \vect{ v }_{ 3 } } = \rho \of{ \vect{ v }_{ 1 } , \vect{ v }_{ 2 } } + \rho \of{ \vect{ v }_{ 2 } , \vect{ v }_{ 3 } } \end{align*}
    for all vectors \(\vect{ v }_{ 1 } , \vect{ v }_{ 2 } , \vect{ v }_{ 3 } \in V \text{,}\) \(\rho \) satisfies the triangle inequality. Since \(\rho \) is non-degenerate, symmetric, and satisfies the triangle inequality, it is a metric on \(V \text{.}\)
    🔗
    🔗
    So, given a normed vector space, we may easily convert it into a metric space by defining the distance between two vectors to be the norm of their difference. Many important metric spaces arise as a result of this construction.
    🔗

    Example 3.1.14. \(p \)-metrics.

    Let \(n \in \N \) be a natural number, and fix an extended real number \(1 \leq p \leq \infty \text{.}\) The metrics \(d_{ p } \) on \(n \)-dimensional real and complex coordinate spaces \(\R^{ n } \) and \(\C^{ n } \) by the \(p \)-norms \(\norm{ \oparg }_{ p } \) are called \(p \)-metrics. These \(p \)-metrics are defined by the formula
    \begin{equation*} d_{ p } \of{ \vect{ w } , \vect{ z } } = \norm{ \vect{ z }_{ 1 } - \vect{ z }_{ 2 } }_{ p } = \of{ \sum_{ k = 1 }^{ n }{ \abs{ w_{ k } - z_{ k } }^{ p } } }^{ \frac{ 1 }{ p } }\text{.} \end{equation*}
    In particular, we note that the metrics \(d_{ 2 } = d \) induced by the Euclidean norms \(\norm{ \oparg }_{ 2 } = \norm{ \oparg } \) are the Euclidean metrics.
    🔗
    🔗
    So any norm structure on a real or complex vector space can be reanalyzed as also providing a metric structure. However, a metric on a real or complex vector space need not be induced by a norm. Moreover, there exist metric spaces which cannot be constructed from a normed vector space. So, while normed vector spaces are very useful for defining specific metric spaces, they do not capture all of what can be said about metric spaces, and so will not merit any more special consideration just yet. With this illuminating example of normed vector spaces in mind, we now resume our study of metric spaces in their full generality.
