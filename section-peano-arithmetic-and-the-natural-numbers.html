    Section 1.3 Peano Arithmetic and the Natural Numbers

    In this section, we formally introduce the natural number system \(\N \text{.}\) Also called the counting numbers, the natural numbers form the simplest of the number systems; in fact, many people would date the advent of mathematics itself to the invention/discovery of these numbers.
    🔗

    Subsection 1.3.1 Axiomatizing the Natural Numbers

    There are many nonequivalent formalizations and models of the natural number system, but the standard axiomatization is due to Giuseppe Peano in the 19th century. Peano arithmetic is nonconstructive; it gives a description of the natural number system but not a model. We won’t introduce any specific model of the natural number system, but, rest assured, there are several. For our purposes, \(\N \) itself can be any set which conforms to the following characterization:
    🔗

    Definition 1.3.1. Natural numbers.

    The natural number system \(\N \) is a nonempty set with an element \(0 \in \N \text{,}\) called zero, along with functions \(\Succ \colon \N \to \N \) and \(\Add , \Mult \colon \N \times \N \to \N \text{,}\) called the successor function and the addition and multiplication operations and a subset \(\Comp \subseteq \N \times \N \) of the Cartesian product \(\N \times \N \) of \(\N \) with itself, called the standard ordering, which satisfy the following properties:
    1. The successor \(\Succ \of{ n } \) of any natural number \(n \in \N \) is non-zero.
      🔗
      🔗
    2. The successor function \(\Succ \colon \N \to \N \) is injective.
      🔗
      🔗
    3. \(\Add \of{ n , 0 } = n \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    4. \(\Add \of{ m , \Succ \of{ n } } = \Succ \of{ \Add \of{ m , n } } \) for all natural numbers \(m , n \in \N \text{.}\)
      🔗
      🔗
    5. \(\Mult \of{ n , 0 } = 0 \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    6. \(\Mult \of{ m , \Succ \of{ n } } = \Add \of{ \Mult \of{ m , n } , m } \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    7. For all natural numbers \(l , m \in \N \text{,}\) \(\of{ l , m } \in \Comp \) if and only if \(\Add \of{ l , n } = m \) for some natural number \(n \in \N \text{.}\)
      🔗
      🔗
    8. For all subsets \(U \subseteq \N \text{,}\) \(U = \N \) if and only if it satisfies the following conditions:
      1. \(0 \in U \text{.}\)
        🔗
        🔗
      2. For all natural numbers \(n \in \N \text{,}\) if \(n \in U \text{,}\) then \(\Succ \of{ n } \in U \text{.}\)
        🔗
        🔗
      🔗
      🔗
    Typically, for all natural numbers \(m , n \in \N \) we write \(m + n = \Add \of{ m , n } \) and \(m n = m \cdot n = \Mult \of{ m , n } \text{,}\) and we write \(m \leq n \) if and only if \(\of{ m , n } \in \Comp \text{.}\) The successor of \(0 \) is denoted \(1 = \Succ \of{ 0 } \text{.}\) Likewise, the successor of \(1 \) is denoted \(2 = \Succ \of{ 1 } \text{,}\) the successor of \(2 \) is denoted \(3 = \Succ \of{ 2 } \text{,}\) and so on. We see that
    \begin{equation*} \Succ \of{ n } = \Succ \of{ \Add \of{ n , 0 } } = \Add \of{ n , \Succ \of{ 0 } } = n + 1 \end{equation*}
    for all natural numbers \(n \in \N \text{.}\) With these new notations, we may restate the Peano axioms as follows:
    1. \(n + 1 \neq 0 \) for all \(n \in \N \text{.}\)
      🔗
      🔗
    2. For all natural numbers \(m , n \in \N \text{,}\) if \(m + 1 = n + 1 \text{,}\) then \(m = n \text{.}\)
      🔗
      🔗
    3. \(n + 0 = n \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    4. \(m + \of{ n + 1 } = \of{ m + n } + 1 \) for all natural numbers \(m , n \in \N \text{.}\)
      🔗
      🔗
    5. \(n \cdot 0 = 0 \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    6. \(m \of{ n + 1 } = m n + m \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    7. For all natural numbers \(l , m \in \N \text{,}\) \(l \leq m \) if and only if \(l + n = m \) for some natural number \(n \in \N \text{.}\)
      🔗
      🔗
    8. For all subsets \(U \subseteq \N \text{,}\) \(U = \N \) if and only if it satisfies the following conditions:
      1. \(0 \in U \text{.}\)
        🔗
        🔗
      2. For all natural numbers \(n \in \N \text{,}\) if \(n \in U \text{,}\) then \(n + 1 \in U \text{.}\)
        🔗
        🔗
      🔗
      🔗
    🔗
    🔗
    Under the usual (and original) interpretation of the natural number system \(\N \) as characterizing quantities of discrete physical objects, the above rules (with the possible exception of the final axiom, called the axiom of induction) may seem obviously true but also somewhat arbitrary. It is a fact, though we will not prove it here, that these axioms are independent of each other; it is impossible to derive one of the above statements from only the others. We hope that this, along with the results in this section about arithmetic, convince you that the above axiomatization is a good choice, in that it is efficient in the above sense and that it can be used to prove many arithmetical truths.
    🔗
    First, however, we introduce some ubiquitous terminology and notation. A family \(\of{ x_{ n } }_{ n \in \mathcal{ N } } \) of objects \(x_{ n } \) indexed by some subset \(\mathcal{ N } \subseteq \N \) is called a sequence. If the index set \(\mathcal{ N } \) is of the form
    \begin{equation*} \mathcal{ N } = \setmid{ n \in \N }{ a \leq n \textrm{ and } n \leq b } \end{equation*}
    for some natural numbers \(a , b \in \N \text{,}\) then we denote this family by \(\of{ x_{ n } }_{ n = a }^{ b } \text{.}\) Similarly, if the index set \(\mathcal{ N } \) is of the form
    \begin{equation*} \mathcal{ N } = \setmid{ n \in \N }{ a \leq n } \end{equation*}
    for some natural number \(a \in \N \text{,}\) then we denote this family by \(\of{ x_{ n } }_{ n = a }^{ \infty } \text{.}\) These notations also apply to intersections and unions of sequences of sets indexed by subsets of \(\N \) of the above form.
    🔗

    Lemma 1.3.2. \(0 \) is minimal.

    For all natural numbers \(n \in \N \text{,}\) if \(n \leq 0 \text{,}\) then \(n = 0 \text{.}\)
    🔗
    🔗

    Proof.
    We first observe that \(\set{ 0 } \cup \im \of{ \Succ } = \N \text{.}\) Indeed, note that \(0 \in \set{ 0 } \subseteq \set{ 0 } \cup \im \of{ \Succ } \text{.}\) Moreover, if \(n \in \set{ 0 } \cup \im \of{ \Succ } \text{,}\) then
    \begin{equation*} \Succ \of{ n } \in \im \of{ \Succ } \subseteq \set{ 0 } \cup \im \of{ \Succ }\text{.} \end{equation*}
    Thus \(\set{ 0 } \cup \im \of{ \Succ } = \N \) by the axiom of induction.
    🔗
    Let \(n \in \N \) be a natural number, and suppose that \(n \leq 0 \) for some natural number \(n \in \N \text{.}\) Then \(n + m = 0 \) for some natural number \(m \in \N \text{.}\) Suppose for a contradiction that \(m = p + 1 \) for some natural number \(p \in \N \text{.}\) Then
    \begin{equation*} 0 = n + m = n + \of{ p + 1 } = \of{ n + p } + 1\text{,} \end{equation*}
    which contradicts the first Peano axiom. Thus \(m \notin \im \of{ \Succ } \text{.}\) However, \(m \in \N = \set{ 0 } \cup \im \of{ \Succ } \text{,}\) and so \(m \in \set{ 0 } \text{;}\) that is, \(m = 0 \text{,}\) and so
    \begin{equation*} n = n + 0 = n + m = 0\text{.} \end{equation*}
    🔗
    🔗

    Proposition 1.3.3. Principle of strong induction.

    For all subsets \(U \subseteq \N \text{,}\) \(U = \N \) if and only if it satisfies the following conditions:
    1. \(0 \in U \text{.}\)
      🔗
      🔗
    2. For all natural numbers \(n \in \N \text{,}\) if \(k \in U \) for all \(k \in \N \) so that \(k \leq n \text{,}\) then \(n + 1 \in U \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.
    Clearly \(\N \) satisfies the above conditions, and so it suffices to show that any subsets which satisfies these conditions is equal to \(\N \text{.}\) To that end, let \(U \subseteq \N \) be a subset which satisfies the above conditions, and consider the subset
    \begin{equation*} V = \setmid{ n \in \N }{ k \in U \textrm{ for all } k \in \N \textrm{ so that } k \leq n }\text{.} \end{equation*}
    Since the only \(k \in \N \) so that \(k \leq 0 \) is \(k = 0 \) itself, and \(0 \in U \text{,}\) we have that \(0 \in V \text{.}\) Moreover if \(n \in V \text{,}\) then \(k \in U \) for all \(k \in \N \) so that \(k \leq n \text{,}\) and so \(n + 1 \in U \) by hypothesis. It remains to conclude that \(k \in U \) for all \(k \in \N \) so that \(k \leq n + 1 \text{.}\)
    🔗
    To that end, let \(k \in \N \text{,}\) and suppose that \(k \leq n + 1 \text{.}\) Then \(k + m = n + 1 \) for some natural number \(m \in \N \text{.}\) If \(m = 0 \text{,}\) then \(k = n + 1 \in U \text{.}\) On the other hand, if \(m \neq 0 \text{,}\) then the argument in the proof of Lemma 1.3.2 \(0 \) is minimal implies that \(m = p + 1 \) for some natural number \(p \in \N \text{.}\) Thus
    \begin{equation*} \of{ k + p } + 1 = k + \of{ p + 1 } = k + m = n + 1\text{,} \end{equation*}
    and so the injectivity of the successor function implies that \(k + p = n \text{;}\) that is, \(k \leq n \) and so \(k \in U \text{.}\) Since \(k \in U \) for all \(k \in \N \) so that \(k \leq n + 1 \text{,}\) we conclude that \(n + 1 \in V \text{.}\)
    🔗
    In summary, we have shown that \(0 \in V \) and that for all natural numbers \(n \in \N \text{,}\) if \(n \in V \text{,}\) then \(n + 1 \in V \text{.}\) Thus \(V = \N \) by the axiom of induction. One can check that \(\N = V \subseteq U \subseteq \N \text{,}\) and so \(U = \N \text{.}\)
    🔗
    🔗
    The axiom of induction and Proposition 1.3.3 Principle of strong induction form the basis for ubiquitous methods of proof called proof by induction and proof by strong induction, respectively. You will have the opportunity to practice constructing such proofs in the problems for this section.
    🔗
    🔗

    Subsection 1.3.2 Properties of Arithmetic

    The axioms of Peano arithmetic actually imply a great deal about the properties of the natural number system. We will now state and prove a great deal of results about the successor function and the addition and multiplication operators. Many of these results will give us practice in the fundamental skill of proof by induction, and we will use these results implicitly for the rest of this text.
    🔗

    Proposition 1.3.4. Associativity of addition.

    \(\of{ l + m } + n = l + \of{ m + n } \) for all natural numbers \(l , m , n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    Let \(l , m \in \N \) be natural numbers. We proceed by induction on \(n \in \N \) to show that \(\of{ l + m } + n = l + \of{ m + n } \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} \of{ l + m } + n = \of{ l + m } + 0 = l + m = l + \of{ m + 0 } = l + \of{ m + n }\text{.} \end{equation*}
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(\of{ l + m } + n = l + \of{ m + n } \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{align*} \of{ l + m } + \of{ n + 1 } \amp = \of{ \of{ l + m } + n } + 1 = \of{ \ell + \of{ m + n } } + 1\\ \amp = \ell + \of{ \of{ m + n } + 1 } = \ell + \of{ m + \of{ n + 1 } }\text{.} \end{align*}
    🔗
    🔗
    We conclude that \(\of{ l + m } + n = l + \of{ m + n } \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
    One consequence of Proposition 1.3.4 Associativity of addition is that we may unambiguously parse expressions with multiple sums without the need of parentheses; for example,if \(l , m , n \in \N \) are natural numbers, then the expression \(l + m + n \) is well-defined, because
    \begin{equation*} \of{ l + m } + n = l + \of{ m + n }\text{.} \end{equation*}
    Going forward, we will make use of these expressions wherever it is possible to do so without obscuring the relevant arguments.
    🔗
    Proposition 1.3.4 Associativity of addition implies that the result of adding a finite number of natural numbers depends only on the order of the numbers from left to right, and not on the iterative structure of the summands induced by parentheses. In fact, the order of the summands does not matter either. In order to show this, it helps to notice the following two results about addition.
    🔗

    Lemma 1.3.5. Addition by \(0 \).

    \(n + 0 = 0 + n = n \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    \(n + 0 = n \) for all natural numbers \(n \in \N \) by the Peano axioms, and so it suffices to show \(0 + n = n \text{.}\) We proceed by induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} 0 + n = 0 + 0 = 0 = n\text{.} \end{equation*}
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(0 + n = n \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{equation*} 0 + \of{ n + 1 } = \of{ 0 + n } + 1 = n + 1\text{.} \end{equation*}
    🔗
    🔗
    We conclude that \(0 + n = n \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Lemma 1.3.6. Towards commutativity of addition.

    \(1 + n = n + 1 \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    By induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} 1 + n = 1 + 0 = 0 + 1 = n + 1 \end{equation*}
    by Lemma 1.3.5 Addition by \(0 \).
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(1 + n = n + 1 \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{equation*} 1 + \of{ n + 1 } = \of{ 1 + n } + 1 = \of{ n + 1 } + 1 \end{equation*}
    by Proposition 1.3.4 Associativity of addition.
    🔗
    🔗
    We conclude that \(1 + n = n + 1 \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Proposition 1.3.7. Commutativity of addition.

    \(m + n = n + m \) for all natural numbers \(m , n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    Let \(m \in \N \) be a natural number. We proceed by induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} m + n = m + 0 = m = 0 + m = n + m \end{equation*}
    by Lemma 1.3.5 Addition by \(0 \).
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(m + n = n + m \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{align*} m + \of{ n + 1 } \amp = \of{ m + n } + 1 = \of{ n + m } + 1 = 1 + \of{ n + m }\\ \amp = \of{ 1 + n } + m = \of{ n + 1 } + m \end{align*}
    by Proposition 1.3.4 Associativity of addition and Lemma 1.3.6 Towards commutativity of addition.
    🔗
    🔗
    We conclude that \(m + n = n + m \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
    Proposition 1.3.4 Associativity of addition and Proposition 1.3.7 Commutativity of addition together imply that, when adding natural numbers, neither the order of the summands nor the order of the addition operations performed matters. In order to see that analogous properties also hold for multiplication, we will require the following supporting results. Along the way, we will also prove the distributivity of multiplication over addition.
    🔗

    Lemma 1.3.8. Multiplication by \(1 \).

    \(n \cdot 1 = n \) and \(1 n = n \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    For all natural numbers \(n \in \N \text{,}\)
    \begin{equation*} n \cdot 1 = n \of{ 0 + 1 } = \of{ n \cdot 0 } + n = 0 + n = 0 \end{equation*}
    by Lemma 1.3.5 Addition by \(0 \). Thus it suffices to prove that \(1 n = n \) for all natural numbers \(n \in \N \text{.}\) We proceed by induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} 1 n = 1 \cdot 0 = 0 = n\text{.} \end{equation*}
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(1 n = n \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{equation*} 1 \of{ n + 1 } = 1 n + 1 = n + 1\text{.} \end{equation*}
    🔗
    🔗
    We conclude that \(1 n = n \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Proposition 1.3.9. Left distributivity of multiplication over addition.

    \(l \of{ m + n } = l m + l n \) for all natural numbers \(l , m , n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    Let \(l , m \in \N \) be natural numbers. We proceed by induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} l \of{ m + n } = l \of{ m + 0 } = l m = l m + 0 = l m + l \cdot 0 = l m + l n\text{.} \end{equation*}
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(l \of{ m + n } = l m + l n \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{align*} l \of{ m + \of{ n + 1 } } \amp = l \of{ \of{ m + n } + 1 } = l \of{ m + n } + l\\ \amp = \of{ l m + l n } + l = l m + \of{ l n + l } = l m + l \of{ n + 1 } \end{align*}
    by Proposition 1.3.4 Associativity of addition.
    🔗
    🔗
    We conclude that \(l \of{ m + n } = l m + l n \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Proposition 1.3.10. Associativity of multiplication.

    \(\of{ l m } n = l \of{ m n } \) for all natural numbers \(l , m , n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    Let \(l , m \in \N \) be natural numbers. We proceed by induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} \of{ l m } n = \of{ l m } 0 = 0 = l \cdot 0 = l \of{ m \cdot 0 } = l \of{ m n }\text{.} \end{equation*}
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(\of{ l m } n = l \of{ m n } \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{align*} \of{ l m } \of{ n + 1 } \amp = \of{ l m } n + l m = l \of{ m n } + l m = \\ \amp = l \of{ m n + m } = l \of{ m \of{ n + 1 } } \end{align*}
    by Proposition 1.3.9 Left distributivity of multiplication over addition.
    🔗
    🔗
    We conclude that \(\of{ l m } n = l \of{ m n } \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
    Just as with addition, one consequence of Proposition 1.3.10 Associativity of multiplication is that we may unambiguously parse expressions with multiple products without the need of parentheses; for example,if \(l , m , n \in \N \) are natural numbers, then the expression \(l m n \) is well-defined, because
    \begin{equation*} \of{ l m } n = l \of{ m n }\text{.} \end{equation*}
    Going forward, we will make use of these expressions wherever it is possible to do so without obscuring the relevant arguments.
    🔗

    Proposition 1.3.11. Right distributivity of multiplication over addition.

    \(\of{ l + m } n = l n + m n \) for all natural numbers \(l , m , n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    Let \(l , m \in \N \) be natural numbers. We proceed by induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} \of{ l + m } n = \of{ l + m } \cdot 0 = 0 = 0 + 0 = l \cdot 0 + m \cdot 0 = l n + m n\text{.} \end{equation*}
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(\of{ l + m } n = l n + m n \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{align*} \of{ l + m } \of{ n + 1 } = \of{ l + m } n + \of{ l + m } = l n + l + m n + m \\ \amp = l \of{ n + 1 } + m \of{ n + 1 } \end{align*}
    by repeated applications of Proposition 1.3.4 Associativity of addition and Proposition 1.3.7 Commutativity of addition.
    🔗
    🔗
    We conclude that \(\of{ l + m } n = l n + m n \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Lemma 1.3.12. Multiplication by \(0 \).

    \(0 n = 0 \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    By induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} 0 n = 0 \cdot 0 = 0\text{.} \end{equation*}
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(0 n = 0 \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{equation*} 0 \of{ n + 1 } = 0 n + 0 = 0 + 0 = 0\text{.} \end{equation*}
    🔗
    🔗
    We conclude that \(0 n = 0 \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Proposition 1.3.13. Commutativity of multiplication.

    \(m n = n m \) for all natural numbers \(m , n \in \N \text{.}\)
    🔗
    🔗

    Proof.
    Let \(m \in \N \) be a natural number. We proceed by induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then
    \begin{equation*} m n = m \cdot 0 = 0 = 0 \cdot m = n m \end{equation*}
    by Lemma 1.3.12 Multiplication by \(0 \).
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that \(m n = n m \) for some natural number \(n \in \N \text{,}\) and note that
    \begin{equation*} m \of{ n + 1 } = m n + m = n m + 1 m = \of{ n + 1 } m \end{equation*}
    by Lemma 1.3.8 Multiplication by \(1 \)
    🔗
    🔗
    We conclude that \(m n = n m \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗

    Lemma 1.3.14. Cancellation of addition.

    For all natural numbers \(n \in \N \text{,}\) addition by \(n \) is injective; that is, for all natural numbers \(l , m , n \in \N \text{,}\) if \(l + n = m + n \text{,}\) then \(l = m \text{.}\)
    🔗
    🔗

    Proof.
    Let \(l , m \in \N \) be natural numbers. We proceed by induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \) and \(l + n = m + n \text{,}\) then
    \begin{equation*} l = l + 0 = l + n = m + n = m + 0 = m \end{equation*}
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that there is some natural number \(n \in \N \) so that if \(l + n = m + n \text{,}\) then \(l = m \text{.}\) Note that if \(l + \of{ n + 1 } = m + \of{ n + 1 } \text{,}\) then
    \begin{equation*} \of{ l + n } + 1 = l + \of{ n + 1 } = m + \of{ n + 1 } = \of{ m + n } + 1\text{,} \end{equation*}
    and so \(l + n = m + n \) by the injectivity of the successor function. Thus \(l = m \) by the inductive hypothesis. In summary, we have shown that if \(l + \of{ n + 1 } = m + \of{ n + 1 } \text{,}\) then \(l = m \text{.}\)
    🔗
    🔗
    We conclude that for all natural numbers \(n \in \N \text{,}\) if \(l + n = m + n \text{,}\) then \(l = m \text{.}\)
    🔗
    🔗
    A similar cancellation law exists for multiplication, but its proof will require some properties of the comparison of natural numbers via its standard ordering.
    🔗
    🔗

    Subsection 1.3.3 Properties of Comparison

    The final component of Peano’s axiomatization of the natural numbers \(\N \) is that of comparison. The standard ordering \(\leq \) on the natural numbers has several important and familiar properties which we will now derive from first principles.
    🔗

    Proposition 1.3.15. Comparison is a total order.

    Comparison of natural numbers has the following properties:
    1. Reflexivity.
      \(n \leq n \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    2. Antisymmetry.
      For all natural numbers \(m , n \in \N \text{,}\) if \(m \leq n \) and \(n \geq m \text{,}\) then \(m = n \text{.}\)
      🔗
      🔗
    3. Transitivity.
      For all natural numbers \(l , m , n \in \N \text{,}\) if \(l \leq m \) and \(m \leq n \text{,}\) then \(l \leq n \text{.}\)
      🔗
      🔗
    4. Totality.
      For all natural numbers \(m , n \in \N \text{,}\) either \(m \leq n \) or \(n \leq m \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.
    1. For all natural numbers \(n \in \N \text{,}\) \(n + 0 = n \text{,}\) so that \(n \leq n \text{.}\)
      🔗
      🔗
    2. Let \(m , n \in \N \) be natural numbers, and suppose that \(m \leq n \) and \(n \leq m \text{.}\) Then \(n = m + p \) and \(m = n + q \) for some natural numbers \(p , q \in \N \text{.}\) Note that
      \begin{equation*} m + 0 = m = n + q = m + p + q\text{,} \end{equation*}
      and so \(0 = p + q \) by Lemma 1.3.14 Cancellation of addition; that is, \(q \leq 0 \text{.}\) Lemma 1.3.2 \(0 \) is minimal now implies that \(q = 0 \text{,}\) so that
      \begin{equation*} m = n + q = n + 0 = n\text{.} \end{equation*}
      🔗
      🔗
    3. Let \(l , m , n \in \N \) be natural numbers, and suppose that \(l \leq m \) and \(m \leq n \text{.}\) Then \(l + p = m \) and \(m + q = n \) for some natural numbers \(p , q \in \N \text{,}\) and so
      \begin{equation*} l + \of{ p + q } = \of{ l + p } + q = m + q = n\text{.} \end{equation*}
      We conclude that \(l \leq n \text{.}\)
      🔗
      🔗
    4. Let \(m \in \N \) be a natural number. We proceed by induction on \(n \in \N \text{.}\)
      Base case.
      If \(n = 0 \text{,}\) then \(0 + m = m \) by Lemma 1.3.5 Addition by \(0 \), and so \(n = 0 \leq m \text{.}\)
      🔗
      🔗
      Inductive step.
      Now suppose for the inductive hypothesis that there is a natural number \(n \in \N \) so that either \(m \leq n \) or \(n \leq m \text{.}\) If \(m \leq n \text{,}\) then \(m + p = n \) for some natural number \(p \in \N \text{,}\) and so
      \begin{equation*} m + \of{ p + 1 } = \of{ m + p } + 1 = n + 1\text{;} \end{equation*}
      that is, \(m \leq n + 1 \text{.}\)
      🔗
      Now suppose that \(n \leq m \text{,}\) so that \(n + p = m \) for some natural number \(p \in \N \text{.}\) If \(p = 0 \text{,}\) then
      \begin{equation*} m + 1 = \of{ n + p } + 1 = \of{ n + 0 } + 1 = n + 1\text{,} \end{equation*}
      and so \(m \leq n + 1 \text{.}\) On the other hand, if \(p \neq 0 \text{,}\) then \(p = q + 1 \) for some natural number \(q \in \N \) by an argument in the proof of Lemma 1.3.2 \(0 \) is minimal. Then
      \begin{equation*} \of{ n + 1 } + q = n + 1 + q = n + q + 1 = n + p = m\text{,} \end{equation*}
      and so \(n + 1 \leq m \text{.}\)
      🔗
      🔗
      We conclude that for all natural numbers \(n \in \N \text{,}\) either \(m \leq n \) or \(n \leq m \text{.}\)
      🔗
      🔗
    🔗

    Corollary 1.3.16. Cancellation of multiplication.

    For all natural numbers \(n \in \N \text{,}\) if \(n \neq 0 \text{,}\) then addition by \(n \) is injective; that is, for all natural numbers \(l , m , n \in \N \text{,}\) if \(l n = m n \text{,}\) then either \(l = m \) or \(n = 0 \text{.}\)
    🔗
    🔗

    Proof.
    By induction on \(n \in \N \text{.}\)
    Base case.
    If \(n = 0 \text{,}\) then \(n = 0 \text{,}\) and so the desired conclusion holds for all natural numbers \(l , m \in \N \text{.}\)
    🔗
    🔗
    Inductive step.
    Now suppose for the inductive hypothesis that there is a natural number \(n \in \N \) so that for all natural numbers \(l , m \in \N \text{,}\) if \(l n = m n \text{,}\) then either \(n = 0 \) or \(l = m \text{.}\) Let \(l , m \in \N \) be natural numbers, and suppose that \(l \of{ n + 1 } = m \of{ n + 1 } \text{.}\) Then \(n + 1 \neq 0 \text{,}\) and so it suffices to show that \(l = m \text{.}\)
    🔗
    To that end, note that (4) of Proposition 1.3.15 Comparison is a total order implies that either \(l \leq m \) or \(m \leq l \text{.}\) Without loss of generality, we may suppose that \(l \leq m \text{,}\) so that \(m = l + p \) for some natural number \(p \in \N \text{.}\) We observe that
    \begin{align*} 0 + \of{ l n + l } \amp = l n + l = l \of{ n + 1 } = m \of{ n + 1 }\\ \amp = \of{ l + p } \of{ n + 1 } = l n + l + p n + p = \of{ p n + p } + \of{ l n + l } \end{align*}
    by Lemma 1.3.5 Addition by \(0 \), Proposition 1.3.9 Left distributivity of multiplication over addition, Proposition 1.3.11 Right distributivity of multiplication over addition, Proposition 1.3.4 Associativity of addition, and Proposition 1.3.7 Commutativity of addition.
    🔗
    In particular, Lemma 1.3.14 Cancellation of addition implies that \(0 = p n + p \text{,}\) so that \(p \leq 0 \text{.}\) We conclude by Lemma 1.3.2 \(0 \) is minimal that \(p = 0 \text{,}\) so that
    \begin{equation*} l = m + p = m + 0 = m\text{.} \end{equation*}
    🔗
    🔗
    We conclude that for all natural numbers \(l , m , n \in \N \text{,}\) either if \(l n = m n \text{,}\) then either \(l = m \) or \(n = 0 \text{.}\)
    🔗
    🔗

    Lemma 1.3.17. Comparison and arithmetic.

    Let \(l , m , n , p \in \N \) be natural numbers.
    1. If \(l \leq n \) and \(m \leq p \text{,}\) then \(l + m \leq n + p \text{.}\)
      🔗
      🔗
    2. If \(l \leq n \) and \(m \leq p \text{,}\) then \(l m \leq n p \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.
    1. If \(l \leq n \) and \(m \leq p \text{,}\) then \(n = l + q \) and \(p = m + r \) for some natural numbers \(q , r \in \N \text{,}\) and so
      \begin{equation*} \of{ l + m } + \of{ q + r } = l + m + q + r = l + q + m + r = n + p\text{.} \end{equation*}
      We conclude that \(l + m \leq n + p \text{.}\)
      🔗
      🔗
    2. If \(l \leq n \) and \(m \leq p \text{,}\) then \(n = l + q \) and \(p = m + r \) for some natural numbers \(q , r \in \N \text{,}\) and so
      \begin{align*} l m + \of{ l r + q m + q r } \amp = l m + l r + q m + q r = l \of{ m + r } + q \of{ m + r }\\ \amp = \of{ l + q } \of{ m + r } = n p\text{.} \end{align*}
      We conclude that \(l m \leq n p \text{.}\)
      🔗
      🔗
    🔗
    For ease of reference, we collect the above results (along with some immediate corollaries) in the following theorem.
    🔗

    Theorem 1.3.18. Properties of Peano arithmetic.

    Addition, multiplication, and comparison of natural numbers have the following properties:
    1. Associativity of addition.
      \(\of{ l + m } + n = l + \of{ m + n } \) for all natural numbers \(l , m , n \in \N \text{.}\)
      🔗
      🔗
    2. Commutativity of addition.
      \(m + n = n + m \) for all natural numbers \(m , n \in \N \text{.}\)
      🔗
      🔗
    3. Additive identity.
      \(n + 0 = 0 + n = 0 \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    4. Associativity of multiplication.
      \(\of{ l m } n = l \of{ m n } \) for all natural numbers \(l , m , n \in \N \text{.}\)
      🔗
      🔗
    5. Commutativity of multiplication.
      \(m n = n m \) for all natural numbers \(m , n \in \N \text{.}\)
      🔗
      🔗
    6. Multiplicative annihilation.
      \(n \cdot 0 = 0 n = 0 \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    7. Multiplicative identity.
      \(n \cdot 1 = 1 n = n \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    8. Distributivity of multiplication over addition.
      \(l \of{ m + n } = l m + l n \) and \(\of{ l + m } n = l n + m n \) for all natural numbers \(l , m , n \in \N \text{.}\)
      🔗
      🔗
    9. Reflexivity of comparison.
      \(n \leq n \) for all natural numbers \(n \in \N \text{.}\)
      🔗
      🔗
    10. Antisymmetry of comparison.
      For all natural numbers \(m , n \in \N \text{,}\) if \(m \leq n \) and \(n \leq m \text{,}\) then \(m = n \text{.}\)
      🔗
      🔗
    11. Transitivity of comparison.
      For all natural numbers \(l , m , n \in \N \text{,}\) if \(l \leq m \) and \(m \leq n \text{,}\) then \(l \leq n \text{.}\)
      🔗
      🔗
    12. Totality of comparison.
      For all natural numbers \(m , n \in \N \text{,}\) either \(m \leq n \) or \(n \leq m \text{.}\)
      🔗
      🔗
    13. Compatibility of comparison and addition/multiplication.
      For all natural numbers \(l , m , n , p \in \N \text{,}\) if \(l \leq n \) and \(m \leq p \text{,}\) then
      \begin{equation*} l + m \leq n + p \end{equation*}
      and
      \begin{equation*} l m \leq n p \end{equation*}
      🔗
      🔗
    🔗
    🔗
    Many of these properties of the arithmetic and comparison of natural numbers also extend to the arithmetic of the other classes of numbers we will consider, namely the integers, rational numbers, real numbers, and complex numbers.
    🔗
    🔗
    In the next sections, we will generalize the properties of addition and multiplication and the order-theoretic properties of comparison of natural numbers and arrive at fundamental notions of binar operations and relations, partial and total orders, and the well-ordering principle.
