Definition 1.5.8. Extremal element.

Let \(U \subseteq X \) be a subset of a partially ordered set \(\of{ X , \preceq } \text{.}\)
  1. An element \(u \in U \) is called maximal in \(U \) if for all elements \(v \in U \text{,}\) if \(u \preceq v \text{,}\) then \(v = u \text{.}\)
    🔗
    🔗
  2. An element \(u \in U \) is called minimal in \(U \) if for all elements \(v \in U \text{,}\) if \(v \preceq u \text{,}\) then \(v = u \text{.}\)
    🔗
    🔗
Together, maximal and minimal elements are called extremal elements.
