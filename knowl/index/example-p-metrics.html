Example 3.1.14. \(p \)-metrics.

Let \(n \in \N \) be a natural number, and fix an extended real number \(1 \leq p \leq \infty \text{.}\) The metrics \(d_{ p } \) on \(n \)-dimensional real and complex coordinate spaces \(\R^{ n } \) and \(\C^{ n } \) by the \(p \)-norms \(\norm{ \oparg }_{ p } \) are called \(p \)-metrics. These \(p \)-metrics are defined by the formula
\begin{equation*} d_{ p } \of{ \vect{ w } , \vect{ z } } = \norm{ \vect{ z }_{ 1 } - \vect{ z }_{ 2 } }_{ p } = \of{ \sum_{ k = 1 }^{ n }{ \abs{ w_{ k } - z_{ k } }^{ p } } }^{ \frac{ 1 }{ p } }\text{.} \end{equation*}
In particular, we note that the metrics \(d_{ 2 } = d \) induced by the Euclidean norms \(\norm{ \oparg }_{ 2 } = \norm{ \oparg } \) are the Euclidean metrics.
