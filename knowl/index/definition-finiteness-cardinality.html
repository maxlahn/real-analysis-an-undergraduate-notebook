Definition 1.6.3. Finiteness; cardinality.

A set \(X \) is called finite if there is a natural number \(n \in \N \) and a bijection \(\set{ 1 , \dotsc , n } \to X \text{,}\) in which case \(n \) is uniquely defined by this property and is called the cardinality of \(X \) and denoted \(\card{ X } = n \text{.}\) A set \(X \) is called infinite if no such number and bijection exist.
