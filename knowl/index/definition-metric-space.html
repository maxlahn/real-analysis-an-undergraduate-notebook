Definition 3.1.1. Metric space.

A real-valued function \(\rho \colon X^{ 2 } \to \R \) on the Cartesian square \(X^{ 2 } \) of a set \(X \text{,}\) which is called the ground set and whose elements are called points, is a metric on \(X \) if it has the following properties:
  1. Non-degeneracy.
    For any points \(x_{ 1 } , x_{ 2 } \in X \text{,}\) \(\rho \of{ x_{ 1 } , x_{ 2 } } = 0 \) if and only if \(x_{ 1 } = x_{ 2 } \text{.}\)
    🔗
    🔗
  2. Symmetry.
    \(\rho \of{ x_{ 1 } , x_{ 2 } } = \rho \of{ x_{ 2 } , x_{ 1 } } \) for any points \(x_{ 1 } , x_{ 2 } \in X \text{.}\)
    🔗
    🔗
  3. Triangle inequality.
    \(\rho \of{ x_{ 1 } , x_{ 3 } } \leq \rho \of{ x_{ 1 } , x_{ 2 } } + \rho \of{ x_{ 2 } , x_{ 3 } } \) for any points \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{.}\)
    🔗
    🔗
In this case, the ordered pair \(\of{ X , \rho } \) is called a metric space.
