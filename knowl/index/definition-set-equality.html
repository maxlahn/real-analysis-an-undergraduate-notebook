Definition 1.1.6. Set equality.

Two sets are considered equal if they contain exactly the same elements; that is, one set \(A \) is equal to another set \(B \) when objects are elements of \(A \) if and only if they are elements of \(B \text{.}\) This occurs precisely when both \(A \subseteq B \) and \(B \subseteq A \text{.}\)
