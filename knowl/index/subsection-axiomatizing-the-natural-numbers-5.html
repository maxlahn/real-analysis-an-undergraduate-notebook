First, however, we introduce some ubiquitous terminology and notation. A family \(\of{ x_{ n } }_{ n \in \mathcal{ N } } \) of objects \(x_{ n } \) indexed by some subset \(\mathcal{ N } \subseteq \N \) is called a sequence. If the index set \(\mathcal{ N } \) is of the form
\begin{equation*} \mathcal{ N } = \setmid{ n \in \N }{ a \leq n \textrm{ and } n \leq b } \end{equation*}
for some natural numbers \(a , b \in \N \text{,}\) then we denote this family by \(\of{ x_{ n } }_{ n = a }^{ b } \text{.}\) Similarly, if the index set \(\mathcal{ N } \) is of the form
\begin{equation*} \mathcal{ N } = \setmid{ n \in \N }{ a \leq n } \end{equation*}
for some natural number \(a \in \N \text{,}\) then we denote this family by \(\of{ x_{ n } }_{ n = a }^{ \infty } \text{.}\) These notations also apply to intersections and unions of sequences of sets indexed by subsets of \(\N \) of the above form.
