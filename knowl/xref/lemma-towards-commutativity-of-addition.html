Lemma 1.3.6. Towards commutativity of addition.

\(1 + n = n + 1 \) for all natural numbers \(n \in \N \text{.}\)
🔗
🔗

Proof.
By induction on \(n \in \N \text{.}\)

Base case.

If \(n = 0 \text{,}\) then
\begin{equation*} 1 + n = 1 + 0 = 0 + 1 = n + 1 \end{equation*}
by Lemma 1.3.5 Addition by \(0 \).
🔗
🔗

Inductive step.

Now suppose for the inductive hypothesis that \(1 + n = n + 1 \) for some natural number \(n \in \N \text{,}\) and note that
\begin{equation*} 1 + \of{ n + 1 } = \of{ 1 + n } + 1 = \of{ n + 1 } + 1 \end{equation*}
by Proposition 1.3.4 Associativity of addition.
🔗
🔗
We conclude that \(1 + n = n + 1 \) for all natural numbers \(n \in \N \text{.}\)
