Lemma 1.3.8. Multiplication by \(1 \).

\(n \cdot 1 = n \) and \(1 n = n \) for all natural numbers \(n \in \N \text{.}\)
🔗
🔗

Proof.
For all natural numbers \(n \in \N \text{,}\)
\begin{equation*} n \cdot 1 = n \of{ 0 + 1 } = \of{ n \cdot 0 } + n = 0 + n = 0 \end{equation*}
by Lemma 1.3.5 Addition by \(0 \). Thus it suffices to prove that \(1 n = n \) for all natural numbers \(n \in \N \text{.}\) We proceed by induction on \(n \in \N \text{.}\)

Base case.

If \(n = 0 \text{,}\) then
\begin{equation*} 1 n = 1 \cdot 0 = 0 = n\text{.} \end{equation*}
🔗
🔗

Inductive step.

Now suppose for the inductive hypothesis that \(1 n = n \) for some natural number \(n \in \N \text{,}\) and note that
\begin{equation*} 1 \of{ n + 1 } = 1 n + 1 = n + 1\text{.} \end{equation*}
🔗
🔗
We conclude that \(1 n = n \) for all natural numbers \(n \in \N \text{.}\)
