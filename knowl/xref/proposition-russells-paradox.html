Proposition 1.1.4. Russell’s paradox.

Not all definable collections are sets.
🔗
🔗

Proof.
By contradiction. Let
\begin{equation*} \mathfrak{ R } = \setmid{ S }{ S \notin S } \end{equation*}
be the definable collection of sets which do not contain themselves, and suppose for a contradiction that \(\mathfrak{ R } \) is a set. The question of whether or not \(\mathfrak{ R } \) contains itself is a natural one; however, we will see that both the positive and negative answers lead to contradictions.
🔗
This is because, if \(\mathfrak{ R } \in \mathfrak{ R } \text{,}\) then \(\mathfrak{ R } \notin \mathfrak{ R } \) by the definition of \(\mathfrak{ R } \text{.}\) On the other hand, if \(\mathfrak{ R } \notin \mathfrak{ R } \text{,}\) then \(\mathfrak{ R } \in \mathfrak{ R } \) by the same reasoning. Therefore, \(\mathfrak{ R } \in \mathfrak{ R } \) if and only if \(\mathfrak{ R } \notin \mathfrak{ R } \text{.}\) This contradiction implies that \(\mathfrak{ R } \) is not a set, as desired.
