Proposition 1.3.4. Associativity of addition.

\(\of{ l + m } + n = l + \of{ m + n } \) for all natural numbers \(l , m , n \in \N \text{.}\)
🔗
🔗

Proof.
Let \(l , m \in \N \) be natural numbers. We proceed by induction on \(n \in \N \) to show that \(\of{ l + m } + n = l + \of{ m + n } \text{.}\)

Base case.

If \(n = 0 \text{,}\) then
\begin{equation*} \of{ l + m } + n = \of{ l + m } + 0 = l + m = l + \of{ m + 0 } = l + \of{ m + n }\text{.} \end{equation*}
🔗
🔗

Inductive step.

Now suppose for the inductive hypothesis that \(\of{ l + m } + n = l + \of{ m + n } \) for some natural number \(n \in \N \text{,}\) and note that
\begin{align*} \of{ l + m } + \of{ n + 1 } \amp = \of{ \of{ l + m } + n } + 1 = \of{ \ell + \of{ m + n } } + 1\\ \amp = \ell + \of{ \of{ m + n } + 1 } = \ell + \of{ m + \of{ n + 1 } }\text{.} \end{align*}
🔗
🔗
We conclude that \(\of{ l + m } + n = l + \of{ m + n } \) for all natural numbers \(n \in \N \text{.}\)
