Definition 1.4.5. Cartesian power.

Let \(X \) be a set and \(m \in \N \) be a natural number. The \(m \)th Cartesian power \(X^{ m } \) of \(X \) is the Cartesian product
\begin{equation*} X^{ m } = \prod_{ j = 1 }^{ m }{ X } = \setmid{ \of{ x_{ j } }_{ j = 1 }^{ m } }{ x_{ j } \in X \textrm{ for all } 1 \leq j \leq m }\text{.} \end{equation*}
