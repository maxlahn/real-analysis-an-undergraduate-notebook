Item 3.

The discrete metric \(\delta_{ X } \colon X^{ 2 } \to \set{ 0 , 1 } \) on a set \(X \) is the function defined by the formula
\begin{equation*} \delta_{ X } \of{ x_{ 1 } , x_{ 2 } } = \begin{cases} 1 \amp x_{ 1 } = x_{ 2 } \\ 0 \amp x_{ 1 } \neq x_{ 2 } \text{.}\end{cases} \end{equation*}
As defined above, the discrete metric \(\delta_{ X } \) is a metric on \(X \text{,}\) and the metric space \(\of{ X , \delta_{ X } } \) is called a discrete metric space.
