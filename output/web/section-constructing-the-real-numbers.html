    Section 2.2 Constructing the Real Numbers

    In this section, we construct the real number system \(\R \) by Dedekind cuts, and we explore the resultant properties of arithmetic and comparison. However, we will find the existence of such a formal construction and its properties more useful than the particular details of the construction itself, and we will shortly leave behind the perspective suggested by our construction.
    🔗

    Subsection 2.2.1 Cutting the Rational Numbers

    Text before the first subdivision.
    🔗

    Definition 2.2.1. Dedekind cut.

    A proper, non-empty subset \(0 \subseteq X \subseteq \Q \) of the rational numbers \(\Q \) is called a Dedekind cut if it satisfies the following conditions:
    🔗
    1. For all rational numbers \(q_{ 1 } , q_{ 2 } \in \Q \text{,}\) if \(q_{ 1 } \leq q_{ 2 } \) and \(q_{ 2 } \in X \text{,}\) then \(q_{ 1 } \in X \text{.}\)
      🔗
      🔗
    2. \(X \) does not have a maximum; that is, for all elements \(q_{ 1 } \in X \text{,}\) there is an element \(q_{ 2 } \in X \) so that \(q_{ 1 } \lt q_{ 2 } \text{.}\)
      🔗
      🔗
    🔗

    Example 2.2.2. Dedekind cuts.

    For each rational number \(q \in \Q \text{,}\) consider the subset
    \begin{equation*} X_{ q } = \setmid{ p \in \Q }{ p \lt q } \end{equation*}
    of the rational numbers \(\Q \text{.}\) We observe that \(X_{ q } \) is a proper and non-empty subset of \(\Q \text{,}\) since it contains \(q - 1 \) but not \(q \) itself. Moreover, for all rational numbers \(p_{ 1 } , p_{ 2 } \in \Q \text{,}\) if \(p_{ 1 } \leq p_{ 2 } \) and \(p_{ 2 } \in X_{ q } \text{,}\) then
    \begin{equation*} p_{ 1 } \leq p_{ 2 } \lt q\text{,} \end{equation*}
    and so \(p_{ 1 } \in X_{ q } \text{.}\) Finally, for all elements \(p_{ 1 } \in X \text{,}\) we observe that
    \begin{equation*} p \lt p + \frac{ q - p }{ 2 } \lt p + \of{ q - p } = q\text{,} \end{equation*}
    so that \(p + \frac{ q - p }{ 2 } \in X_{ q } \text{.}\) In summary, we have shown that associated to each rational number \(q \in \Q \) is a Dedekind cut \(X_{ q } \text{.}\)
    🔗
    🔗
    We will see that the above construction realizes the rational numbers \(\Q \) as a subset of the set of Dedekind cuts. The Dedekind cuts which arise in this manner will be called rational. We will shortly see that not every Dedekind cut is rational.
    🔗

    Lemma 2.2.3. Rational Dedekind cuts.

    A Dedekind cut \(X \) is rational if and only if its complement \(\Q \setminus X \) has a minimum.
    🔗
    🔗

    Proof.
    Let \(X \) be a Dedekind cut, and suppose that \(X = X_{ q } \) for some rational number \(q \in \Q \text{.}\) Note that \(X = X_{ q } \) has complement
    \begin{equation*} \Q \setminus X = \Q \setminus X_{ q } = \setmid{ p \in \Q }{ p \notin X_{ q } } = \setmid{ p \in \Q }{ p \geq q }\text{.} \end{equation*}
    This set contains \(q \text{,}\) since \(q \geq q \text{.}\) Moreover, \(p \geq q \) for all elements \(p \in \Q \setminus X_{ q } \text{.}\) Thus \(\min \of{ \Q \setminus X } = q \text{.}\)
    🔗
    Conversely, now suppose that \(\Q \setminus X \) has a minimum element, say \(q = \min \of{ \Q \setminus X } \in \Q \text{.}\) Let \(p \in \Q \text{,}\) and suppose that \(p \geq q \text{.}\) If \(p \in X \text{,}\) then that would imply that \(q \in X \text{.}\) This contradiction implies that \(p \notin X \text{.}\) On the other hand, if \(p \lt q \text{,}\) then \(p \notin \Q \setminus X \text{,}\) since \(\min \of{ \Q \setminus X } = q \text{.}\) Thus
    \begin{equation*} X = \setmid{ p \in \Q }{ p \lt q } = X_{ q } \end{equation*}
    is rational.
    🔗
    🔗

    Proposition 2.2.4. Irrational Dedekind cuts.

    Not all Dedekind cuts are rational.
    🔗
    🔗

    Proof.
    Consider the subset
    \begin{equation*} X = \setmid{ q \in \Q }{ q \lt 0 \textrm{ or } q^{ 2 } \lt 2 } \end{equation*}
    of the rational numbers \(\Q \text{.}\) We observe that \(X \) is a non-empty and proper subset of \(\Q \text{,}\) since it contains \(1 \) but not \(2 \text{.}\) Let \(p_{ 1 } , p_{ 2 } \in \Q \) be rational numbers, and suppose that \(p_{ 1 } \leq p_{ 2 } \) and \(p_{ 2 } \in X \text{.}\) If \(p_{ 2 } \lt 0 \text{,}\) then
    \begin{equation*} p_{ 1 } \leq p_{ 2 } \lt 0\text{,} \end{equation*}
    and so \(p_{ 1 } \in X \text{.}\) On the other hand, if \(p_{ 2 }^{ 2 } \lt 2 \text{,}\) then either \(p_{ 1 } \lt 0 \) or \(0 \leq p_{ 1 } \leq p_{ 2 } \text{,}\) in which case \(p_{ 1 }^{ 2 } \leq p_{ 2 }^{ 2 } \lt 2 \text{.}\) In either case, we conclude that \(p_{ 1 } \in X \text{.}\)
    🔗
    Finally, let \(q \in X \text{.}\) If \(q \leq 0 \text{,}\) then let \(p = 1 \text{,}\) and note that \(p \in X \) and \(q \lt p \text{.}\) On the other hand, suppose that \(q \gt 0 \) and \(q^{ 2 } \lt 2 \text{,}\) and consider the rational number
    \begin{equation*} p = \frac{ 2 q + 2 }{ 2 + q } \in \Q\text{.} \end{equation*}
    Note that
    \begin{equation*} p = \frac{ 2 q + 2 }{ 2 + q } = \frac{ 2 q + q^{ 2 } }{ 2 + q } + \frac{ 2 - q^{ 2 } }{ 2 + q } = q + \frac{ 2 - q^{ 2 } }{ 2 + q } \gt q\text{,} \end{equation*}
    and
    \begin{align*} 2 - p^{ 2 } \amp = \frac{ 2 \of{ 2 + q }^{ 2 } }{ \of{ 2 + q }^{ 2 } } - \frac{ \of{ 2 q + 2 }^{ 2 } }{ \of{ 2 + q }^{ 2 } }\\ \amp = \frac{ 4 - 2 q^{ 2 } }{ \of{ 2 + q }^{ 2 } } = \frac{ 2 \of{ 2 - q^{ 2 } } }{ \of{ 2 + q }^{ 2 } } \gt 0\text{,} \end{align*}
    so that \(p^{ 2 } \lt 2 \text{;}\) that is, \(p \in X \text{.}\) We conclude that \(X \) is a Dedekind cut.
    🔗
    We note that \(X \) has complement
    \begin{equation*} \Q \setminus X = \setmid{ q \in \Q }{ q \geq 0 \textrm{ and } q^{ 2 } \geq 2 }\text{.} \end{equation*}
    In fact, since \(q^{ 2 } \neq 2 \) for all rational numbers \(q \in \Q \text{,}\)
    \begin{equation*} \Q \setminus X = \setmid{ q \in \Q }{ q \geq 0 \textrm{ and } q^{ 2 } \gt 2 }\text{.} \end{equation*}
    This set has no minimum element, and so \(X \) is not rational.
    🔗
    🔗
