    Section 1.1 What Is a Set?

    Since the early 20th century, the practice of mathematics has undergone a radical transformation, which grew out of the idea that the whole of the mathematical canon can and should be rephrased in the language of set theory. Informally, a set is a collection of objects, which are called its elements. If an object \(x \) is an element of a set \(S \text{,}\) then we write \(x \in S \text{;}\) otherwise, if \(x \) is not an element of \(S \text{,}\) then we write \(x \notin S \text{.}\)
    🔗

    Subsection 1.1.1 Definitions of Sets

    You may complain that the terms “collection” and “object” are vague and undefined, and you would be right to object; the above characterization of sets is definitely not as rigorous as we might desire, and is in fact internally inconsistent.
    🔗
    However, the technical difficulties associated with building a rigorous and internally consistent set theory are outside the scope of this course and in this context, far more trouble than they are worth. In fact, it’s entirely possible that such an internally consistent set theory does not exist!
    🔗
    As such, we will leave this ambiguity in favor of moving on to more salient topics. Even though we will not encounter any such issues in this text, the “definition” above does lead to some interesting inconsistencies. We will explore one such contradiction, called Russell’s paradox, shortly. First, however, we introduce some shorthand notation which will be useful in specifying sets we wish to discuss.
    🔗

    Convention 1.1.1. Roster notation.

    It’s easy to completely describe a set which contains only a finite number of elements; we can simply write those elements down! Concretely, if \(S \) is the set whose elements are precisely the objects \(x_{ 1 } , \dotsc , x_{ n } \text{,}\) then we may write
    \begin{equation*} S = \set{ x_{ 1 } , \dotsc , x_{ n } } . \end{equation*}
    The above notation for specifying sets is called roster notation. Not all sets are finite, however, and it is obviously impossible to write down a complete roster of all of the elements of an infinite set. Therefore, we cannot generalize roster notation to sets which contain infinitely many elements.
    🔗
    🔗

    Definition 1.1.2. Definable collection.

    Let \(\Phi \) be a unary Boolean predicate (a statement whose truth value depends on a single variable \(x \)). The collection of objects \(x \) which satisfy \(\Phi \) is said to be defined by \(\Phi \text{,}\) and is denoted
    \begin{equation*} \setmid{ x }{ \Phi \of{ x } }\text{.} \end{equation*}
    Such a collection of objects which satisfy such a Boolean predicate is called a definable collection. We will see that the collection of objects that satisfy a given Boolean predicate is not necessarily a set; this is the content of Russell’s paradox.
    🔗
    🔗

    Example 1.1.3. Sets.

    The following are examples of sets:
    • The empty set, denoted \(\emptyset \) or \(\set{ } \text{,}\) is the set with no elements; that is, \(x \notin \emptyset \) for all objects \(x \text{.}\)
      🔗
      🔗
    • The set \(\set{ 1 , 2 , \mathrm{ red } , \mathrm{ blue } } \) is a non-empty set with only finitely many elements. Note that this set contains different types of objects. In general, we will not place any explicit restrictions on the types of objects allowed inside a set, although some formalizations of axiomatic set theories rely on this to avoid inconsistencies such as Russell’s paradox.
      🔗
      🔗
    • The set \(\N = \set{ 0 , 1 , 2 , \dotsc } \) of natural numbers (also sometimes called the whole numbers or counting numbers) is a non-empty set with infinitely many elements. We will characterize the natural numbers \(\N \) in the usual manner of Peano arithmetic later in this chapter.
      🔗
      🔗
    🔗
    🔗
    As we mentioned before, the set theory based on the “definition” above, which is called naïve set theory, is in fact inconsistent; taking this “definition” as the basis for an axiomatic system, we can derive a contradiction, which implies that we may in fact derive all possible statements, including those which are false! This inconsistency was first demonstrated by Bertrand Russell at the beginning of the 20th century, and bears his name.
    🔗

    Proposition 1.1.4. Russell’s paradox.

    Not all definable collections are sets.
    🔗
    🔗

    Proof.
    By contradiction. Let
    \begin{equation*} \mathfrak{ R } = \setmid{ S }{ S \notin S } \end{equation*}
    be the definable collection of sets which do not contain themselves, and suppose for a contradiction that \(\mathfrak{ R } \) is a set. The question of whether or not \(\mathfrak{ R } \) contains itself is a natural one; however, we will see that both the positive and negative answers lead to contradictions.
    🔗
    This is because, if \(\mathfrak{ R } \in \mathfrak{ R } \text{,}\) then \(\mathfrak{ R } \notin \mathfrak{ R } \) by the definition of \(\mathfrak{ R } \text{.}\) On the other hand, if \(\mathfrak{ R } \notin \mathfrak{ R } \text{,}\) then \(\mathfrak{ R } \in \mathfrak{ R } \) by the same reasoning. Therefore, \(\mathfrak{ R } \in \mathfrak{ R } \) if and only if \(\mathfrak{ R } \notin \mathfrak{ R } \text{.}\) This contradiction implies that \(\mathfrak{ R } \) is not a set, as desired.
    🔗
    🔗
    Russell’s paradox tells us that naïve set theory is inconsistent. However, as we stated before, it is no easy task to construct a consistent set theory which is strong enough to describe the sophisticated structures we wish to investigate while also weak enough to be internally consistent. However, since the question of whether modern formal set theories are consistent is beyond the scope of this text. With this in mind, we will continue formulating definitions and results in the language of naïve set theory, keeping in mind that rigorous systems of formal set theory (which do not suffer from Russell’s paradox) exist in which they can be expressed and proven.
    🔗
    🔗

    Subsection 1.1.2 Set Containment

    As we have just seen, not all definable collections of mathematical objects are sets. However, any collection of objects which themselves belong to a set is also a set in its own right; such a set of elements of a given set is called a subset.
    🔗

    Definition 1.1.5. Subset; superset.

    A set \(S \) is a subset of a set \(T \) if \(x \in T \) for all elements \(x \in S \text{.}\) In this case, we say that \(S \) is contained in \(T \text{,}\) and we write \(S \subseteq T \text{.}\) Less commonly, we may call \(T \) a superset of \(S \) if \(S \) is a subset of \(T \text{.}\) In this case, we say that \(T \) contains \(S \text{,}\) and we write \(T \supseteq S \text{.}\)
    🔗
    Also, a set \(S \) is a proper subset of a set \(T \) if \(S \subseteq T \) but \(T \not \subseteq S \text{.}\) In this case, we say that \(S \) is strictly contained in \(T \text{,}\) and we write \(S \subsetneq T \text{.}\) Analogously, \(T \) is a proper superset of \(S \) if \(S \) is a proper subset of \(T \text{.}\) In this case, we say that \(T \) strictly contains \(S \text{,}\) and we write \(T \supsetneq S \text{.}\)
    🔗
    🔗

    Definition 1.1.6. Set equality.

    Two sets are considered equal if they contain exactly the same elements; that is, one set \(A \) is equal to another set \(B \) when objects are elements of \(A \) if and only if they are elements of \(B \text{.}\) This occurs precisely when both \(A \subseteq B \) and \(B \subseteq A \text{.}\)
    🔗
    🔗

    Example 1.1.7. Examples of subsets.

    The following are examples of subsets of sets:
    • For any set \(S \text{,}\) the empty set \(\emptyset \) and \(S \) itself are subsets of \(S \text{.}\)
      🔗
      🔗
    • The set \(\set{ 1 , 3 , 5 , \dotsc } \) of odd natural numbers is a proper subset of the set \(\N \) of natural numbers. Written more concisely,
      \begin{equation*} \set{ 1 , 3 , 5 , \dotsc } \subsetneq \N\text{.} \end{equation*}
      🔗
      🔗
    🔗
    🔗

    Lemma 1.1.8. Transitivity of set containment.

    For any sets \(S \text{,}\) \(T \text{,}\) and \(U \text{,}\) if \(S \subseteq T \) and \(T \subseteq U \text{,}\) then \(S \subseteq U \text{.}\)
    🔗
    🔗

    Proof.
    Suppose that \(S \subseteq T \) and \(T \subseteq U \text{,}\) and let \(x \in S \text{.}\) Since \(x \in S \) and \(S \subseteq T \text{,}\) \(x \in T \text{.}\) Moreover, since \(x \in T \) and \(T \subseteq U \text{,}\) \(x \in U \text{.}\)
    🔗
    In summary, we have shown that \(x \in U \) for all \(x \in S \text{.}\) We conclude that \(S \subseteq U \text{.}\)
    🔗
    🔗

    Convention 1.1.9. Set-builder notation.

    Although not all definable collections are sets, any definable subcollection of a set is itself a set; that is, given a set \(S \) and a unary Boolean predicate \(\Phi \text{,}\) the definable collection
    \begin{equation*} \setmid{ x }{ x \in S \textrm{ and } \Phi \of{ x } } \end{equation*}
    is a set (and in fact a subset of \(S \)). This subset will also be denoted in set-builder notation by
    \begin{equation*} \setmid{ x \in S }{ \Phi \of{ x } } = \setmid{ x }{ x \in S \textrm{ and } \Phi \of{ x } }\text{.} \end{equation*}
    🔗
    🔗

    Definition 1.1.10. Power set.

    The power set of a set \(S \) is the set of subsets of \(S \text{;}\) this set is denoted \(\ps \of{ S } \text{,}\) so that
    \begin{equation*} \ps \of{ S } = \setmid{ T }{ T \subseteq S }\text{.} \end{equation*}
    🔗
    🔗

    Lemma 1.1.11. Power sets are non-empty.

    The power set \(\ps \of{ S } \) of any set \(S \) is non-empty; that is, \(\ps \of{ S } \neq \emptyset \text{.}\)
    🔗
    🔗

    Proof.
    Fix a set \(S \text{,}\) and note that the statement \(x \in S \) for all \(x \in \emptyset \) is vacuously true, so that \(\emptyset \subseteq S \text{.}\) Thus \(\emptyset \in \ps \of{ S } \text{,}\) and so \(\ps \of{ S } \neq \emptyset \) is non-empty.
    🔗
    🔗

    Proof.
    Fix a set \(S \text{,}\) and note that the statement \(x \in S \) for all \(x \in S \) is a tautology, so that \(S \subseteq S \text{.}\) Thus \(S \in \ps \of{ S } \text{,}\) and so \(\ps \of{ S } \neq \emptyset \) is non-empty.
    🔗
    🔗

    Insight 1.1.12. Analysis of the proofs of Lemma 1.1.11.

    Both of the above proofs of Lemma 1.1.11 Power sets are non-empty are correct, although they are nonequivalent. Both are valuable for different reasons; the first proof shows that the power set of a set is nonempty since the empty set $ \emptyset $ is a subset of any set, and the second proof shows the same conclusion since any set is a subset of itself. These proofs are called constructive, in that they explicitly exhibited an object satisfied the desired properties (being an element of the power set of a given set). Proofs which are not constructive are called non-constructive.
    🔗
    🔗
    In general, sets are likely to have many subsets, so that power sets of sets are likely to have a great many elements. We will investigate the notion of the number of elements of a set, called its cardinality, more precisely and in far greater depth at the end of this chapter.
    🔗
    🔗

    Subsection 1.1.3 Set Operations

    Test reference Subsection 1.1.3 Set Operations
    🔗
    We now introduce several ways to construct new sets from old ones which loosely correspond to the logical operators of conjunction \(\land \text{,}\) disjunction \(\lor \text{,}\) and negation \(\lnot \text{,}\) pronounced “and”, “or” and “not”, respectively.
    🔗

    Definition 1.1.13. Intersection; union; complement.

    The intersection \(S \cap T \) of two sets \(S \) and \(T \) is the set whose elements are both elements of \(S \) and elements of \(T \text{;}\) that is,
    \begin{equation*} S \cap T = \setmid{ x }{ x \in S \textrm{ and } x \in T }\text{.} \end{equation*}
    The union \(S \cup T \) of \(S \) and \(T \) is the set whose elements are either elements of \(S \) or elements of \(T \text{;}\) that is,
    \begin{equation*} S \cup T = \setmid{ x }{ x \in S \textrm{ or } x \in T }\text{.} \end{equation*}
    The complement \(S \setminus T \) of \(T \) in \(S \) is the set whose elements are elements of \(S \) but not elements of \(T \text{;}\) that is,
    \begin{equation*} S \setminus T = \setmid{ x }{ x \in S \textrm{ and } x \notin T }\text{.} \end{equation*}
    🔗
    🔗

    Proposition 1.1.14. Set operations and containment.

    Fix sets \(S \text{,}\) \(T \text{,}\) and \(U \text{.}\)
    1. \(U \subseteq S \cap T \) if and only if \(U \subseteq S \) and \(U \subseteq T \text{.}\)
      🔗
      🔗
    2. \(S \cup T \subseteq U \) if and only if \(S \subseteq U \) and \(T \subseteq U \text{.}\)
      🔗
      🔗
    3. \(U \subseteq S \setminus T \) if and only if \(U \subseteq S \) and \(U \cap T = \emptyset \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.
    1. First suppose that \(U \subseteq S \cap T \text{.}\) Since \(S \cap T \subseteq S \text{,}\) Lemma 1.1.8 Transitivity of set containment implies that \(U \subseteq S \text{.}\) Similarly, since \(S \cap T \subseteq T \text{,}\) Lemma 1.1.8 Transitivity of set containment implies that \(U \subseteq T \text{.}\)
      🔗
      Conversely, now suppose that \(U \subseteq S \) and \(U \subseteq T \text{,}\) and let \(x \in U \text{.}\) Since \(x \in U \) and \(U \subseteq S \text{,}\) \(x \in S \text{.}\) Similarly, since \(x \in U \) and \(U \subseteq T \text{,}\) \(x \in T \text{.}\) Since \(x \in S \) and \(x \in T \text{,}\) we conclude that \(x \in S \cap T \text{.}\) In summary, we have shown that \(x \in S \cap T \) for all \(x \in U \text{,}\) so that \(U \subseteq S \cap T \text{.}\)
      🔗
      🔗
    2. First suppose that \(S \cup T \subseteq U \text{.}\) Since \(S \subseteq S \cup T \text{,}\) Lemma 1.1.8 Transitivity of set containment implies that \(S \subseteq U \text{.}\) Similarly, since \(T \subseteq S \cup T \text{,}\) Lemma 1.1.8 Transitivity of set containment implies that \(T \subseteq U \text{.}\)
      🔗
      Conversely, now suppose that \(S \subseteq U \) and \(T \subseteq U \text{.}\) Let \(x \in S \cup T \text{,}\) so that \(x \in S \) or \(x \in T \text{.}\) If \(x \in S \text{,}\) then \(x \in U \text{,}\) since \(S \subseteq U \text{.}\) Similarly, if \(x \in T \text{,}\) then \(x \in U \text{,}\) since \(T \subseteq U \text{.}\) We conclude that \(x \in U \text{.}\) In summary, we have shown that \(x \in U \) for all \(x \in S \cup T \text{,}\) so that \(S \cup T \subseteq U \text{.}\)
      🔗
      🔗
    3. First suppose that \(U \subseteq S \setminus T \text{,}\) and let \(x \in U \text{.}\) Since \(x \in U \) and \(U \subseteq S \setminus T \text{,}\) \(x \in S \setminus T \text{,}\) so that \(x \in S \) and \(x \notin T \text{.}\) In summary, we have shown that \(x \in S \) for all \(x \in U \text{,}\) so that \(U \subseteq S \text{.}\) Moreover, we have shown that \(x \notin T \) for all \(x \in U \text{,}\) so that no element of \(U \) is an element of \(T \text{;}\) that is, \(U \cap T = \emptyset \text{.}\)
      🔗
      Conversely, now suppose that \(U \subseteq S \) and \(U \cap T = \emptyset \text{,}\) and let \(x \in U \text{.}\) Since \(x \in U \) and \(U \subseteq S \text{,}\) \(x \in S \text{.}\) Moreover, since \(x \in U \) and \(U \cap T = \emptyset \text{,}\) \(x \notin T \text{.}\) Since \(x \in S \) and \(x \notin T \text{,}\) we conclude that \(x \in S \setminus T \text{.}\) In summary, we have shown that \(x \in S \setminus T \) for all \(x \in U \text{,}\) and so \(U \subseteq S \setminus T \text{.}\)
      🔗
      🔗
    🔗

    Corollary 1.1.15. Properties of the set operations.

    Fix sets \(S \text{,}\) \(T \text{,}\) and \(U \text{.}\)
    1. \(S \cap S = S \cup S = S \cup \emptyset = S \setminus \emptyset = S \text{,}\) and \(S \cap \emptyset = S \setminus S = \emptyset \setminus S = \emptyset \text{.}\)
      🔗
      🔗
    2. Commutativity: \(S \cap T = T \cap S \) and \(S \cup T = T \cup S \text{.}\)
      🔗
      🔗
    3. Associativity: \(\of{ S \cap T } \cap U = S \cap \of{ T \cap U } \) and \(\of{ S \cup T } \cup U = S \cup \of{ T \cup U } \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.
    1. Since \(S \subseteq S \text{,}\) \(S \subseteq S \cap S \) and \(S \cup S \subseteq S \) by (1) and (2) of Proposition 1.1.14 Set operations and containment, respectively. Of course \(S \cap S \subseteq S \subseteq S \cup S \text{,}\) so that \(S \cap S = S = S \cup S \text{.}\)
      🔗
      Similarly, since \(S \subseteq S \) and \(\emptyset \subseteq S \text{,}\) \(S \cup \emptyset \subseteq S \) by (2) of Proposition 1.1.14 Set operations and containment. Of course \(S \subseteq S \cup \emptyset \text{,}\) and so \(S \cup \emptyset = S \text{.}\)
      🔗
      \(\emptyset \subseteq S \cap \emptyset \subseteq \emptyset \text{,}\) and so \(S \cap \emptyset = \emptyset \text{.}\) Moreover, together with the observation that \(S \subseteq S \text{,}\) this implies by (1) of Proposition 1.1.14 Set operations and containment that \(S \subseteq S \setminus \emptyset \text{.}\) Of course, \(S \setminus \emptyset \subseteq S \text{,}\) and so \(S \setminus \emptyset = S \text{.}\)
      🔗
      Similarly, \(\emptyset \subseteq \emptyset \setminus S \subseteq \emptyset \text{,}\) and so \(\emptyset \setminus S = \emptyset \text{.}\)
      🔗
      Finally, note that any element of \(S \setminus S \) is both an element of \(S \) and not an element of \(S \text{,}\) so that no such element can exist. Thus \(S \setminus S = \emptyset \text{.}\)
      🔗
      🔗
    2. Since \(S \cap T \subseteq T \) and \(S \cap T \subseteq S \text{,}\) (1) of Proposition 1.1.14 Set operations and containment implies that \(S \cap T \subseteq T \cap S \text{.}\) By symmetry, \(T \cap S \subseteq S \cap T \text{,}\) so that \(S \cap T = T \cap S \text{.}\)
      🔗
      Similarly, since \(S \subseteq T \cup S \) and \(T \subseteq T \cup S \text{,}\) (2) of Proposition 1.1.14 Set operations and containment implies that \(S \cup T \subseteq T \cup S \text{.}\) By symmetry, \(T \cup S \subseteq S \cup T \text{,}\) so that \(S \cup T = T \cup S \text{.}\)
      🔗
      🔗
    3. We note that
      \begin{align*} \of{ S \cap T } \cap U \amp \subseteq S \cap T \subseteq T\\ \of{ S \cap T } \cap U \amp \subseteq U\text{,} \end{align*}
      so that \(\of{ S \cap T } \cap U \subseteq T \cap U \) by (1) of Proposition 1.1.14 Set operations and containment. Similarly,
      \begin{equation*} \of{ S \cap T } \cap U \subseteq S \cap T \subseteq T\text{,} \end{equation*}
      so that \(\of{ S \cap T } \cap U \subseteq S \cap \of{ T \cap U } \) by another application of (1) of Proposition 1.1.14 Set operations and containment.
      🔗
      Conversely, we note that
      \begin{align*} S \cap \of{ T \cap U } \amp \subseteq S\\ S \cap \of{ T \cap U } \amp \subseteq T \cap U \subseteq T\text{,} \end{align*}
      so that \(S \cap \of{ T \cap U } \subseteq S \cap T \) by (1) of Proposition 1.1.14 Set operations and containment. Similarly,
      \begin{equation*} S \cap \of{ T \cap U } \subseteq T \cap U \subseteq U\text{,} \end{equation*}
      so that \(S \cap \of{ T \cap U } \subseteq \of{ S \cap T } \cap U \) by another application of (1) of Proposition 1.1.14 Set operations and containment. Thus
      \begin{equation*} \of{ S \cap T } \cap U = S \cap \of{ T \cap U }\text{.} \end{equation*}
      🔗
      🔗
    🔗
    In the next section, we will generalize the first two ways (intersection and union) to arbitrary sets of sets; this generalization will mirror the similarities between conjunction \(\land \) and universal quantification \(\forall \) and between disjunction \(\lor \) and existential quantification \(\exists \text{.}\)

    4. Set operations and containment.

    Let \(S \) and \(T \) be sets.
    🔗
    (a)
    Prove that \(S \cap T = S \) if and only if \(S \subseteq T \text{.}\)
    🔗
    Solution.
    Since \(S \cap T \subseteq S \text{,}\) it suffices to show that \(S \subseteq S \cap T \) if and only if \(S \subseteq T \text{.}\) To that end, note that if \(S \subseteq S \cap T \text{,}\) then (1) of Proposition 1.1.14 Set operations and containment implies that \(S \subseteq T \text{.}\)
    🔗
    Conversely, we note that \(S \subseteq S \text{,}\) so that if \(S \subseteq T \text{,}\) then (1) of Proposition 1.1.14 Set operations and containment implies that \(S \subseteq S \cap T \text{.}\)
    🔗
    🔗
    🔗
    (b)
    Prove that \(S \cup T = S \) if and only if \(T \subseteq S \text{.}\)
    🔗
    Solution.
    Since \(S \cup S \cup T \text{,}\) it suffices to show that \(S \cup T \subseteq S \) if and only if \(T \subseteq S \text{.}\) To that end, note that if \(S \cup T \subseteq S \text{,}\) then (2) of Proposition 1.1.14 Set operations and containment implies that \(T \subseteq S \text{.}\)
    🔗
    Conversely, we note that \(S \subseteq S \text{,}\) so that if \(T \subseteq S \text{,}\) then (2) of Proposition 1.1.14 Set operations and containment implies that \(S \cup T \subseteq S \text{.}\)
    🔗

    (c)
    Prove that \(S \setminus T = S \) if and only if \(S \cap T = \emptyset \text{.}\)
    🔗
    Solution.
    First suppose that \(S \cap T = \emptyset \text{.}\) Then \(x \notin T \) for all \(x \in S \text{,}\) and so
    \begin{equation*} S \setminus T = \setmid{ x }{ x \in S \textrm{ and } x \notin T } = \setmid{ x }{ x \in S } = S\text{.} \end{equation*}
