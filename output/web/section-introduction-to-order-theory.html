    Section 1.5 Introduction to Order Theory

    In this section, we will investigate another class of binary relations, which can be used to establish a notion of an ordering on a set. Touchstone examples will be the standard ordering of the natural numbers and the ordering of sets by containment.
    🔗

    Subsection 1.5.1 Partial Orders

    We have seen that a finitary relation on a set can be viewed as elevating its elements as distinguished finite sequences in that set. Some binary relations on a set may be interpreted in this way as establishing an ordering, with the first entries of each distinguished ordered pair preceding the second entries in this ordering.
    🔗
    What are the salient properties of the standard ordering of the natural numbers? We will see that, in order to establish a similar notion of ordering on a set, a given binary operation should satisfy similar properties. However, even if a binary operation satisfies some, but not all, of the desired properties, we will see that we can still interpret it as a weaker type of ordering relation. However, all ordering relations should, at the very least, satisfy reflexivity and transitivity. Such a reflexive and transitive binary operation is called a preorder.
    🔗

    Definition 1.5.1. Preorder.

    A binary relation \(\precsim \) on a set \(X \) is called a preorder if it is reflexive and transitive; that is, \(\precsim \) is a preorder if it satisfies the following conditions:
    1. Reflexivity.
      \(x \precsim x \) for all elements \(x \in X \text{.}\)
      🔗
      🔗
    2. Transitivity.
      For all elements \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{,}\) if \(x_{ 1 } \precsim x_{ 2 } \) and \(x_{ 2 } \precsim x_{ 3 } \text{,}\) then \(x_{ 1 } \precsim x_{ 3 } \text{.}\)
      🔗
      🔗
    If \(\precsim \) is a preorder on a set \(X \text{,}\) then the ordered pair \(\of{ X , \precsim } \) is called a preordered set (or a proset).
    🔗
    🔗

    Lemma 1.5.2. Preorders from pullbacks.

    The pullback \(\leq_{ f } \) of the standard ordering \(\leq \) on the natural numbers \(\N \) along a function \(f \colon X \to \N \) on a set \(X \) is a preorder on \(X \text{.}\)
    🔗
    🔗

    Proof.
    First let \(x \in X \text{,}\) and note that \(f \of{ x } \leq f \of{ x } \) by (1) of Proposition 1.3.15 Comparison is a total order. In particular, \(x \leq_{ f } x \text{,}\) and so \(\leq_{ f } \) is reflexive.
    🔗
    Now let \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{,}\) and suppose that \(x_{ 1 } \leq_{ f } x_{ 2 } \) and \(x_{ 2 } \leq_{ f } x_{ 3 } \text{.}\) Then \(f \of{ x_{ 1 } } \leq f \of{ x_{ 2 } } \) and \(f \of{ x_{ 2 } } \leq f \of{ x_{ 3 } } \text{,}\) and so \(f \of{ x_{ 1 } } \leq f \of{ x_{ 3 } } \) by (3) of Proposition 1.3.15 Comparison is a total order. In particular, \(x_{ 1 } \leq_{ f } x_{ 3 } \text{,}\) and so \(\leq_{ f } \) is transitive. Since \(\leq_{ f } \) is both reflexive and transitive, it is a preorder on \(X \text{.}\)
    🔗
    🔗
    It turns out that, at least relative to the standard ordering \(\leq \) of the natural numbers \(\N \text{,}\) the defining properties of a preorder are still too weak to be of much use in the general case. After all, an equivalence relation is just a symmetric preorder, and there is no way of meaningfully assigning a notion of an ordering based on an equivalence relation. This is because an equivalence relation is symmetric; in order to consider an ordering based on a binary operation, we would want to avoid any symmetry at all. Therefore, we must require that an ordering relation satisfy stronger assumptions.
    🔗

    Definition 1.5.3. Partial order.

    A binary relation \(\preceq \) on a set \(X \) is called a partial order if it satisfies the following conditions:
    1. Reflexivity.
      \(x \preceq x \) for all elements \(x \in X \text{.}\)
      🔗
      🔗
    2. Anti-symmetry.
      For all elements \(x_{ 1 } , x_{ 2 } \in X \text{,}\) if \(x_{ 1 } \preceq x_{ 2 } \) and \(x_{ 2 } \preceq x_{ 1 } \text{,}\) then \(x_{ 1 } = x_{ 2 } \text{.}\)
      🔗
      🔗
    3. Transitivity.
      For all elements \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{,}\) if \(x_{ 1 } \preceq x_{ 2 } \) and \(x_{ 2 } \preceq x_{ 3 } \text{,}\) then \(x_{ 1 } \preceq x_{ 3 } \text{.}\)
      🔗
      🔗
    If \(\preceq \) is a partial order on a set \(X \text{,}\) then the ordered pair \(\of{ X , \preceq } \) is called a partially ordered set (or a poset).
    🔗
    🔗

    Example 1.5.4. Partial orders.

    The following are examples of partial orders on sets:
    1. The standard ordering \(\leq \) on the natural numbers \(\N \) is a partial order on \(\N \) by (1), (2), and (3) of Proposition 1.3.15 Comparison is a total order.
      🔗
      🔗
    2. Containment \(\subseteq \) is a partial order on the power set \(\ps \of{ X } \) of a set \(X \) by Definition 1.1.6 Set equality and Lemma 1.1.8 Transitivity of set containment.
      🔗
      🔗
    🔗
    🔗
    At the end of this section, we will explore a property of partial orders called totality which distinguishes between our touchstone examples of the standard ordering \(\leq \) of natural numbers and the containment \(\subseteq \) of sets.
    🔗
    🔗

    Subsection 1.5.2 Boundedness and Extrema

    Partial orders satisfy many of the properties of the standard ordering of the natural numbers. These properties allow us to generalize the definitions of many of the usual notions associated with an ordering, such as upper and lower bounds and maximal and minimal elements, to partially ordered sets.
    🔗

    Definition 1.5.5. Boundedness.

    Let \(U \subseteq X \) be a subset of a partially ordered set \(\of{ X , \preceq } \text{.}\)
    1. An element \(x \in X \) is called an upper bound for \(U \) if \(u \preceq x \) for all elements \(u \in U \text{.}\) If such an upper bound for \(U \) exists, then \(U \) is called bounded from above in \(X \text{.}\) Otherwise, if no such upper bound for \(U \) exists, then \(U \) is called unbounded from above.
      🔗
      🔗
    2. An element \(x \in X \) is called a lower bound for \(U \) if \(x \preceq u \) for all elements \(u \in U \text{.}\) If such a lower bound for \(U \) exists, then \(U \) is called bounded from below in \(X \text{.}\)
      🔗
      🔗
    A subset \(U \subseteq X \) is called bounded in \(X \) if it is both bounded from above in \(X \) and bounded from below in \(X \text{,}\) and \(U \) is called unbounded in \(X \) otherwise if it is either unbounded from above in \(X \) or unbounded from below in \(X \text{.}\)
    🔗
    🔗

    Lemma 1.5.6. Extrema are unique.

    Let \(U \subseteq X \) be a subset of a partially ordered set \(\of{ X , \preceq } \text{.}\) At most one element of \(U \) is an upper bound for \(U \) in \(X \text{,}\) and at most one element of \(U \) is a lower bound for \(U \) in \(X \text{.}\)
    🔗
    🔗

    Proof.
    Let \(u_{ 1 } , u_{ 2 } \in U \text{,}\) and suppose first that \(u_{ 1 } \) and \(u_{ 2 } \) are upper bounds for \(U \) in \(X \text{,}\) so that \(u \preceq u_{ 1 } \) and \(u \preceq u_{ 2 } \) for all elements \(u \in U \text{.}\) In particular, \(u_{ 1 } \preceq u_{ 2 } \) and \(u_{ 2 } \preceq u_{ 1 } \text{,}\) and so \(u_{ 1 } = u_{ 2 } \) by anti-symmetry.
    🔗
    Similarly, now suppose that \(u_{ 1 } \) and \(u_{ 2 } \) are lower bounds for \(U \) in \(X \text{,}\) so that \(u_{ 1 } \preceq u \) and \(u_{ 2 } \preceq u \) for all elements \(u \in U \text{.}\) In particular, \(u_{ 1 } \preceq u_{ 2 } \) and \(u_{ 2 } \preceq u_{ 1 } \text{,}\) and so \(u_{ 1 } = u_{ 2 } \) by anti-symmetry.
    🔗
    🔗

    Definition 1.5.7. Extrema.

    Let \(U \subseteq X \) be a subset of a partially ordered set \(\of{ X , \preceq } \text{.}\)
    1. If there is an element of \(U \) which is an upper bound for \(U \) in \(X \text{,}\) then this unique element is called the maximum of \(U \) and is denoted \(\max \of{ U } \text{.}\)
      🔗
      🔗
    2. If there is an element of \(U \) which is a lower bound for \(U \) in \(X \text{,}\) then this unique element is called the minimum of \(U \) and is denoted \(\min \of{ U } \text{.}\)
      🔗
      🔗
    Together, maxima and minima are called extrema.
    🔗
    🔗
    The uniqueness of extrema in partially ordered sets is one way in which partial orders resemble the standard ordering of the natural number system. However, unlike in the standard ordering of the natural number system, there is in general a distinction between extrema, which are defined above, and extremal elements, which are defined below.
    🔗

    Definition 1.5.8. Extremal element.

    Let \(U \subseteq X \) be a subset of a partially ordered set \(\of{ X , \preceq } \text{.}\)
    1. An element \(u \in U \) is called maximal in \(U \) if for all elements \(v \in U \text{,}\) if \(u \preceq v \text{,}\) then \(v = u \text{.}\)
      🔗
      🔗
    2. An element \(u \in U \) is called minimal in \(U \) if for all elements \(v \in U \text{,}\) if \(v \preceq u \text{,}\) then \(v = u \text{.}\)
      🔗
      🔗
    Together, maximal and minimal elements are called extremal elements.
    🔗
    🔗

    Example 1.5.9. Extrema and extremal elements.

    The following are examples of extrema and extremal elements:
    1. The minimum of the natural numbers \(\N \) is \(\min \of{ \N } = 0 \text{,}\) which is the unique minimal element. However, \(\N \) has no maximum or maximal element.
      🔗
      🔗
    2. Consider set containment \(\subseteq \) on the power set \(\ps \of{ X } \) of a set \(X \text{.}\) \(\ps \of{ X } \) has minimum and maximum \(\min \of{ \ps \of{ X } } = \emptyset \) and \(\max \of{ \ps \of{ X } } = X \text{,}\) which are also the unique minimal and maximal elements, respectively.
      🔗
      🔗
    3. Now consider set containment \(\subseteq \) on the set \(X \) of proper non-empty subsets of the set \(\set{ 1 , 2 , 3 } \text{;}\) that is,
      \begin{equation*} X = \set{ \set{ 1 } , \set{ 2 } , \set{ 3 } , \set{ 1 , 2 } , \set{ 1 , 3 } , \set{ 2 , 3 } }\text{.} \end{equation*}
      Then \(X \) has no maximum or minimum, but \(\set{ 1 } \text{,}\) \(\set{ 2 } \text{,}\) and \(\set{ 3 } \) are minimal elements, and \(\set{ 1 , 2 } \text{,}\) \(\set{ 1 , 3 } \text{,}\) and \(\set{ 2 , 3 } \) are maximal elements.
      🔗
      🔗
    🔗
    🔗

    Proposition 1.5.10. Extrema are extremal elements.

    Let \(U \subseteq X \) be a subset of a partially ordered set \(\of{ X , \preceq } \text{.}\) If \(U \) has a maximum, then \(\max \of{ U } \) is the unique maximal element of \(U \text{.}\) If \(U \) has a minimum, then \(\min \of{ U } \) is the unique minimal element of \(U \text{.}\)
    🔗
    🔗

    Proof.
    First suppose that \(U \) has a maximum. Let \(u \in U \text{,}\) and suppose that \(\max \of{ U } \preceq u \text{.}\) Of course \(u \preceq \max \of{ U } \text{,}\) and so anti-symmetry implies that \(u = \max \of{ U } \text{;}\) that is, \(\max \of{ U } \) is maximal in \(U \text{.}\)
    🔗
    On the other hand, now consider a maximal element \(u \in U \text{.}\) Since \(u \preceq \max \of{ U } \text{,}\) the maximality of \(u \) implies that \(u = \max \of{ U } \text{.}\)
    🔗
    Now suppose that \(U \) has a minimum. Let \(u \in U \text{,}\) and suppose that \(u \preceq \min \of{ U } \text{.}\) Of course \(\min \of{ U } \preceq u \text{,}\) and so anti-symmetry implies that \(u = \min \of{ U } \text{;}\) that is, \(\min \of{ U } \) is minimal in \(U \text{.}\)
    🔗
    On the other hand, now consider a minimal element \(u \in U \text{.}\) Since \(\min \of{ U } \preceq u \text{,}\) the minimality of \(u \) implies that \(u = \min \of{ U } \text{.}\)
    🔗
    🔗
    The distinction between extrema and extremal elements is one way in which more general partial orders differ from the standard ordering of the natural numbers. However, this subtlety does not arise in all partial orders. We will shortly explore a class of partial orders in which the aforementioned distinction between extrema and extremal elements does not exist.
    🔗
    🔗

    Subsection 1.5.3 Comparability and Total Orders

    The fact that the converse of Proposition 1.5.10 Extrema are extremal elements does not hold in general stems from the fact that in a partially ordered set, not all pairs of elements are comparable. That is, given two elements of a partially ordered set, it is not necessarily true that one will precede the other.
    🔗

    Definition 1.5.11. Comparability.

    Two elements \(x_{ 1 } , x_{ 2 } \in X \) in a partially ordered set \(\of{ X , \preceq } \) are called comparable if either \(x_{ 1 } \preceq x_{ 2 } \) or \(x_{ 2 } \preceq x_{ 1 } \text{,}\) and incomparable otherwise.
    🔗
    🔗
    Any two natural numbers are comparable in the standard ordering. However, this is not true for partially ordered sets in general. As an example, consider the subsets \(S = \set{ 0 , 1 } \) and \(T = \set{ 1 , 2 } \) of the natural numbers \(\N \text{.}\) One can check that \(S \not \subseteq T \) and \(T \not \subseteq S \text{,}\) so that \(S \) and \(T \) are incomparable in the power set \(\ps \of{ \N } \) of the natural numbers \(\N \text{.}\) So partially ordered sets may have elements which are incomparable. This is the reason why partial orders are called partial; partial orders in which all elements are comparable are called total.
    🔗

    Definition 1.5.12. Total order.

    A binary relation \(\leq \) on a set \(X \) is called a total order if it is anti-symmetric and transitive and each pair of elements is comparable; that is, \(\leq \) is a total order if it satisfies the following conditions:
    1. Antisymmetry.
      For all elements \(x_{ 1 } , x_{ 2 } \in X \text{,}\) if \(x_{ 1 } \leq x_{ 2 } \) and \(x_{ 2 } \leq x_{ 1 } \text{,}\) then \(x_{ 1 } = x_{ 2 } \text{.}\)
      🔗
      🔗
    2. Transitivity.
      For all elements \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{,}\) if \(x_{ 1 } \leq x_{ 2 } \) and \(x_{ 2 } \leq x_{ 3 } \text{,}\) then \(x_{ 1 } \leq x_{ 3 } \text{.}\)
      🔗
      🔗
    3. Totality.
      For all elements \(x_{ 1 } , x_{ 2 } \in X \text{,}\) either \(x_{ 1 } \leq x_{ 2 } \) or \(x_{ 2 } \leq x_{ 1 } \text{.}\)
      🔗
      🔗
    If \(\leq \) is a total order on a set \(X \text{,}\) then the ordered pair \(\of{ X , \leq } \) is called a totally ordered set.
    🔗
    🔗

    Example 1.5.13. Total order.

    The standard ordering \(\leq \) on the natural numbers \(\N \) is a total order by (2), (3), and (4) of Proposition 1.3.15 Comparison is a total order.
    🔗
    🔗

    Lemma 1.5.14. Total orders are partial orders.

    Every total order on a set \(X \) is a partial order on \(X \text{.}\)
    🔗
    🔗

    Proof.
    Let \(\leq \) be a total order on a set \(X \text{.}\) Since \(\leq \) is antisymmetric and transitive, it remains to show that \(\leq \) is reflexive. To that end, let \(x \in X \text{,}\) and note that totality implies that either \(x \leq x \) or \(x \leq x \text{;}\) that is \(\leq \) is reflexive (and hence a partial order on \(X \)).
    🔗
    🔗

    Proposition 1.5.15. Extremal elements are extrema in totally ordered sets.

    Let \(\preceq \) be a total order on a set \(X \text{.}\)
    1. If \(X \) contains a maximal element \(e \in X \text{,}\) then \(e = \max \of{ X } \text{.}\)
      🔗
      🔗
    2. If \(X \) contains a minimal element \(e \in X \text{,}\) then \(e = \min \of{ X } \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.
    1. Suppose that \(e \in X \) is a maximal element, and let \(x \in X \text{.}\) Totality implies that \(x \leq e \) or \(e \leq x \text{.}\) If \(e \leq x \text{,}\) then maximality implies that \(x = e \text{,}\) so that \(x \leq e \) by reflexivity. Thus \(e \leq x \) for all \(x \in X \text{,}\) and so \(\max \of{ X } = e \text{.}\)
      🔗
      🔗
    2. Now suppose that \(e \in X \) is a minimal element, and let \(x \in X \text{.}\) Totality implies that \(x \leq e \) or \(e \leq x \text{.}\) If \(x \leq e \text{,}\) then minimality implies that \(x = e \text{,}\) so that \(e \leq x \) by reflexivity. Thus \(x \leq e \) for all \(x \in X \text{,}\) and so \(\min \of{ X } = e \text{.}\)
      🔗
      🔗
    🔗
    We end with an important property of the standard ordering \(\leq \) on the natural numbers \(\N \) related to the axiom of induction.
    🔗

    Theorem 1.5.16. Well-ordering principle.

    Every non-empty subset of the natural numbers \(\N \) has a minimum.
    🔗
    🔗

    Proof.
    Let \(U \subseteq \N \) be a subset of the natural numbers \(\N \text{,}\) and suppose that \(U \) does not have a minimum. We proceed by strong induction on \(n \in \N \) to show that \(n \notin U \text{.}\)
    🔗
    Base case.
    Suppose that \(n = 0 \text{,}\) and suppose for a contradiction that \(n \in U \text{.}\) Since \(0 \leq n \) for all natural numbers \(n \in \N \) by Lemma 1.3.2 \(0 \) is minimal and (2) of Proposition 1.5.15 Extremal elements are extrema in totally ordered sets, then \(n = \min \of{ U } \text{.}\) This contradiction implies that \(n \notin U \text{.}\)
    🔗
    🔗
    Inductive step.
    Suppose for the inductive hypothesis that there is some natural number \(n \in \N \) so that \(k \notin U \) for all natural numbers \(k \in \N \) so that \(0 \leq k \leq n \text{.}\) Suppose for a contradiction that \(n + 1 \in U \text{,}\) and let \(u \in U \text{.}\) Totality implies that either \(u \leq n \) or \(n \leq u \text{.}\) The inductive hypothesis implies that \(u \not \leq n \text{,}\) so that \(n \leq u \text{.}\)
    🔗
    So \(u = n + p \) for some natural number \(p \in \N \text{.}\) If \(p = 0 \text{,}\) then
    \begin{equation*} n = n + 0 = n + p = u \in U\text{,} \end{equation*}
    and so \(p \neq 0 \text{.}\) Thus \(p = q + 1 \) for some natural number \(q \in \N \text{,}\) and so
    \begin{equation*} n + 1 \leq n + q + 1 = n + p = u\text{.} \end{equation*}
    In summary, we have shown that \(n + 1 \leq u \) for all \(u \in U \text{,}\) so that \(\min \of{ U } = n + 1 \text{.}\) This contradiction implies that \(n + 1 \notin U \text{.}\)
    🔗
    🔗
    We conclude that \(n \notin U \) for all natural numbers \(n \in \N \text{,}\) and so \(U = \emptyset \text{.}\)
    🔗
    🔗
    The Well-ordering principle is sometimes taken to be an axiom and sometimes derived from the axiom of induction as in the above proof. In fact, these results are logically equivalent, as one can derive the axiom of induction from the Well-ordering principle.
