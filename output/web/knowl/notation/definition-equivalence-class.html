Definition 1.4.16. Equivalence class; quotient set.

Let \(\equiv \) be an equivalence relation on a set \(X \text{.}\) The equivalence class \(\class{ x }_{ \equiv } \) of an element \(x \in X \) is the subset of \(X \) defined by
\begin{equation*} \class{ x }_{ \equiv } = \setmid{ y \in X }{ x \equiv y }\text{;} \end{equation*}
that is, \(\class{ x }_{ \equiv } \) is the set of elements which are equivalent to \(x \) under the relation \(\equiv \text{.}\) The quotient \(X / { \equiv } \) of \(X \) by \(\equiv \) is the set of equivalence classes; that is,
\begin{equation*} X / { \equiv } = \setmid{ \class{ x }_{ \equiv } \in \ps \of{ X } }{ x \in X }\text{.} \end{equation*}
