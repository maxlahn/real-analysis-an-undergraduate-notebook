Example 1.1.3. Sets.

The following are examples of sets:
  • The empty set, denoted \(\emptyset \) or \(\set{ } \text{,}\) is the set with no elements; that is, \(x \notin \emptyset \) for all objects \(x \text{.}\)
    🔗
    🔗
  • The set \(\set{ 1 , 2 , \mathrm{ red } , \mathrm{ blue } } \) is a non-empty set with only finitely many elements. Note that this set contains different types of objects. In general, we will not place any explicit restrictions on the types of objects allowed inside a set, although some formalizations of axiomatic set theories rely on this to avoid inconsistencies such as Russell’s paradox.
    🔗
    🔗
  • The set \(\N = \set{ 0 , 1 , 2 , \dotsc } \) of natural numbers (also sometimes called the whole numbers or counting numbers) is a non-empty set with infinitely many elements. We will characterize the natural numbers \(\N \) in the usual manner of Peano arithmetic later in this chapter.
