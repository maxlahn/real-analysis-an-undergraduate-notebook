Associative binary operations can be extended recursively to act on finite sequences of any length. For example, given a finite sequence \(\of{ n_{ j } }_{ j = 1 }^{ m } \) of natural numbers \(n_{ j } \in \N \text{,}\) we write
\begin{align*} \sum_{ j = 1 }^{ m }{ n_{ j } } \amp = n_{ 1 } + n_{ 2 } + \dotsb + n_{ m } \amp \prod_{ j = 1 }^{ m }{ n_{ j } } \amp = n_{ 1 } n_{ 2 } \dotsm n_{ m }\text{.} \end{align*}
