Lemma 1.3.12. Multiplication by \(0 \).

\(0 n = 0 \) for all natural numbers \(n \in \N \text{.}\)
🔗
🔗

Proof.
By induction on \(n \in \N \text{.}\)

Base case.

If \(n = 0 \text{,}\) then
\begin{equation*} 0 n = 0 \cdot 0 = 0\text{.} \end{equation*}
🔗
🔗

Inductive step.

Now suppose for the inductive hypothesis that \(0 n = 0 \) for some natural number \(n \in \N \text{,}\) and note that
\begin{equation*} 0 \of{ n + 1 } = 0 n + 0 = 0 + 0 = 0\text{.} \end{equation*}
🔗
🔗
We conclude that \(0 n = 0 \) for all natural numbers \(n \in \N \text{.}\)
