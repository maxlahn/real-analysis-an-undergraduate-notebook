Proposition 1.3.11. Right distributivity of multiplication over addition.

\(\of{ l + m } n = l n + m n \) for all natural numbers \(l , m , n \in \N \text{.}\)
🔗
🔗

Proof.
Let \(l , m \in \N \) be natural numbers. We proceed by induction on \(n \in \N \text{.}\)

Base case.

If \(n = 0 \text{,}\) then
\begin{equation*} \of{ l + m } n = \of{ l + m } \cdot 0 = 0 = 0 + 0 = l \cdot 0 + m \cdot 0 = l n + m n\text{.} \end{equation*}
🔗
🔗

Inductive step.

Now suppose for the inductive hypothesis that \(\of{ l + m } n = l n + m n \) for some natural number \(n \in \N \text{,}\) and note that
\begin{align*} \of{ l + m } \of{ n + 1 } = \of{ l + m } n + \of{ l + m } = l n + l + m n + m \\ \amp = l \of{ n + 1 } + m \of{ n + 1 } \end{align*}
by repeated applications of Proposition 1.3.4 Associativity of addition and Proposition 1.3.7 Commutativity of addition.
🔗
🔗
We conclude that \(\of{ l + m } n = l n + m n \) for all natural numbers \(n \in \N \text{.}\)
