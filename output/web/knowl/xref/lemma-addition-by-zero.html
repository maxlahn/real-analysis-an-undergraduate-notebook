Lemma 1.3.5. Addition by \(0 \).

\(n + 0 = 0 + n = n \) for all natural numbers \(n \in \N \text{.}\)
🔗
🔗

Proof.
\(n + 0 = n \) for all natural numbers \(n \in \N \) by the Peano axioms, and so it suffices to show \(0 + n = n \text{.}\) We proceed by induction on \(n \in \N \text{.}\)

Base case.

If \(n = 0 \text{,}\) then
\begin{equation*} 0 + n = 0 + 0 = 0 = n\text{.} \end{equation*}
🔗
🔗

Inductive step.

Now suppose for the inductive hypothesis that \(0 + n = n \) for some natural number \(n \in \N \text{,}\) and note that
\begin{equation*} 0 + \of{ n + 1 } = \of{ 0 + n } + 1 = n + 1\text{.} \end{equation*}
🔗
🔗
We conclude that \(0 + n = n \) for all natural numbers \(n \in \N \text{.}\)
