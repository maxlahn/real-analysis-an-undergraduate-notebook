Lemma 1.3.2. \(0 \) is minimal.

For all natural numbers \(n \in \N \text{,}\) if \(n \leq 0 \text{,}\) then \(n = 0 \text{.}\)
🔗
🔗

Proof.
We first observe that \(\set{ 0 } \cup \im \of{ \Succ } = \N \text{.}\) Indeed, note that \(0 \in \set{ 0 } \subseteq \set{ 0 } \cup \im \of{ \Succ } \text{.}\) Moreover, if \(n \in \set{ 0 } \cup \im \of{ \Succ } \text{,}\) then
\begin{equation*} \Succ \of{ n } \in \im \of{ \Succ } \subseteq \set{ 0 } \cup \im \of{ \Succ }\text{.} \end{equation*}
Thus \(\set{ 0 } \cup \im \of{ \Succ } = \N \) by the axiom of induction.
🔗
Let \(n \in \N \) be a natural number, and suppose that \(n \leq 0 \) for some natural number \(n \in \N \text{.}\) Then \(n + m = 0 \) for some natural number \(m \in \N \text{.}\) Suppose for a contradiction that \(m = p + 1 \) for some natural number \(p \in \N \text{.}\) Then
\begin{equation*} 0 = n + m = n + \of{ p + 1 } = \of{ n + p } + 1\text{,} \end{equation*}
which contradicts the first Peano axiom. Thus \(m \notin \im \of{ \Succ } \text{.}\) However, \(m \in \N = \set{ 0 } \cup \im \of{ \Succ } \text{,}\) and so \(m \in \set{ 0 } \text{;}\) that is, \(m = 0 \text{,}\) and so
\begin{equation*} n = n + 0 = n + m = 0\text{.} \end{equation*}
