Proposition 1.6.1. Injections of finite sets.

For any natural numbers \(m , n \in \N \text{,}\) there exists an injective map \(\set{ 1 , \dotsc , m } \to \set{ 1 , \dotsc , n } \) if and only if \(m \leq n \text{.}\)
🔗
🔗

Proof.
If \(m \leq n \text{,}\) then \(\set{ 1 , \dotsc , m } \subseteq \set{ 1 , \dotsc , n } \text{,}\) and so the inclusion map \(\iota_{ \set{ 1 , \dotsc , m } }^{ \set{ 1 , \dotsc , n } } \colon \set{ 1 , \dotsc , m } \to \set{ 1 , \dotsc , n } \) is an injective map from \(\set{ 1 , \dotsc , m } \) to \(\set{ 1 , \dotsc , n } \text{.}\) So it suffices to show the reverse implication.
🔗
To that end, consider the subset
\begin{equation*} S = \setmid{ m \in \N }{ \textrm{There are } n \in \N \textrm{ and } f \colon \set{ 1 , \dotsc , m } \to \set{ 1 , \dotsc , n } \textrm{ so that } m \not \leq n \textrm{ and } f \textrm{ is injective} } \end{equation*}
of the natural numbers \(\N \text{.}\) If suffices to show that \(S = \emptyset \text{.}\) To that end, we first note that \(0 \leq n \) for all natural numbers \(n \in \N \text{,}\) so that \(0 \notin S \text{.}\)
🔗
Now let \(m \in \N \) be a natural number, and suppose that \(m + 1 \in S \text{.}\) Then there is a natural number \(n \in \N \) so that \(m + 1 \not \leq n \) and an injective function \(f \colon \set{ 1 , \dotsc , m + 1 } \to \set{ 1 , \dotsc , n } \text{.}\) We note that \(n \neq 0 \text{,}\) since there are no functions from a non-empty set \(\set{ 1 , \dotsc , m + 1 } \) to the empty set \(\emptyset \text{.}\) Thus \(n = p + 1 \) for some natural number \(p \in \N \text{.}\)
🔗
If \(m \leq p \text{,}\) then \(m + 1 \leq p + 1 = n \text{,}\) and so \(m \not \leq p \text{.}\) It therefore suffices to construct an injective function \(\set{ 1 , \dotsc , m } \to \set{ 1 , \dotsc , p } \text{.}\) To that end, fix a bijection \(g \colon \set{ 1 , \dotsc , p + 1 } \to \set{ 1 , \dotsc , p + 1 } \) so that \(g \of{ f \of{ m + 1 } } = p + 1 \text{,}\) and let \(h \colon \set{ 1 , \dotsc , m } \to \set{ 1 , \dotsc , p } \) be the function defined by the formula
\begin{equation*} h \of{ j } = g \of{ f \of{ j } }\text{.} \end{equation*}
Since \(g \) and \(f \) are injective, \(h \) is injective by (a) of [provisional cross-reference: problem-compositions-of-injections-surjections-and-bijections]. We conclude that \(n \in S \text{.}\)
🔗
In summary, we have shown that \(0 \notin S \) and for all natural numbers \(n \in \N \text{,}\) if \(n + 1 \in S \text{,}\) then \(n \in S \text{.}\) These conclusions are equivalent to \(0 \in \N \setminus S \) and for all natural numbers \(n \in \N \text{,}\) if \(n \in \N \setminus S \text{,}\) then \(n + 1 \in \N \setminus S \text{.}\) We conclude that \(\N \setminus S = \N \) by the axiom of induction, and so \(S = \emptyset \text{.}\)
