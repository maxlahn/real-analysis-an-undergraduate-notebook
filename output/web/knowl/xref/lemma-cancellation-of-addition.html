Lemma 1.3.14. Cancellation of addition.

For all natural numbers \(n \in \N \text{,}\) addition by \(n \) is injective; that is, for all natural numbers \(l , m , n \in \N \text{,}\) if \(l + n = m + n \text{,}\) then \(l = m \text{.}\)
🔗
🔗

Proof.
Let \(l , m \in \N \) be natural numbers. We proceed by induction on \(n \in \N \text{.}\)

Base case.

If \(n = 0 \) and \(l + n = m + n \text{,}\) then
\begin{equation*} l = l + 0 = l + n = m + n = m + 0 = m \end{equation*}
🔗
🔗

Inductive step.

Now suppose for the inductive hypothesis that there is some natural number \(n \in \N \) so that if \(l + n = m + n \text{,}\) then \(l = m \text{.}\) Note that if \(l + \of{ n + 1 } = m + \of{ n + 1 } \text{,}\) then
\begin{equation*} \of{ l + n } + 1 = l + \of{ n + 1 } = m + \of{ n + 1 } = \of{ m + n } + 1\text{,} \end{equation*}
and so \(l + n = m + n \) by the injectivity of the successor function. Thus \(l = m \) by the inductive hypothesis. In summary, we have shown that if \(l + \of{ n + 1 } = m + \of{ n + 1 } \text{,}\) then \(l = m \text{.}\)
🔗
🔗
We conclude that for all natural numbers \(n \in \N \text{,}\) if \(l + n = m + n \text{,}\) then \(l = m \text{.}\)
