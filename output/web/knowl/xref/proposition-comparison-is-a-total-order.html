Proposition 1.3.15. Comparison is a total order.

Comparison of natural numbers has the following properties:
  1. Reflexivity.
    \(n \leq n \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
  2. Antisymmetry.
    For all natural numbers \(m , n \in \N \text{,}\) if \(m \leq n \) and \(n \geq m \text{,}\) then \(m = n \text{.}\)
    🔗
    🔗
  3. Transitivity.
    For all natural numbers \(l , m , n \in \N \text{,}\) if \(l \leq m \) and \(m \leq n \text{,}\) then \(l \leq n \text{.}\)
    🔗
    🔗
  4. Totality.
    For all natural numbers \(m , n \in \N \text{,}\) either \(m \leq n \) or \(n \leq m \text{.}\)
    🔗
    🔗
🔗
🔗

Proof.
  1. For all natural numbers \(n \in \N \text{,}\) \(n + 0 = n \text{,}\) so that \(n \leq n \text{.}\)
    🔗
    🔗
  2. Let \(m , n \in \N \) be natural numbers, and suppose that \(m \leq n \) and \(n \leq m \text{.}\) Then \(n = m + p \) and \(m = n + q \) for some natural numbers \(p , q \in \N \text{.}\) Note that
    \begin{equation*} m + 0 = m = n + q = m + p + q\text{,} \end{equation*}
    and so \(0 = p + q \) by Lemma 1.3.14 Cancellation of addition; that is, \(q \leq 0 \text{.}\) Lemma 1.3.2 \(0 \) is minimal now implies that \(q = 0 \text{,}\) so that
    \begin{equation*} m = n + q = n + 0 = n\text{.} \end{equation*}
    🔗
    🔗
  3. Let \(l , m , n \in \N \) be natural numbers, and suppose that \(l \leq m \) and \(m \leq n \text{.}\) Then \(l + p = m \) and \(m + q = n \) for some natural numbers \(p , q \in \N \text{,}\) and so
    \begin{equation*} l + \of{ p + q } = \of{ l + p } + q = m + q = n\text{.} \end{equation*}
    We conclude that \(l \leq n \text{.}\)
    🔗
    🔗
  4. Let \(m \in \N \) be a natural number. We proceed by induction on \(n \in \N \text{.}\)

    Base case.

    If \(n = 0 \text{,}\) then \(0 + m = m \) by Lemma 1.3.5 Addition by \(0 \), and so \(n = 0 \leq m \text{.}\)
    🔗
    🔗

    Inductive step.

    Now suppose for the inductive hypothesis that there is a natural number \(n \in \N \) so that either \(m \leq n \) or \(n \leq m \text{.}\) If \(m \leq n \text{,}\) then \(m + p = n \) for some natural number \(p \in \N \text{,}\) and so
    \begin{equation*} m + \of{ p + 1 } = \of{ m + p } + 1 = n + 1\text{;} \end{equation*}
    that is, \(m \leq n + 1 \text{.}\)
    🔗
    Now suppose that \(n \leq m \text{,}\) so that \(n + p = m \) for some natural number \(p \in \N \text{.}\) If \(p = 0 \text{,}\) then
    \begin{equation*} m + 1 = \of{ n + p } + 1 = \of{ n + 0 } + 1 = n + 1\text{,} \end{equation*}
    and so \(m \leq n + 1 \text{.}\) On the other hand, if \(p \neq 0 \text{,}\) then \(p = q + 1 \) for some natural number \(q \in \N \) by an argument in the proof of Lemma 1.3.2 \(0 \) is minimal. Then
    \begin{equation*} \of{ n + 1 } + q = n + 1 + q = n + q + 1 = n + p = m\text{,} \end{equation*}
    and so \(n + 1 \leq m \text{.}\)
    🔗
    🔗
    We conclude that for all natural numbers \(n \in \N \text{,}\) either \(m \leq n \) or \(n \leq m \text{.}\)
    🔗
    🔗
