Proposition 1.3.10. Associativity of multiplication.

\(\of{ l m } n = l \of{ m n } \) for all natural numbers \(l , m , n \in \N \text{.}\)
🔗
🔗

Proof.
Let \(l , m \in \N \) be natural numbers. We proceed by induction on \(n \in \N \text{.}\)

Base case.

If \(n = 0 \text{,}\) then
\begin{equation*} \of{ l m } n = \of{ l m } 0 = 0 = l \cdot 0 = l \of{ m \cdot 0 } = l \of{ m n }\text{.} \end{equation*}
🔗
🔗

Inductive step.

Now suppose for the inductive hypothesis that \(\of{ l m } n = l \of{ m n } \) for some natural number \(n \in \N \text{,}\) and note that
\begin{align*} \of{ l m } \of{ n + 1 } \amp = \of{ l m } n + l m = l \of{ m n } + l m = \\ \amp = l \of{ m n + m } = l \of{ m \of{ n + 1 } } \end{align*}
by Proposition 1.3.9 Left distributivity of multiplication over addition.
🔗
🔗
We conclude that \(\of{ l m } n = l \of{ m n } \) for all natural numbers \(n \in \N \text{.}\)
