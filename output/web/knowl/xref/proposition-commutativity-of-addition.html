Proposition 1.3.7. Commutativity of addition.

\(m + n = n + m \) for all natural numbers \(m , n \in \N \text{.}\)
🔗
🔗

Proof.
Let \(m \in \N \) be a natural number. We proceed by induction on \(n \in \N \text{.}\)

Base case.

If \(n = 0 \text{,}\) then
\begin{equation*} m + n = m + 0 = m = 0 + m = n + m \end{equation*}
by Lemma 1.3.5 Addition by \(0 \).
🔗
🔗

Inductive step.

Now suppose for the inductive hypothesis that \(m + n = n + m \) for some natural number \(n \in \N \text{,}\) and note that
\begin{align*} m + \of{ n + 1 } \amp = \of{ m + n } + 1 = \of{ n + m } + 1 = 1 + \of{ n + m }\\ \amp = \of{ 1 + n } + m = \of{ n + 1 } + m \end{align*}
by Proposition 1.3.4 Associativity of addition and Lemma 1.3.6 Towards commutativity of addition.
🔗
🔗
We conclude that \(m + n = n + m \) for all natural numbers \(n \in \N \text{.}\)
