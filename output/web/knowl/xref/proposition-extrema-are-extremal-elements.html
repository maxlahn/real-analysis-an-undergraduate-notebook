Proposition 1.5.10. Extrema are extremal elements.

Let \(U \subseteq X \) be a subset of a partially ordered set \(\of{ X , \preceq } \text{.}\) If \(U \) has a maximum, then \(\max \of{ U } \) is the unique maximal element of \(U \text{.}\) If \(U \) has a minimum, then \(\min \of{ U } \) is the unique minimal element of \(U \text{.}\)
🔗
🔗

Proof.
First suppose that \(U \) has a maximum. Let \(u \in U \text{,}\) and suppose that \(\max \of{ U } \preceq u \text{.}\) Of course \(u \preceq \max \of{ U } \text{,}\) and so anti-symmetry implies that \(u = \max \of{ U } \text{;}\) that is, \(\max \of{ U } \) is maximal in \(U \text{.}\)
🔗
On the other hand, now consider a maximal element \(u \in U \text{.}\) Since \(u \preceq \max \of{ U } \text{,}\) the maximality of \(u \) implies that \(u = \max \of{ U } \text{.}\)
🔗
Now suppose that \(U \) has a minimum. Let \(u \in U \text{,}\) and suppose that \(u \preceq \min \of{ U } \text{.}\) Of course \(\min \of{ U } \preceq u \text{,}\) and so anti-symmetry implies that \(u = \min \of{ U } \text{;}\) that is, \(\min \of{ U } \) is minimal in \(U \text{.}\)
🔗
On the other hand, now consider a minimal element \(u \in U \text{.}\) Since \(\min \of{ U } \preceq u \text{,}\) the minimality of \(u \) implies that \(u = \min \of{ U } \text{.}\)
