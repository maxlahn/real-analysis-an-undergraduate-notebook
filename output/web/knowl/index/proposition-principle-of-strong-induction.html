Proposition 1.3.3. Principle of strong induction.

For all subsets \(U \subseteq \N \text{,}\) \(U = \N \) if and only if it satisfies the following conditions:
  1. \(0 \in U \text{.}\)
    🔗
    🔗
  2. For all natural numbers \(n \in \N \text{,}\) if \(k \in U \) for all \(k \in \N \) so that \(k \leq n \text{,}\) then \(n + 1 \in U \text{.}\)
    🔗
    🔗
🔗
🔗

Proof.
Clearly \(\N \) satisfies the above conditions, and so it suffices to show that any subsets which satisfies these conditions is equal to \(\N \text{.}\) To that end, let \(U \subseteq \N \) be a subset which satisfies the above conditions, and consider the subset
\begin{equation*} V = \setmid{ n \in \N }{ k \in U \textrm{ for all } k \in \N \textrm{ so that } k \leq n }\text{.} \end{equation*}
Since the only \(k \in \N \) so that \(k \leq 0 \) is \(k = 0 \) itself, and \(0 \in U \text{,}\) we have that \(0 \in V \text{.}\) Moreover if \(n \in V \text{,}\) then \(k \in U \) for all \(k \in \N \) so that \(k \leq n \text{,}\) and so \(n + 1 \in U \) by hypothesis. It remains to conclude that \(k \in U \) for all \(k \in \N \) so that \(k \leq n + 1 \text{.}\)
🔗
To that end, let \(k \in \N \text{,}\) and suppose that \(k \leq n + 1 \text{.}\) Then \(k + m = n + 1 \) for some natural number \(m \in \N \text{.}\) If \(m = 0 \text{,}\) then \(k = n + 1 \in U \text{.}\) On the other hand, if \(m \neq 0 \text{,}\) then the argument in the proof of Lemma 1.3.2 \(0 \) is minimal implies that \(m = p + 1 \) for some natural number \(p \in \N \text{.}\) Thus
\begin{equation*} \of{ k + p } + 1 = k + \of{ p + 1 } = k + m = n + 1\text{,} \end{equation*}
and so the injectivity of the successor function implies that \(k + p = n \text{;}\) that is, \(k \leq n \) and so \(k \in U \text{.}\) Since \(k \in U \) for all \(k \in \N \) so that \(k \leq n + 1 \text{,}\) we conclude that \(n + 1 \in V \text{.}\)
🔗
In summary, we have shown that \(0 \in V \) and that for all natural numbers \(n \in \N \text{,}\) if \(n \in V \text{,}\) then \(n + 1 \in V \text{.}\) Thus \(V = \N \) by the axiom of induction. One can check that \(\N = V \subseteq U \subseteq \N \text{,}\) and so \(U = \N \text{.}\)
