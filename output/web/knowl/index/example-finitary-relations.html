Example 1.4.11. Finitary relations.

The following are examples of finitary operations:
  1. In light of the convention \(X^{ 1 } = X \text{,}\) a \(1 \)-ary operation (or more simply, a unary operation) on a set \(X \) is just a subset of \(X \text{.}\)
    🔗
    🔗
  2. A \(2 \)-ary relation (or more simply, a binary relation) on a set \(X \) is a subset \(R \subseteq X^{ 2 } \text{.}\) Two examples of such binary relations are the containment relation \(\subseteq \) on the power set \(\ps \of{ X } \) of a set \(X \) and the standard ordering \(\Comp \subseteq \N^{ 2 } \) of the natural numbers \(\N \text{.}\)
    🔗
    Just as there is a special notation for binary operations, there is a special relation for binary relations as well. Specifically, for a given binary relation \(R \subseteq X^{ 2 } \) on a set \(X \text{,}\) we write \(x_{ 1 } \mathrel{ R } x_{ 2 } \) to mean that \(\of{ x_{ 1 } , x_{ 2 } } \in R \text{.}\)
