Definition 1.4.8. Commutativity; associativity.

Let \(\ast \colon X^{ 2 } \to X \) be a binary operation on a set \(X \text{.}\)
  1. \(\ast \) is called commutative if
    \begin{equation*} x_{ 1 } \ast x_{ 2 } = x_{ 2 } \ast x_{ 1 } \end{equation*}
    for all elements \(x_{ 1 } , x_{ 2 } \in X \text{.}\)

    🔗
  2. \(\ast \) is called associative if
    \begin{equation*} \of{ x_{ 1 } \ast x_{ 2 } } \ast x_{ 3 } = x_{ 1 } \ast \of{ x_{ 2 } \ast x_{ 3 } } \end{equation*}
    for all elements \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{.}\)
