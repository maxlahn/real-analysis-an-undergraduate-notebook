Definition 1.3.1. Natural numbers.

The natural number system \(\N \) is a nonempty set with an element \(0 \in \N \text{,}\) called zero, along with functions \(\Succ \colon \N \to \N \) and \(\Add , \Mult \colon \N \times \N \to \N \text{,}\) called the successor function and the addition and multiplication operations and a subset \(\Comp \subseteq \N \times \N \) of the Cartesian product \(\N \times \N \) of \(\N \) with itself, called the standard ordering, which satisfy the following properties:
  1. The successor \(\Succ \of{ n } \) of any natural number \(n \in \N \) is non-zero.
    🔗
    🔗
  2. The successor function \(\Succ \colon \N \to \N \) is injective.
    🔗
    🔗
  3. \(\Add \of{ n , 0 } = n \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
  4. \(\Add \of{ m , \Succ \of{ n } } = \Succ \of{ \Add \of{ m , n } } \) for all natural numbers \(m , n \in \N \text{.}\)
    🔗
    🔗
  5. \(\Mult \of{ n , 0 } = 0 \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
  6. \(\Mult \of{ m , \Succ \of{ n } } = \Add \of{ \Mult \of{ m , n } , m } \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
  7. For all natural numbers \(l , m \in \N \text{,}\) \(\of{ l , m } \in \Comp \) if and only if \(\Add \of{ l , n } = m \) for some natural number \(n \in \N \text{.}\)
    🔗
    🔗
  8. For all subsets \(U \subseteq \N \text{,}\) \(U = \N \) if and only if it satisfies the following conditions:
    1. \(0 \in U \text{.}\)
      🔗
      🔗
    2. For all natural numbers \(n \in \N \text{,}\) if \(n \in U \text{,}\) then \(\Succ \of{ n } \in U \text{.}\)
      🔗
      🔗
    🔗
    🔗
Typically, for all natural numbers \(m , n \in \N \) we write \(m + n = \Add \of{ m , n } \) and \(m n = m \cdot n = \Mult \of{ m , n } \text{,}\) and we write \(m \leq n \) if and only if \(\of{ m , n } \in \Comp \text{.}\) The successor of \(0 \) is denoted \(1 = \Succ \of{ 0 } \text{.}\) Likewise, the successor of \(1 \) is denoted \(2 = \Succ \of{ 1 } \text{,}\) the successor of \(2 \) is denoted \(3 = \Succ \of{ 2 } \text{,}\) and so on. We see that
\begin{equation*} \Succ \of{ n } = \Succ \of{ \Add \of{ n , 0 } } = \Add \of{ n , \Succ \of{ 0 } } = n + 1 \end{equation*}
for all natural numbers \(n \in \N \text{.}\) With these new notations, we may restate the Peano axioms as follows:
  1. \(n + 1 \neq 0 \) for all \(n \in \N \text{.}\)
    🔗
    🔗
  2. For all natural numbers \(m , n \in \N \text{,}\) if \(m + 1 = n + 1 \text{,}\) then \(m = n \text{.}\)
    🔗
    🔗
  3. \(n + 0 = n \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
  4. \(m + \of{ n + 1 } = \of{ m + n } + 1 \) for all natural numbers \(m , n \in \N \text{.}\)
    🔗
    🔗
  5. \(n \cdot 0 = 0 \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
  6. \(m \of{ n + 1 } = m n + m \) for all natural numbers \(n \in \N \text{.}\)
    🔗
    🔗
  7. For all natural numbers \(l , m \in \N \text{,}\) \(l \leq m \) if and only if \(l + n = m \) for some natural number \(n \in \N \text{.}\)
    🔗
    🔗
  8. For all subsets \(U \subseteq \N \text{,}\) \(U = \N \) if and only if it satisfies the following conditions:
    1. \(0 \in U \text{.}\)
      🔗
      🔗
    2. For all natural numbers \(n \in \N \text{,}\) if \(n \in U \text{,}\) then \(n + 1 \in U \text{.}\)
      🔗
      🔗
    🔗
    🔗
🔗
