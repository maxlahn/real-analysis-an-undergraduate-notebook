Definition 1.5.7. Extrema.

Let \(U \subseteq X \) be a subset of a partially ordered set \(\of{ X , \preceq } \text{.}\)
  1. If there is an element of \(U \) which is an upper bound for \(U \) in \(X \text{,}\) then this unique element is called the maximum of \(U \) and is denoted \(\max \of{ U } \text{.}\)
    🔗
    🔗
  2. If there is an element of \(U \) which is a lower bound for \(U \) in \(X \text{,}\) then this unique element is called the minimum of \(U \) and is denoted \(\min \of{ U } \text{.}\)
    🔗
    🔗
Together, maxima and minima are called extrema.
