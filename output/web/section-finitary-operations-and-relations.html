    Section 1.4 Finitary Operations and Relations

    In this section, we will develop the notion of (finitary) operations and relations on sets. Intuitively, a finitary operation on a set is an assignment of an element of that set to some finite number of other elements, and a finitary relation on a set is a designation of certain groups of some finite number of elements as associated.
    🔗

    Subsection 1.4.1 Products and Choice

    Previously, we have used the mathematical convention of indexing to generalize the intersection and union of two sets. We may also use indexing to generalize the Cartesian product of two sets to act on arbitrary indexed families of sets. In this section, we will develop this generalized Cartesian product, and use it to introduce operations and relations on a set. Instead of containing ordered pairs of elements, our new Cartesian product consists of indexed families of elements.
    🔗

    Definition 1.4.1. Cartesian product.

    Let \(\of{ U_{ j } }_{ j \in \mathcal{ J } } \) be a family of subsets \(U_{ j } \subseteq X \) of a set \(X \text{,}\) called factors, indexed by a set \(\mathcal{ J } \text{.}\) The Cartesian product \(\prod_{ j \in \mathcal{ J } }{ U_{ j } } \) of this indexed family is the set
    \begin{equation*} \prod_{ j \in \mathcal{ J } }{ U_{ j } } = \setmid{ \of{ u_{ j } }_{ j \in \mathcal{ J } } \in X^{ \mathcal{ J } } }{ u_{ j } \in U_{ j } \textrm{ for all } j \in \mathcal{ J } } \end{equation*}
    of families \(\of{ u_{ j } }_{ j \in \mathcal{ J } } \) of elements \(u_{ j } \in X \) with the property that \(u_{ j } \in U_{ j } \) for all indices \(j \in \mathcal{ J } \text{.}\)
    🔗
    For each index \(k \in \mathcal{ J } \text{,}\) the \(k \)th projection map \(\pi_{ \prod_{ j \in \mathcal{ J } }{ U_{ j } } }^{ U_{ k } } \colon \prod_{ j \in \mathcal{ J } }{ U_{ j } } \to U_{ k } \) is the map from the Cartesian product \(\prod_{ j \in \mathcal{ J } }{ U_{ j } } \) to the \(k \)th factor \(U_{ k } \) defined by the formula
    \begin{equation*} \pi_{ \prod_{ j \in \mathcal{ J } }{ U_{ j } } }^{ U_{ k } } \of{ u_{ j } }_{ j \in \mathcal{ J } } = u_{ k }\text{.} \end{equation*}
    🔗
    🔗

    Convention 1.4.2. Empty and singleton products.

    What does a family indexed by the empty set look like? The empty set \(\emptyset \colon \emptyset \to X \) is the unique map from the empty set \(\emptyset \) to a set \(X \text{.}\) Therefore, there is only one such family
    \begin{equation*} \of{ x_{ j } }_{ j \in \emptyset } = \emptyset \end{equation*}
    of elements \(x_{ j } \) of \(X \) indexed by the empty set \(\emptyset \text{.}\) This unique indexed family is usually denoted \(\of{} = \emptyset \text{.}\) By the above remark,
    \begin{equation*} \prod_{ j \in \emptyset }{ U_{ j } } = \set{ \of{} } = \set{ \emptyset }\text{.} \end{equation*}
    Similarly, a family \(\of{ x_{ j } }_{ j \in \set{ j } } \) of elements \(x_{ j } \in X \) of a set \(S \) indexed by a singleton set \(\set{ j } \) is uniquely determined by the image \(x_{ j } \) of \(j \text{.}\) Such an indexed family is denoted
    \begin{equation*} \of{ x_{ j } } = \of{ x_{ j } }_{ j \in \set{ j } } . \end{equation*}
    We usually act under the convention \(\of{ x_{ j } } = x_{ j } \text{,}\) so that
    \begin{equation*} \prod_{ j \in \set{ j } }{ U_{ j } } = U_{ j } . \end{equation*}
    🔗
    🔗
    It may seem intuitive or obvious that the Cartesian product of an indexed family of nonempty sets is itself nonempty. After all, one can specify an indexed family of elements of these sets just by choosing an element of each nonempty set. This notion of a choice of elements of nonempty sets gives the following axiom its name.
    🔗

    Axiom 1.4.3. Axiom of choice.

    Let \(\of{ U_{ j } }_{ j \in \mathcal{ J } } \) be an indexed family of subsets \(U_{ j } \subseteq X \) of a set \(X \text{.}\) If \(U_{ j } \neq \emptyset \) for all indices \(j \in \mathcal{ J } \text{,}\) then the Cartesian product \(\prod_{ j \in \mathcal{ J } }{ U_{ j } } \neq \emptyset \) is also non-empty.
    🔗
    🔗
    We will take the Axiom of choice as true for the remainder of this text. While seemingly too specific to be of much use, the axiom of choice can be used to prove many useful results. In particular, we can use it to complete the characterization of invertibility given in Theorem 1.2.15 Invertibility.
    🔗

    Corollary 1.4.4. Invertibility.

    Let \(f \colon X \to Y \) be a map from a set \(X \) to a set \(Y \text{.}\)
    1. \(f \) is left-invertible if and only if \(f \) is injective or \(X = \emptyset \) is empty.
      🔗
      🔗
    2. \(f \) is right-invertible if and only if \(f \) is surjective.
      🔗
      🔗
    3. \(f \) is invertible if and only if \(f \) is bijective.
      🔗
      🔗
    🔗
    🔗

    Proof.
    1. This follows directly from (1) and (2) of Theorem 1.2.15 Invertibility.
      🔗
      🔗
    2. The forward implication is (3) of Theorem 1.2.15 Invertibility. Thus we may suppose that \(f \) is surjective. Then for each output \(y \in Y \text{,}\) the pre-image
      \begin{equation*} f^{ -1 } \of{ y } = \setmid{ x \in X }{ f \of{ x } = y } \neq \emptyset \end{equation*}
      is a non-empty subset of \(X \text{,}\) and so Axiom 1.4.3 Axiom of choice implies that the Cartesian product
      \begin{equation*} \prod_{ y \in Y }{ f^{ -1 } \of{ y } } \neq \emptyset \end{equation*}
      is a non-empty subset of \(X^{ Y } \text{.}\) Choose an element \(g = \of{ x_{ y } }_{ y \in Y } \in \prod_{ y \in Y }{ f^{ -1 } \of{ y } } \text{,}\) and note that for all \(y \in Y \text{,}\) \(g \of{ y } = x_{ y } \in f^{ -1 } \of{ y } \text{,}\) so that
      \begin{equation*} \of{ f \circ g } \of{ y } = f \of{ g \of{ y } } = y = \id_{ Y } \of{ y }\text{;} \end{equation*}
      that is, \(f \circ g = \id_{ Y } \) so that \(g \) is a right inverse to \(f \text{.}\) Hence \(f \) is right-invertible.
      🔗
      🔗
    3. This is (3) of Theorem 1.2.15 Invertibility.
      🔗
      🔗
    🔗

    Definition 1.4.5. Cartesian power.

    Let \(X \) be a set and \(m \in \N \) be a natural number. The \(m \)th Cartesian power \(X^{ m } \) of \(X \) is the Cartesian product
    \begin{equation*} X^{ m } = \prod_{ j = 1 }^{ m }{ X } = \setmid{ \of{ x_{ j } }_{ j = 1 }^{ m } }{ x_{ j } \in X \textrm{ for all } 1 \leq j \leq m }\text{.} \end{equation*}
    🔗
    🔗
    We will now move on to study the Cartesian powers of sets. Maps from the Cartesian power of a set to the set itself will be called finitary operations on that set, and subsets of the Cartesian power of a set will be called finitary relations on that set.
    🔗
    🔗

    Subsection 1.4.2 Finitary Operations

    Informally, a finitary operation on a set is an assignment of its elements to all finite sequence of a certain length in that set. That is, a finitary operation on a set is a map from some finite Cartesian power of that set to the set itself. We will view a finitary operation (through the active characterization of what maps do) as a process which combines elements of a set into a new element.
    🔗

    Definition 1.4.6. Finitary operation.

    Let \(X \) be a set and \(n \in \N \) be a natural number. A finitary operation of arity \(n \) (or more simply a \(n \)-ary operation) is a map \(f \colon X^{ n } \to X \text{.}\)
    🔗
    🔗

    Example 1.4.7. Finitary operations.

    The following are examples of finitary operations:
    1. A \(0 \)-ary operation (or more simply, a nullary operation) on a set \(X \) is just a map \(f \colon \set{ \emptyset } \to X \) from \(X^{ 0 } = \set{ \emptyset } \) to \(X \text{.}\) Such a map \(f \colon \set{ \emptyset } \to X \) is uniquely determined by the image \(f \of{ \emptyset } \in X \) of the unique element \(\emptyset \in X^{ 0 } \text{.}\) In this way, we may view the nullary operations on \(X \) as synonymous with the elements of \(X \text{.}\)
      🔗
      🔗
    2. A \(1 \)-ary operation (or more simply, a unary operation) on a set \(X \) is just a map \(f \colon X \to X \text{.}\) One example of such a unary operation on the natural numbers \(\N \) is the successor function \(\Succ \colon \N \to \N \text{.}\)
      🔗
      🔗
    3. A \(2 \)-ary operation (or more simply, a binary operation) on a set \(X \) is a map \(f \colon X^{ 2 } \to X \text{.}\) Two examples of such binary operations on the natural numbers \(\N \) are the addition and multiplication operations \(\Add , \Mult \colon \N^{ 2 } \to \N \text{.}\)
      🔗
      Binary operations are usually denoted with symbols such as \(+ \text{,}\) \(\times \text{,}\) \(\cdot \text{,}\) \(\ast \text{,}\) and \(\star \text{.}\) For a given binary operation \(\ast \colon X \times X \to X \text{,}\) we write
      \begin{equation*} x_{ 1 } \ast x_{ 2 } = \ast \of{ x_{ 1 } , x_{ 2 } } \end{equation*}
      for all elements \(x_{ 1 } , x_{ 2 } \in X \text{.}\)
      🔗
      🔗
    🔗
    🔗
    In many ways, algebra is the study of finitary operations on sets. Particularly emphasized in group theory and ring theory is the study of binary operation on sets. We won’t delve too deep into the study of binary operations, but we will introduce some nice properties which some binary operations may satisfy.
    🔗

    Definition 1.4.8. Commutativity; associativity.

    Let \(\ast \colon X^{ 2 } \to X \) be a binary operation on a set \(X \text{.}\)
    1. \(\ast \) is called commutative if
      \begin{equation*} x_{ 1 } \ast x_{ 2 } = x_{ 2 } \ast x_{ 1 } \end{equation*}
      for all elements \(x_{ 1 } , x_{ 2 } \in X \text{.}\)
      🔗
      🔗
    2. \(\ast \) is called associative if
      \begin{equation*} \of{ x_{ 1 } \ast x_{ 2 } } \ast x_{ 3 } = x_{ 1 } \ast \of{ x_{ 2 } \ast x_{ 3 } } \end{equation*}
      for all elements \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Example 1.4.9. Commutative/associative binary operations.

    The following are examples of commutativity and associativity in binary operations:
    1. Addition \(+ \colon \N^{ 2 } \to \N \) and multiplication \(\cdot \colon \N^{ 2 } \to \N \) are commutative and associative binary operations on \(\N \) by (1), (2), (4), and (5) of Theorem 1.3.18 Properties of Peano arithmetic.
      🔗
      🔗
    2. Intersection \(\cap \colon \ps \of{ X }^{ 2 } \to \ps \of{ X } \) and union \(\cup \colon \ps \of{ X }^{ 2 } \to \ps \of{ X } \) are commutative and associative binary operations on the power set \(\ps \of{ X } \) of a set \(X \) by (2) and (3) of Corollary 1.1.15 Properties of the set operations.
      🔗
      🔗
    3. Composition \(\circ \colon \of{ X^{ X } }^{ 2 } \to X^{ X } \) is an associative but non-commutative binary operation on the set \(X^{ X } \) of maps from a set \(X \) to itself.
      🔗
      🔗
    4. Let \(e \colon \N^{ 2 } \to \N \) be the exponentiation function defined by the formula \(e \of{ m , n } = m^{ n } \text{;}\) that is, \(e \) is defined recursively by the formulae \(e \of{ m , 0 } = 1 \) and \(e \of{ m , n + 1 } = e^{ m , n } \cdot m \text{.}\) Then \(e \) is a binary operation on \(\N \text{,}\) but \(e \) is neither commutative nor associative.
      🔗
      🔗
    🔗
    🔗
    Associative binary operations can be extended recursively to act on finite sequences of any length. For example, given a finite sequence \(\of{ n_{ j } }_{ j = 1 }^{ m } \) of natural numbers \(n_{ j } \in \N \text{,}\) we write
    \begin{align*} \sum_{ j = 1 }^{ m }{ n_{ j } } \amp = n_{ 1 } + n_{ 2 } + \dotsb + n_{ m } \amp \prod_{ j = 1 }^{ m }{ n_{ j } } \amp = n_{ 1 } n_{ 2 } \dotsm n_{ m }\text{.} \end{align*}
    🔗
    As in the case of finitary operations, we will introduce finitary relations of all arities, but we will emphasize binary relations in order to study congruences and order relations, which are ubiquitous throughout both classical and modern mathematics and which will recur throughout this text.
    🔗
    🔗

    Subsection 1.4.3 Finitary Relations

    A finitary relation on a set is just a subset of some finite Cartesian power of the set. This subset is usually interpreted as distinguishing its elements certain finite sequences of elements of the set as important in some way.
    🔗

    Definition 1.4.10. Finitary relation.

    Let \(X \) be a set and \(n \in \N \) be a natural number. A finitary relation of arity \(n \) (or more simply a \(n \)-ary relation) is a subset \(R \subseteq X^{ n } \text{.}\)
    🔗
    🔗
    We will view a finitary relation on a set as elevating specific finite sequences of elements of that set as distinguished.
    🔗

    Example 1.4.11. Finitary relations.

    The following are examples of finitary operations:
    1. In light of the convention \(X^{ 1 } = X \text{,}\) a \(1 \)-ary operation (or more simply, a unary operation) on a set \(X \) is just a subset of \(X \text{.}\)
      🔗
      🔗
    2. A \(2 \)-ary relation (or more simply, a binary relation) on a set \(X \) is a subset \(R \subseteq X^{ 2 } \text{.}\) Two examples of such binary relations are the containment relation \(\subseteq \) on the power set \(\ps \of{ X } \) of a set \(X \) and the standard ordering \(\Comp \subseteq \N^{ 2 } \) of the natural numbers \(\N \text{.}\)
      🔗
      Just as there is a special notation for binary operations, there is a special relation for binary relations as well. Specifically, for a given binary relation \(R \subseteq X^{ 2 } \) on a set \(X \text{,}\) we write \(x_{ 1 } \mathrel{ R } x_{ 2 } \) to mean that \(\of{ x_{ 1 } , x_{ 2 } } \in R \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Definition 1.4.12. Pullback of a finitary relation.

    Let \(X \) and \(Y \) be sets and \(n \in \N \) be a natural number. The pullback \(R_{ f } \) of an \(n \)-ary relation \(R \) on \(Y \) along a map \(f \colon X \to Y \) is the \(n \)-ary relation on \(X \) defined by
    \begin{equation*} R_{ f } = \setmid{ \of{ x_{ k } }_{ k = 1 }^{ n } \in X^{ n } }{ \of{ f \of{ x_{ k } } }_{ k = 1 }^{ n } \in R }\text{.} \end{equation*}
    🔗
    🔗

    Example 1.4.13. Restriction of a finitary relation.

    Let \(X \) be a set and \(n \in \N \) be a natural number. The restriction \(\rest{ R }_{ U } \) of an \(n \)-ary relation \(R \) on \(X \) to a subset \(U \subseteq X \) is the pullback of \(R \) along the inclusion map \(\iota_{ U }^{ X } \colon U \to X \text{;}\) that is,
    \begin{align*} \rest{ R }_{ U } \amp = R_{ \iota_{ U }^{ X } } = \setmid{ \of{ u_{ k } }_{ k = 1 }^{ n } \in U^{ n } }{ \of{ \iota_{ U }^{ X } \of{ u_{ k } } }_{ k = 1 }^{ n } \in R } = R \cap U^{ n }\\ \amp = \setmid{ \of{ x_{ k } }_{ k = 1 }^{ n } \in X^{ n } }{ x_{ 1 } , \dotsc , x_{ n } \in U \textrm{ and } \of{ x_{ k } }_{ k = 1 }^{ n } \in R } = R \cap U^{ n }\text{.} \end{align*}
    🔗
    🔗
    One particularly nice type of binary relation is the equivalence relation. This is a generalization of the notion of equality, which has many of the salient properties of the equality relation. We will see that an equivalence relation partitions a set into disjoint subsets known as equivalence classes. When we consider more sophisticated structures than just sets, we will use equivalence relations to construct quotient objects, as is done in algebraic settings such as group theory or ring theory.
    🔗

    Definition 1.4.14. Equivalence relation.

    Let \(\equiv \) be a binary relation on a set \(X \text{.}\)
    1. \(\equiv \) is called reflexive if \(x \equiv x \) for all elements \(x \in X \text{.}\)
      🔗
      🔗
    2. \(\equiv \) is called symmetric if for all elements \(x_{ 1 } , x_{ 2 } \in X \text{,}\) if \(x_{ 1 } \equiv x_{ 2 } \text{,}\) then \(x_{ 2 } \equiv x_{ 1 } \text{.}\)
      🔗
      🔗
    3. \(\equiv \) is called transitive if for all elements \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{,}\) if \(x_{ 1 } \equiv x_{ 2 } \) and \(x_{ 2 } \equiv x_{ 3 } \text{,}\) then \(x_{ 1 } \equiv x_{ 3 } \text{.}\)
      🔗
      🔗
    \(\equiv \) is called an equivalence relation if it is reflexive, symmetric, and transitive.
    🔗
    🔗

    Lemma 1.4.15. Pullbacks of equivalence relations.

    The pullback \(\equiv_{ f } \) of an equivalence relation \(\equiv \) on a set \(Y \) along a map \(f \colon X \to Y \) from a set \(X \) to \(Y \) is an equivalence relation on \(X \text{.}\)
    🔗
    🔗

    Proof.
    First let \(x \in X \text{,}\) and note that \(f \of{ x } \equiv f \of{ x } \) by the reflexivity of \(\equiv \text{,}\) and so \(x \equiv_{ f } x \text{;}\) that is, \(\equiv_{ f } \) is reflexive.
    🔗
    Now let \(x_{ 1 } , x_{ 2 } \in X \text{,}\) and suppose that \(x_{ 1 } \equiv_{ f } x_{ 2 } \text{.}\) Then \(f \of{ x_{ 1 } } \equiv f \of{ x_{ 2 } } \text{,}\) and so \(f \of{ x_{ 2 } } \equiv f \of{ x_{ 1 } } \) by the symmetry of \(\equiv \text{.}\) In particular, \(x_{ 2 } \equiv_{ f } x_{ 1 } \text{,}\) and so \(\equiv_{ f } \) is symmetric.
    🔗
    Finally, let \(x_{ 1 } , x_{ 2 } , x_{ 3 } \in X \text{,}\) and suppose that \(x_{ 1 } \equiv_{ f } x_{ 2 } \) and \(x_{ 2 } \equiv_{ f } x_{ 3 } \text{.}\) Then \(f \of{ x_{ 1 } } \equiv f \of{ x_{ 2 } } \) and \(f \of{ x_{ 2 } } \equiv f \of{ x_{ 3 } } \text{,}\) and so \(f \of{ x_{ 1 } } \equiv f \of{ x_{ 3 } } \) by the transitivity of \(\equiv \text{.}\) In particular, \(x_{ 1 } \equiv_{ f } x_{ 3 } \text{,}\) and so \(\equiv_{ f } \) is transitive.
    🔗
    Since \(\equiv_{ f } \) is reflexive, symmetric, and transitive, it is an equivalence relation on \(X \text{.}\)
    🔗
    🔗

    Definition 1.4.16. Equivalence class; quotient set.

    Let \(\equiv \) be an equivalence relation on a set \(X \text{.}\) The equivalence class \(\class{ x }_{ \equiv } \) of an element \(x \in X \) is the subset of \(X \) defined by
    \begin{equation*} \class{ x }_{ \equiv } = \setmid{ y \in X }{ x \equiv y }\text{;} \end{equation*}
    that is, \(\class{ x }_{ \equiv } \) is the set of elements which are equivalent to \(x \) under the relation \(\equiv \text{.}\) The quotient \(X / { \equiv } \) of \(X \) by \(\equiv \) is the set of equivalence classes; that is,
    \begin{equation*} X / { \equiv } = \setmid{ \class{ x }_{ \equiv } \in \ps \of{ X } }{ x \in X }\text{.} \end{equation*}
    🔗
    🔗
    As mentioned above, an equivalence relation on a given set partitions that set into pairwise disjoint equivalence classes.
    🔗

    Proposition 1.4.17. Equivalence partition.

    The equivalence classes of an equivalence relation \(\equiv \) on a set \(X \) form a partition of \(X \text{;}\) that is, two equivalence classes are either equal or disjoint, and
    \begin{equation*} X = \bigcup_{ c \in X / { \equiv } }{ c }\text{.} \end{equation*}
    🔗
    🔗

    Proof.
    Each equivalence class is a subset of \(X \text{,}\) and so
    \begin{equation*} \bigcup_{ c \in X / { \equiv } }{ c } \subseteq X\text{.} \end{equation*}
    In order to show the reverse containment, let \(x \in X \text{,}\) and note that \(x \equiv x \) by reflexivity; that is, \(x \in \class{ x }_{ \equiv } \text{,}\) and so
    \begin{equation*} x \in \class{ x }_{ \equiv } = \bigcup_{ c \in X / { \equiv } }{ c }\text{.} \end{equation*}
    We conclude that
    \begin{equation*} X = \bigcup_{ c \in X / { \equiv } }{ c }\text{.} \end{equation*}
    🔗
    It remains to show that two equivalence classes are either equal or disjoint. To that end, consider elements \(x_{ 1 } , x_{ 2 } \in X \text{.}\) First suppose that \(x_{ 1 } \equiv x_{ 2 } \text{,}\) and let \(x \in \class{ x_{ 2 } }_{ \equiv } \text{.}\) Then \(x_{ 2 } \equiv x \text{,}\) and so transitivity implies that \(x_{ 1 } \equiv x \text{;}\) that is, \(x \in \class{ x_{ 1 } }_{ \equiv } \text{.}\)
    🔗
    In summary, we have shown that if \(x_{ 1 } \equiv x_{ 2 } \text{,}\) then \(\class{ x_{ 2 } }_{ \equiv } \subseteq \class{ x_{ 1 } }_{ \equiv } \text{.}\) Symmetry implies that if \(x_{ 1 } \equiv x_{ 2 } \text{,}\) then \(x_{ 2 } \equiv x_{ 1 } \text{,}\) and so \(\class{ x_{ 1 } }_{ \equiv } \subseteq \class{ x_{ 2 } }_{ \equiv } \text{;}\) that is, if \(x_{ 1 } \equiv x_{ 2 } \text{,}\) then \(\class{ x_{ 1 } }_{ \equiv } = \class{ x_{ 2 } }_{ \equiv } \text{.}\)
    🔗
    Conversely, now suppose that the equivalence classes \(\class{ x_{ 1 } }_{ \equiv } \) and \(\class{ x_{ 2 } }_{ \equiv } \) are not disjoint, so that \(x \in \class{ x_{ 1 } }_{ \equiv } \cap \class{ x_{ 2 } }_{ \equiv } \) for some element \(x \in X \text{.}\) Since \(x \in \class{ x_{ 1 } }_{ \equiv } \) and \(x \in \class{ x_{ 2 } }_{ \equiv } \text{,}\) \(x \equiv x_{ 1 } \) and \(x \equiv x_{ 2 } \text{.}\) We conclude by symmetry and transitivity that \(x_{ 1 } \equiv x_{ 2 } \text{.}\)
    🔗
    🔗
    Equivalence relations are ubiquitous throughout mathematics, and we will work with them for the remainder of this text.
    🔗
