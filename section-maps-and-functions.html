    Section 1.2 Maps and Functions

    In this section, we introduce the fundamental notion of a map between sets. Informally, a map is an assignment of an element of one set to each element of another. You probably have been exposed to maps in previous math courses in the form of real-valued functions of one real variable. We will treat maps in far greater generality, which will allow us to perform mathematical analysis in a wider array of spaces.
    🔗

    Subsection 1.2.1 Introduction to Maps

    First, however, we define the Cartesian product of two sets. This relies on a notion of an ordered pair of objects \(x \) and \(y \text{,}\) which we denote \(\of{ x , y } \text{.}\) Like formalized set theory, rigorous constructions of ordered pairs are, in this context, unnecessarily complicated and annoying to work with. As such, we will take them to be well-defined. The salient properties of an ordered pair \(\of{ x , y } \) are its entries \(x \) and \(y \text{,}\) which are called its first and second entries, respectively.
    🔗

    Definition 1.2.1. Cartesian product.

    The Cartesian product \(X \times Y \) of two sets \(X \) and \(Y \) is the set which contains all ordered pairs whose first entry is an element of \(X \) and whose second entry is an element of \(Y \text{;}\) that is,
    \begin{equation*} X \times Y = \setmid{ \of{ x , y } }{ x \in X \textrm{ and } y \in Y }\text{.} \end{equation*}
    🔗
    🔗

    Example 1.2.2. An example of Cartesian products.

    Let \(X = \set{ 1 , 2 } \) and \(Y = \set{ 3 , 4 , 5 } \text{.}\) Then
    \begin{equation*} X \times Y = \set{ \of{ 1 , 3 } , \of{ 1 , 4 } , \of{ 1 , 5 } , \of{ 2 , 3 } , \of{ 2 , 4 } , \of{ 2 , 5 } }\text{.} \end{equation*}
    Note that \(X \times Y \neq X \cup Y \text{.}\)
    🔗
    🔗
    Recall that we would like a map between two sets to be an association of the elements of one set to the elements of the other. We will model this association via ordered pairs. To that end, a map between two sets will just be a subset of their Cartesian product which satisfies certain existence and uniqueness properties.
    🔗

    Definition 1.2.3. Map; function.

    A subset \(f \subseteq X \times Y \) of the Cartesian product \(X \times Y \) of two sets \(X \) and \(Y \) is called a map from \(X \) (which is called the domain of \(f \) and whose elements are called inputs) to \(Y \) (which is called the codomain of \(f \) and whose elements are called outputs) if it satisfies the following conditions:
    1. Totality: For all inputs \(x \in X \text{,}\) there is an output \(y \in Y \) so that \(\of{ x , y } \in f \text{.}\)
      🔗
      🔗
    2. Uniqueness: For all inputs \(x \in X \) and outputs \(y_{ 1 } , y_{ 2 } \in Y \text{,}\) if \(\of{ x , y_{ 1 } } , \of{ x , y_{ 2 } } \in f \text{,}\) then \(y_{ 1 } = y_{ 2 } \text{.}\)
      🔗
      🔗
    So \(f \) is a map from \(X \) to \(Y \) if and only if for each input \(x \in X \text{,}\) there is a unique output \(y \in Y \) so that \(\of{ x , y } \in f \text{.}\) We will write \(f \colon X \to Y \) to indicate that \(f \) is a map from \(X \) to \(Y \text{,}\) and we will write \(f \colon x \mapsto y \) to indicate that \(\of{ x , y } \in f \text{.}\) The set of maps from \(X \) to \(Y \) is denoted \(Y^{ X } \subseteq X \times Y \text{.}\)
    🔗
    We will call a map \(f \colon X \to Y \) whose codomain \(Y \) is a number system like the natural numbers \(Y = \N \text{,}\) the integers \(Y = \Z \text{,}\) the rational numbers \(Y = \Q \text{,}\) the real numbers \(Y = \R \text{,}\) or the complex numbers \(Y = \C \) a \(Y \)-valued function on \(X \text{.}\) We will investigate the natural numbers \(\N \) in [provisional cross-reference: section-peano-arithmetic-and-the-natural-numbers], and [provisional cross-reference: chapter-constructing-the-number-systems] will be dedicated to constructing the other number systems with which we will become acquainted throughout the course of this text.
    🔗
    🔗
    So a map between sets is just a subset of their Cartesian product such that each element of the domain appears in precisely one pair with an element of the codomain. The above existence and uniqueness properties of a map implies that there is an unambiguous association between the elements of the domain and the elements of the codomain.
    🔗

    Example 1.2.4. Examples of maps.

    The following are examples of maps between sets:
    1. Let \(X = \set{ 1 , 2 } \) and \(Y = \set{ \mathrm{ red } , \mathrm{ blue } } \text{,}\) and consider the following subsets of \(X \times Y \text{:}\)
      \begin{align*} f_{ 1 } \amp = \set{ \of{ 1 , \mathrm{ red } } , \of{ 2 , \mathrm{ red } } }\\ f_{ 2 } \amp = \set{ \of{ 1 , \mathrm{ red } } , \of{ 2 , \mathrm{ blue } } }\\ f_{ 3 } \amp = \set{ \of{ 1 , \mathrm{ blue } } , \of{ 2 , \mathrm{ red } } }\\ f_{ 4 } \amp = \set{ \of{ 1 , \mathrm{ blue } } , \of{ 2 , \mathrm{ blue } } }\text{.} \end{align*}
      As defined above, \(f_{ 1 } \text{,}\) \(f_{ 2 } \text{,}\) \(f_{ 3 } \text{,}\) and \(f_{ 4 } \) are maps from \(X \) to \(Y \text{.}\) In fact, these are the only such maps; that is,
      \begin{equation*} Y^{ X } = \set{ f_{ 1 } , f_{ 2 } , f_{ 3 } , f_{ 4 } }\text{.} \end{equation*}
      🔗
      🔗
    2. The empty set \(\emptyset \colon \emptyset \to Y \) is the unique map from the empty set \(\emptyset \) to any set \(Y \text{;}\) that is,
      \begin{equation*} Y^{ \emptyset } = \set{ \emptyset }\text{.} \end{equation*}
      Note that none of the elements of the codomain \(Y \) are associated with elements of the empty domain \(\emptyset \text{.}\) However, this does not imply that \(\emptyset \) is not a map; in general, not all elements of the codomain of a map need be associated with elements of the domain, since the existence property applies only to the domain.
      🔗
      🔗
    3. Let \(U \subseteq X \) be a subset of a set \(X \text{.}\) The inclusion map \(\iota_{ U }^{ X } \colon U \to X \) of \(U \) in \(X \) is the map from \(U \) to \(X \) defined by
      \begin{equation*} \iota_{ U }^{ X } = \setmid{ \of{ u , x } \in U \times X }{ u = x } . \end{equation*}
      That is, \(\iota_{ U }^{ X } \of{ u } = u \) for all \(u \in U \text{.}\)
      🔗
      The inclusion map \(\iota_{ X }^{ X } \) of \(X \) in itself is called the identity map on \(X \text{,}\) and is denoted \(\id_{ X } \colon X \to X \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Remark 1.2.5. Passive and active characterizations of maps.

    It may be difficult to see how the above definition of a map is a generalization of the notion of a real-valued function of a real variable. In fact, the above definition might seem more closely related to the graph of such a function, since the uniqueness property of maps is just a generalization of the vertical line test for functions. We will refer to this perspective on maps as subsets of a Cartesian product satisfying certain abstract properties as the passive characterization of what maps are. In contrast, we often like to think of a map as an active process, which takes an input and transforms it into the unique output with which it is associated. We will refer to this perspective as the active characterization of what maps do. When it benefits us to favor one perspective over another, we will emphasize either the active or passive characterization over the other. The following definitions allow us to convert between these two equivalent characterizations of maps.
    🔗
    🔗

    Definition 1.2.6. Image; pre-image.

    Let \(f \colon X \to Y \) be a map from a set \(X \) to a set \(Y \text{.}\) The image \(f \of{ x } \) of an input \(x \in X \) under \(f \) is the unique output so that \(\of{ x , f \of{ x } } \in Y \text{.}\) The image \(f \of{ U } \) of a subset \(U \subseteq X \) is the subset of the codomain \(Y \) defined by
    \begin{equation*} f \of{ U } = \setmid{ f \of{ x } \in Y }{ x \in U }\text{.} \end{equation*}
    The image \(f \of{ X } \) of \(X \) itself is called the image of \(f \text{,}\) and is denoted \(\im \of{ f } = f \of{ X } \text{.}\)
    🔗
    The pre-image \(f^{ -1 } \of{ V } \) of a subset \(V \subseteq Y \) of the codomain \(Y \) under \(f \) is the set of all inputs whose image lies in \(V \text{;}\) that is,
    \begin{equation*} f^{ -1 } \of{ V } = \setmid{ x \in X }{ f \of{ x } \in V }\text{.} \end{equation*}
    The pre-image \(f^{ -1 } \of{ y } \) of an output \(y \in Y \) under \(f \) is just the pre-image of the singleton set \(\set{ y } \) under \(f \text{;}\) that is,
    \begin{equation*} f^{ -1 } \of{ y } = f^{ -1 } \of{ \set{ y } } = \setmid{ x \in X }{ f \of{ x } = y } \end{equation*}
    🔗
    🔗
    Any map between sets induces a map from any subset of its domain to its codomain via a process known as restriction. In light of the active characterization of what maps do, this may seem almost trivial. However, we will formalize this notion of restriction in terms of the passive characterization of what maps are in order to prove rigorously our intuitions are correct. You may be better served by conceptualizing the restriction of a map to be the same active process of transforming inputs into outputs but with a smaller domain, rather than the passive definition below.
    🔗

    Definition 1.2.7. Restriction.

    The restriction \(\rest{ f }_{ U } \) of a map \(f \colon X \to Y \) between sets \(X \) and \(Y \) to a subset \(U \subseteq X \) of its domain \(X \) is the subset of the Cartesian product \(U \times Y \) defined by
    \begin{align*} \rest{ f }_{ U } \amp = \setmid{ \of{ u , y } \in U \times Y }{ f \of{ u } = y } = \setmid{ \of{ u , y } \in U \times Y }{ \of{ u , y } \in f }\\ \amp = \of{ U \times Y } \cap f\text{.} \end{align*}
    🔗
    🔗
    As our intuition about the active characterization of what maps do tells us, the restriction of a map to a subset of its domain is itself a map from the chosen subset to the domain of the original map.
    🔗

    Proposition 1.2.8. Restrictions are maps.

    Let \(f \colon X \to Y \) be a map from a set \(X \) to a set \(Y \text{.}\) For any subset \(U \subseteq X \text{,}\) the restriction \(\rest{ f }_{ U } \) of \(f \) to \(U \) is a map from \(U \) to \(Y \text{.}\)
    🔗
    🔗

    Proof.
    Let \(u \in U \text{.}\) Then \(u \in X \) is an input to \(f \text{,}\) and so \(\of{ u , y } \in f \) for some output \(y \in Y \text{.}\) We conclude that
    \begin{equation*} \of{ x , y } \in \setmid{ \of{ u , y } \in U \times Y }{ \of{ u , y } \in f } = \rest{ f }_{ U }\text{,} \end{equation*}
    so that \(\rest{ f }_{ U } \) satisfies the totality condition.
    🔗
    Let \(u \in U \) and \(y_{ 1 } , y_{ 2 } \in U \text{,}\) and suppose that \(\of{ u , y_{ 1 } } , \of{ u , y_{ 2 } } \in \rest{ f }_{ U } \text{.}\) Since
    \begin{equation*} \rest{ f }_{ U } = \of{ U \times Y } \cap f \subseteq f\text{,} \end{equation*}
    we have that \(\of{ u , y_{ 1 } } , \of{ u , y_{ 2 } } \in f \text{,}\) and so \(y_{ 1 } = y_{ 2 } \text{;}\) that is, \(\rest{ f }_{ U } \) satisfies the uniqueness condition. Since \(\rest{ f }_{ U } \) satisfies the totality condition and the uniqueness condition, it is a map from \(U \) to \(Y \text{.}\)
    🔗
    🔗
    We will shortly see another way to construct restrictions as compositions with inclusion maps.
    🔗
    🔗

    Subsection 1.2.2 Composition and Invertibility

    Viewed through the perspective of the active characterization of what maps do, it may seem obvious that we can combine two maps between sets by first acting on it with one map and then acting on its image by the other map. This new process yields the composition of the two maps, and it may be intuitively clear that this composition is itself a map. However, we will formalize this notion of composition in the passive characterization, so that we may prove this fact rigorously. Again, it may better serve you to conceptualize the composition of maps in the context of the active characterization of what maps do rather than the passive characterization of what maps are in order to better inform your intuition about composition.
    🔗

    Definition 1.2.9. Composition.

    Let \(f \colon X \to Y \) and \(g \colon Y \to Z \) be maps between sets \(X \text{,}\) \(Y \text{,}\) and \(Z \text{.}\) The composition \(g \circ f \) of \(g \) with \(f \) is the map from \(X \) to \(Y \) defined by the formula
    \begin{equation*} \of{ g \circ f } \of{ x } = g \of{ f \of{ x } }\text{.} \end{equation*}
    🔗
    🔗
    We aren’t restricted to the composition of just two maps. In fact, we can compose any finite number of maps with compatible domains and codomains by iterating the above construction.
    🔗

    Proposition 1.2.10. Associativity of composition.

    Let \(f \colon W \to X \text{,}\) \(g \colon X \to Y \text{,}\) and \(h \colon Y \to Z \) be maps between sets \(W \text{,}\) \(X \text{,}\) \(Y \text{,}\) and \(Z \text{.}\) Then
    \begin{equation*} \of{ h \circ g } \circ f = h \circ \of{ g \circ f }\text{.} \end{equation*}
    🔗
    🔗

    Proof.
    Note that
    \begin{align*} \of{ \of{ h \circ g } \circ f } \of{ w } \amp = \of{ h \circ g } \of{ f \of{ w } } = h \of{ g \of{ f \of{ w } } }\\ \amp = h \of{ \of{ g \circ f } \of{ w } } = \of{ h \circ \of{ g \circ f } } \of{ w } \end{align*}
    for all \(w \in W \text{.}\) Since \(\of{ h \circ g } \circ f \) and \(h \circ \of{ g \circ f } \) agree on all inputs \(w \in W \text{,}\)
    \begin{equation*} \of{ h \circ g } \circ f = h \circ \of{ g \circ f }\text{.} \end{equation*}
    🔗
    🔗
    The above notion of composition introduces a natural question about maps between sets:
    🔗
    This question of map invertibility is fundamental, and will recur throughout this text.
    🔗

    Definition 1.2.11. Invertibility.

    Let \(f \colon X \to Y \) be a map from a set \(X \) to a set \(Y \text{.}\) A map \(g \colon Y \to X \) is a left inverse to \(f \) if \(g \circ f = \id_{ X } \text{,}\) and \(g \) is called a right inverse to \(f \) if \(f \circ g = \id_{ Y } \text{.}\) \(g \) is called an inverse to \(f \) if it is both a left and right inverse to \(f \text{;}\) that is, \(g \) is an inverse to \(f \) if \(g \circ f = \id_{ X } \) and \(f \circ g = \id_{ Y } \text{.}\)
    🔗
    A map is called left-invertible (resp. right-invertible, invertible) if it has a left inverse (resp. right inverse, inverse).
    🔗
    🔗

    Example 1.2.12. Examples of inverses.

    The following are examples of left inverses, right inverses, and inverses of maps.
    1. Let \(\emptyset \subsetneq U \subseteq X \) be a non-empty subset of a set \(X \text{,}\) and fix an element \(u \in U \text{.}\) Consider the constant map \(c \colon X \to U \) from \(X \) to \(U \) defined piecewise by the formula
      \begin{equation*} c \of{ x } = \begin{cases} x , \amp x \in U \\ u , \amp x \notin U \text{.}\end{cases} \end{equation*}
      Note that
      \begin{equation*} \of{ c \circ \iota_{ U }^{ X } } \of{ u } = c \of{ \iota_{ U }^{ X } \of{ u } } = c \of{ u } = u = \id_{ U } \of{ u } \end{equation*}
      for all \(u \in U \text{,}\) and so \(c \circ \iota_{ U }^{ X } = \id_{ U } \text{;}\) that is, \(c \) is a left inverse to the inclusion map \(\iota_{ U }^{ X } \colon U \to X \) of \(U \) in \(X \text{.}\)
      🔗
      🔗
    2. Fix sets \(X \) and \(Y \text{,}\) and suppose that \(Y \neq \emptyset \) is non-empty. Let \(\pi \colon X \times Y \to X \) be the projection map defined by the formula \(\pi \of{ x , y } = x \text{.}\)
      🔗
      Fix an element \(y \in Y \text{,}\) and consider the map \(c \colon X \to X \times Y \) from \(X \) to \(X \times Y \) defined by the formula \(c \of{ x } = \of{ x , y } \text{.}\) Then
      \begin{equation*} \of{ \pi \circ c } \of{ x } = \pi \of{ c \of{ x } } = \pi \of{ x , y } = x = \id_{ X } \of{ x } \end{equation*}
      for all \(x \in X \text{,}\) and so \(\pi \circ c = \id_{ X } \text{;}\) that is, \(c \) is a right inverse to the projection map \(\pi \text{.}\)
      🔗
      🔗
    3. The identity map \(\id_{ X } \colon X \to X \) on a set \(X \) is its own inverse, since \(\id_{ X } \circ \id_{ X } = \id_{ X } \text{.}\) This is actually a special case of the first example above, where we take \(U = X \text{.}\) More generally, \(f \circ \id_{ X } = f \) for all maps \(f \) with domain \(X \text{,}\) and \(\id_{ X } \circ g = g \) for all maps \(g \) with codomain \(X \text{.}\)
      🔗
      🔗
    🔗
    🔗
    We now give alternative characterizations of left-invertible, right-invertible, and invertible maps. Intuitively, a map is left-invertible if it does not associate two distinct inputs with the same output, and a map is right-invertible if each output is associated with at least one input. These properties mirror the totality and uniqueness conditions for maps, and will turn out to together imply the existence of a two-sided inverse, which is both a left and right inverse.
    🔗

    Definition 1.2.13. Injectivity; surjectivity; bijectivity.

    Let \(f \colon X \to Y \) be a map from a set \(X \) to a set \(Y \text{.}\)
    1. \(f \) is called injective (or one-to-one) if for all inputs \(x_{ 1 } , x_{ 2 } \in X \text{,}\) if \(f \of{ x_{ 1 } } = f \of{ x_{ 2 } } \text{,}\) then \(x_{ 1 } = x_{ 2 } \text{.}\) In this case, \(f \) is also called an injection of \(X \) into \(Y \text{.}\)
      🔗
      🔗
    2. \(f \) is called surjective (or onto) if for all outputs \(y \in Y \) there is an input \(x \in X \) so that \(y = f \of{ x } \text{.}\) In this case, \(f \) is also called a surjection of \(X \) onto \(Y \text{.}\)
      🔗
      🔗
    3. \(f \) is called bijective if it is both injective and surjective. In this case, \(f \) is also called a bijection or a one-to-one correspondence between \(X \) and \(Y \text{.}\)
      🔗
      🔗
    🔗
    🔗

    Example 1.2.14. Examples of injections, surjections, and bijections.

    The following are examples of injective, surjective, and bijective functions.
    1. Let \(U \subseteq X \) be a subset of a set \(X \text{,}\) and consider \(u_{ 1 } , u_{ 2 } \in U \text{.}\) If \(\iota_{ U }^{ X } \of{ u_{ 1 } } = \iota_{ U }^{ X } \of{ u_{ 2 } } \text{,}\) then
      \begin{equation*} u_{ 1 } = \iota_{ U }^{ X } \of{ u_{ 1 } } = \iota_{ U }^{ X } \of{ u_{ 2 } } = u_{ 2 }\text{.} \end{equation*}
      So the inclusion map \(\iota_{ U }^{ X } \colon U \to X \) is injective.
      🔗
      🔗
    2. Fix sets \(X \) and \(Y \text{,}\) and suppose that \(Y \neq \emptyset \) is non-empty. Let \(\pi \colon X \times Y \to X \) be the projection map defined by the formula \(\pi \of{ x , y } = x \text{.}\)
      🔗
      Let \(x \in X \text{,}\) and note that since \(Y \neq \emptyset \) is non-empty, there is some element \(y \in Y \text{.}\) We note that \(\pi \of{ x , y } = x \text{.}\) Since each output \(x \in X \) is the image of some input under \(\pi \text{,}\) \(\pi \) is surjective.
      🔗
      🔗
    3. The identity map \(\id_{ X } \colon X \to X \) on a set \(X \) is injective by virtue of being an inclusion map. \(\id_{ X } \) is also surjective, since \(\id_{ X } \of{ x } = x \) for all \(x \in X \text{.}\) Thus \(\id_{ X } \) is bijective.
      🔗
      🔗
    🔗
    🔗
    It is no accident that the examples for left- and right- invertibility coincided with the examples for injectivity and surjectivity. It turns out that injectivity is essentially equivalent to left-invertibility, and surjectivity is equivalent to right-invertibility.
    🔗

    Theorem 1.2.15. Invertibility.

    Let \(f \colon X \to Y \) be a map from a set \(X \) to a set \(Y \text{.}\)
    1. If \(f \) is left-invertible, then it is injective.
      🔗
      🔗
    2. If \(X \neq \emptyset \) is non-empty and \(f \) is injective, then it is left-invertible.
      🔗
      🔗
    3. If \(f \) is right-invertible, then it is surjective.
      🔗
      🔗
    🔗
    🔗

    Proof.
    1. Suppose that \(f \) is left-invertible, so that \(g \circ f = \id_{ X } \) for some map \(g \colon Y \to X \text{.}\) That \(f \) is injective follows from the injectivity of the identity map \(\id_{ X } \) and (d) of [provisional cross-reference: problem-compositions-of-injections-surjections-and-bijections].
      🔗
      🔗
    2. Conversely, now suppose that \(X \neq \emptyset \) is non-empty and \(f \) is injective. Then for each output \(y \in Y \text{,}\) there is at most one input \(x \in X \) so that \(y = f \of{ x } \text{.}\) Specifically, for each \(y \in \im \of{ f } \text{,}\) there is a unique input \(x \in X \) so that \(y = f \of{ x } \text{.}\) Denote by \(g \of{ y } = x \) this unique input.
      🔗
      Since \(X \neq \emptyset \) is non-empty, it contains an element \(x \in X \text{.}\) Define \(g \of{ y } = x \) for all \(y \in Y \setminus \im \of{ f } \text{.}\) We have defined a map \(g \colon Y \to X \text{.}\) Note that
      \begin{equation*} \of{ g \circ f } \of{ x } = g \of{ f \of{ x } } = x = \id_{ X } \of{ x } \end{equation*}
      for all inputs \(x \in X \text{,}\) and so \(g \circ f = \id_{ X } \text{;}\) that is, \(g \) is a left inverse to \(f \text{,}\) and so \(f \) is left-invertible.
      🔗
      🔗
    3. Finally, suppose that \(f \) is right-invertible, so that \(f \circ g = \id_{ Y } \) for some map \(g \colon Y \to X \text{.}\) That \(f \) is surjective follows from the surjectivity of the identity map \(\id_{ Y } \) and (e) of [provisional cross-reference: problem-compositions-of-injections-surjections-and-bijections].
      🔗
      🔗
    🔗
    Proving the remaining direction of the equivalence between right invertibility and surjectivity requires a strong assumption about set theory called the axiom of choice, which we do not yet have the language to express in its simplest form. We will talk about the axiom of choice shortly, when we generalize the Cartesian product of two sets using indexing maps.
    🔗

    Corollary 1.2.16. Invertibility.

    A map is invertible if and only if it is bijective.
    🔗
    🔗

    Proof.
    Let \(f \colon X \to Y \) be a map from a set \(X \) to a set \(Y \text{.}\) If \(f \) is invertible, it is both left- and right-invertible, and so \(f \) is both injective and surjective by (1) and (3) of Theorem 1.2.15 Invertibility. Hence \(f \) is bijective.
    🔗
    Conversely, now suppose that \(f \) is bijective, so that \(f \) is both injective and surjective. If \(X = \emptyset \text{,}\) then
    \begin{align*} Y \amp = \im \of{ f } = f \of{ X } = f \of{ \emptyset }\\ \amp = \setmid{ f \of{ x } \in Y }{ x \in \emptyset } = \emptyset \end{align*}
    by the surjectivity of \(f \text{.}\) Thus \(f \colon \emptyset \to \emptyset \) is the unique map from \(\emptyset \) to itself, the identity map \(\id_{ \emptyset } \colon \emptyset \to \emptyset \text{.}\) In particular, \(f \) is invertible.
    🔗
    So we may assume that \(X \neq \emptyset \) is non-empty. Since \(f \) is injective, it is left-invertible by (2) of Theorem 1.2.15 Invertibility. So \(g \circ f = \id_{ X } \) for some map \(g \colon Y \to X \text{;}\) that is, \(g \of{ f \of{ x } } = x \) for all inputs \(x \in X \text{.}\) It suffices to show that \(g \) is also a right inverse to \(f \text{.}\)
    🔗
    To that end, consider an output \(y \in Y \text{.}\) Since \(f \) is surjective, \(y = f \of{ x } \) for some input \(x \in X \text{.}\) Then
    \begin{equation*} \of{ f \circ g } \of{ y } = f \of{ g \of{ x } } = f \of{ g \of{ f \of{ x } } } = f \of{ x } = y = \id_{ Y } \of{ y }\text{.} \end{equation*}
    Hence \(f \circ g = \id_{ Y } \text{,}\) and so \(g \) is an inverse to \(f \text{;}\) that is, \(f \) is invertible.
    🔗
    🔗
    We will investigate the invertibility and bijectivity of maps and the consequences these properties have on the domains and codomains of such maps more at the end of this chapter, when we study cardinality and the size of sets. For now, however, we will move on to an important convention which is ubiquitous throughout mathematics: the notions of indexed sets and families.
    🔗
    🔗

    Subsection 1.2.3 Indexing

    Considered as an association between sets, a map can be thought to index its image by the elements of its domain. That is, a map can transfer information from its domain to its image. Just what this transferred information looks like can vary tremendously, but suffice it to say that this notion of indexing is fundamental and will recur throughout this text.
    🔗

    Definition 1.2.17. Indexed family; indexed set.

    A set \(X \) is indexed by a set \(\mathcal{ J } \) if there is a surjection \(x \colon \mathcal{ J } \to X \text{.}\) \(\mathcal{ J } \) is called an indexing set, and for each input \(j \in \mathcal{ J } \text{,}\) called an index, its image is denoted \(x_{ j } = x \of{ j } \text{.}\)
    🔗
    The map \(x \) is called an (indexed) family of elements of \(X \text{,}\) and is denoted \(\of{ x_{ j } }_{ j \in \mathcal{ J } } \text{.}\)
    🔗
    Consider a map \(x \colon \mathcal{ J } \to X \) of a set \(\mathcal{ J } \text{,}\) called the indexing set and whose elements are called indices, onto a set \(X \text{.}\) The image of such an index \(j \in \mathcal{ J } \) under such an (indexed) family \(x \) is often denoted \(x_{ j } = x \of{ j } \text{,}\) and \(x \) is often denoted \(\of{ x_{ j } }_{ j \in \mathcal{ J } } \text{.}\) We say that \(\im \of{ x } \) is indexed by \(\mathcal{ J } \text{,}\) and we write
    \begin{equation*} \set{ x_{ j } }_{ j \in \mathcal{ J } } = \setmid{ x_{ j } \in X }{ j \in \mathcal{ J } } = \im \of{ x }\text{.} \end{equation*}
    🔗
    🔗
    We can also use this notion of indexing to generalize the intersection and union of sets to act on arbitrary collections of sets. This generalization of the intersection and union of two sets relies on the natural similarities between logical conjunction \(\land \) and universal quantification \(\forall \) and between logical disjunction \(\lor \) and existential quantification \(\exists \text{.}\)
    🔗

    Definition 1.2.18. Intersection; union.

    Let \(\of{ U_{ j } }_{ j \in \mathcal{ J } } \) be a family of subsets \(U_{ j } \subseteq X \) of a set \(X \) indexed by a set \(\mathcal{ J } \text{.}\) The intersection \(\bigcap_{ j \in \mathcal{ J } }{ U_{ j } } \) of this indexed family is the subset of \(X \) defined by
    \begin{equation*} \bigcap_{ j \in \mathcal{ J } }{ U_{ j } } = \setmid{ x \in X }{ x \in U_{ j } \textrm{ for all } j \in \mathcal{ J } }\text{.} \end{equation*}
    The union \(\bigcup_{ j \in \mathcal{ J } }{ U_{ j } } \) of this indexed family is the subset of \(X \) defined by
    \begin{equation*} \bigcup_{ j \in \mathcal{ J } }{ U_{ j } } = \setmid{ x \in X }{ x \in U_{ j } \textrm{ for some } j \in \mathcal{ J } }\text{.} \end{equation*}
    🔗
    🔗
    These generalizations of the intersection and union of two sets extend naturally to generalizations of each of the results proven in the previous section.
    🔗

    Theorem 1.2.19. Analogous properties of indexed intersections and unions.

    Let \(\of{ U_{ j } }_{ j \in \mathcal{ J } } \) be a family of subsets \(U_{ j } \subseteq X \) of a set \(X \) indexed by a set \(\mathcal{ J } \text{.}\)
    1. Analogously to Proposition 1.1.14 Set operations and containment, for any subset \(V \subseteq X \text{,}\) \(V \subseteq \bigcap_{ j \in \mathcal{ J } }{ U_{ j } } \) if and only if \(V \subseteq U_{ j } \) for all indices \(j \in \mathcal{ J } \text{,}\) and \(\bigcup_{ j \in \mathcal{ J } }{ U_{ j } } \subseteq V \) if and only if \(U_{ j } \subseteq V \) for all indices \(j \in \mathcal{ J } \text{.}\)
      🔗
      🔗
    2. Analogously to (2) of Corollary 1.1.15 Properties of the set operations,
      \begin{equation*} \bigcap_{ k \in \mathcal{ K } }{ U_{ f \of{ k } } } = \bigcap_{ j \in \mathcal{ J } }{ U_{ j } } \end{equation*}
      and
      \begin{equation*} \bigcup_{ k \in \mathcal{ K } }{ U_{ f \of{ k } } } = \bigcup_{ j \in \mathcal{ J } }{ U_{ j } } \end{equation*}
      for any set \(\mathcal{ K } \) and bijection \(f \colon \mathcal{ K } \to \mathcal{ J } \text{.}\)
      🔗
      🔗
    3. Analogously to [provisional cross-reference: problem-distributivity-of-set-operations],
      \begin{equation*} V \cup \bigcap_{ j \in \mathcal{ J } }{ U_{ j } } = \bigcap_{ j \in \mathcal{ J } }{ \of{ V \cup U_{ j } } } \end{equation*}
      and
      \begin{equation*} V \cap \bigcup_{ j \in \mathcal{ J } }{ U_{ j } } = \bigcup_{ j \in \mathcal{ J } }{ \of{ V \cap U_{ j } } } \end{equation*}
      for any subset \(V \subseteq X \text{.}\)
      🔗
      🔗
    4. Analogously to [provisional cross-reference: problem-de-morgans-laws],
      \begin{equation*} V \setminus \bigcap_{ j \in \mathcal{ J } }{ U_{ j } } = \bigcup_{ j \in \mathcal{ J } }{ \of{ V \setminus U_{ j } } } \end{equation*}
      and
      \begin{equation*} V \setminus \bigcup_{ j \in \mathcal{ J } }{ U_{ j } } = \bigcap_{ j \in \mathcal{ J } }{ \of{ V \setminus U_{ j } } } \end{equation*}
      for any subset \(V \subseteq X \text{.}\)
      🔗
      🔗
    🔗

    🔗

    Prove that if \(f \) and \(g \) are injective, then so is their composition \(g \circ f \colon X \to Z \text{.}\)
    🔗
